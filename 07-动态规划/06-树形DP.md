# 树形 DP

> Made by Aki.
> 最后更新于 2026.06.03

树形 DP 是在树结构上进行动态规划的一类问题。它的核心思想是利用树的递归性质，自底向上地计算每个节点的最优解。

---

## 一、引言

来看一个经典问题：

> 给定一棵 $n$ 个节点的树，每条边有一个权值。求树上任意两点之间的最长路径（路径长度为边权之和）。

这就是**树的直径问题**——树形 DP 的经典应用。

为什么这个问题适合用树形 DP？因为树的直径一定是从某个节点出发，经过它的子树，到达另一个节点。我们可以利用这个性质，自底向上地计算每个节点的最优解。

## 三、核心推导

### 3.1 树的直径

#### 问题定义

给定一棵 $n$ 个节点的树，每条边有一个权值。求树上任意两点之间的最长路径。

#### 状态定义

对于每个节点 $u$，我们定义 $d[u]$ 为：**从 $u$ 出发，向下走到叶子节点的最长路径长度**。

#### 状态转移方程

对于节点 $u$，它有若干子节点 $v_1, v_2, \dots, v_k$。从 $u$ 出发的最长路径，一定是先走到某个子节点 $v_i$，然后继续向下走。

$$
d[u] = \max_{v \in \text{children}(u)} \{d[v] + w(u, v)\}
$$

其中 $w(u, v)$ 是边 $(u, v)$ 的权值。

#### 树的直径计算

树的直径一定是某个节点 $u$ 的两个子树的最长路径之和：

$$
\text{diameter} = \max_{u} \{\max_{v_1, v_2 \in \text{children}(u), v_1 \ne v_2} \{d[v_1] + w(u, v_1) + d[v_2] + w(u, v_2)\}\}
$$

#### 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 100005;
vector<pair<int, int>> adj[MAXN];  // 邻接表，存 (子节点, 边权)
int d[MAXN];  // 从每个节点出发的最长路径
int diameter = 0;  // 树的直径

void dfs(int u, int parent) {
    d[u] = 0;
    for (auto& [v, w] : adj[u]) {
        if (v == parent) continue;
        dfs(v, u);
        // 更新树的直径：用当前最长路径和子节点的最长路径更新
        diameter = max(diameter, d[u] + d[v] + w);
        // 更新从 u 出发的最长路径
        d[u] = max(d[u], d[v] + w);
    }
}

int main() {
    int n;
    cin >> n;
    for (int i = 1; i < n; i++) {
        int u, v, w;
        cin >> u >> v >> w;
        adj[u].push_back({v, w});
        adj[v].push_back({u, w});
    }
    dfs(1, -1);
    cout << diameter << endl;
    return 0;
}
```

**时间复杂度**：$O(n)$
**空间复杂度**：$O(n)$

**❓ 为什么要在更新直径之后再更新 $d[u]$？**

因为直径需要的是两个子树的最长路径之和。如果先更新 $d[u]$，那么 $d[u]$ 已经包含了当前子树的信息，再用它更新直径就会出错。

### 3.2 树的重心

#### 问题定义

给定一棵 $n$ 个节点的树，找到一个节点，使得删除这个节点后，剩余子树中最大子树的节点数最小。

这个节点叫做**树的重心**。

#### 状态定义

对于每个节点 $u$，我们定义 $sz[u]$ 为：**以 $u$ 为根的子树的节点数**。

#### 状态转移方程

$$
sz[u] = 1 + \sum_{v \in \text{children}(u)} sz[v]
$$

#### 重心计算

对于节点 $u$，删除它后，剩余子树包括：
- 它的每个子树，节点数为 $sz[v]$
- 它的父节点方向的子树，节点数为 $n - sz[u]$

最大子树的节点数为：

$$
\max\_subtree[u] = \max(\max_{v \in \text{children}(u)} \{sz[v]\}, n - sz[u])
$$

重心就是使 $\max\_subtree[u]$ 最小的节点。

#### 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 100005;
vector<int> adj[MAXN];
int sz[MAXN];
int max_subtree[MAXN];  // 删除节点后最大子树的节点数
int n;  // 树的节点数（全局变量，供 dfs 使用）
int centroid = -1;
int min_max_subtree = INT_MAX;

void dfs(int u, int parent) {
    sz[u] = 1;
    max_subtree[u] = 0;
    for (int v : adj[u]) {
        if (v == parent) continue;
        dfs(v, u);
        sz[u] += sz[v];
        max_subtree[u] = max(max_subtree[u], sz[v]);
    }
    // 父节点方向的子树
    max_subtree[u] = max(max_subtree[u], n - sz[u]);
    // 更新重心
    if (max_subtree[u] < min_max_subtree) {
        min_max_subtree = max_subtree[u];
        centroid = u;
    }
}

int main() {
    cin >> n;
    for (int i = 1; i < n; i++) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    dfs(1, -1);
    cout << centroid << endl;
    return 0;
}
```

**时间复杂度**：$O(n)$
**空间复杂度**：$O(n)$

**❓ 树的重心有什么性质？**

1. 树的重心最多有两个。
2. 如果树有两个重心，那么它们相邻，且删除它们后最大子树的节点数都是 $n/2$。
3. 树的重心到所有节点的距离之和最小。

### 3.3 树上背包

#### 问题定义

有 $n$ 个节点的树，每个节点有一定的价值 $v_i$ 和重量 $w_i$。从根节点出发，选择一个连通子图，使得总重量不超过 $W$，总价值最大。

#### 状态定义

我们定义 $dp[u][j]$ 为：**在以 $u$ 为根的子树中，选择总重量为 $j$ 的连通子图（必须包含 $u$）的最大价值**。

#### 状态转移方程

对于节点 $u$，我们先初始化：

$$
dp[u][w_u] = v_u
$$

然后依次合并每个子节点 $v$ 的信息：

$$
dp[u][j] = \max_{0 \le k \le j - w_u} \{dp[u][j - k] + dp[v][k]\}
$$

这是一个类似背包的转移。

#### 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 105;
vector<int> adj[MAXN];
int w[MAXN], v[MAXN];
int dp[MAXN][MAXN];  // dp[u][j] 表示以 u 为根的子树中，重量为 j 的最大价值
int W;

void dfs(int u, int parent) {
    // 初始化：只选 u 自己
    dp[u][w[u]] = v[u];
    
    for (int vv : adj[u]) {
        if (vv == parent) continue;
        dfs(vv, u);
        // 合并子节点的信息（类似背包）
        for (int j = W; j >= w[u]; j--) {
            for (int k = 0; k <= j - w[u]; k++) {
                dp[u][j] = max(dp[u][j], dp[u][j - k] + dp[vv][k]);
            }
        }
    }
}

int main() {
    int n;
    cin >> n >> W;
    for (int i = 1; i <= n; i++) {
        cin >> w[i] >> v[i];
    }
    for (int i = 1; i < n; i++) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    dfs(1, -1);
    int ans = 0;
    for (int j = 0; j <= W; j++) {
        ans = max(ans, dp[1][j]);
    }
    cout << ans << endl;
    return 0;
}
```

**时间复杂度**：$O(nW^2)$
**空间复杂度**：$O(nW)$

**❓ 为什么要逆序遍历 $j$？**

因为这是 01 背包的变形：每个节点只能选一次。逆序遍历保证在合并子节点时，不会重复选择同一个节点。

## 四、代码实现

以下是本篇所有核心问题的代码模板。

### 树的直径

```cpp
const int MAXN = 100005;
vector<pair<int, int>> adj[MAXN];
int d[MAXN];
int diameter = 0;

void dfs(int u, int parent) {
    d[u] = 0;
    for (auto& [v, w] : adj[u]) {
        if (v == parent) continue;
        dfs(v, u);
        diameter = max(diameter, d[u] + d[v] + w);
        d[u] = max(d[u], d[v] + w);
    }
}
```

### 树的重心

```cpp
const int MAXN = 100005;
vector<int> adj[MAXN];
int sz[MAXN];
int max_subtree[MAXN];
int centroid = -1;
int min_max_subtree = INT_MAX;

void dfs(int u, int parent) {
    sz[u] = 1;
    max_subtree[u] = 0;
    for (int v : adj[u]) {
        if (v == parent) continue;
        dfs(v, u);
        sz[u] += sz[v];
        max_subtree[u] = max(max_subtree[u], sz[v]);
    }
    max_subtree[u] = max(max_subtree[u], n - sz[u]);
    if (max_subtree[u] < min_max_subtree) {
        min_max_subtree = max_subtree[u];
        centroid = u;
    }
}
```

### 树上背包

> 注意：以下模板依赖全局变量 $W$（背包容量），使用前请定义 `int W;`。

```cpp
const int MAXN = 105;
vector<int> adj[MAXN];
int w[MAXN], v[MAXN];
int dp[MAXN][MAXN];

void dfs(int u, int parent) {
    // 初始化为 -INF，表示无效状态
    for (int j = 0; j <= W; j++) dp[u][j] = -1e9;
    dp[u][w[u]] = v[u];
    for (int vv : adj[u]) {
        if (vv == parent) continue;
        dfs(vv, u);
        for (int j = W; j >= w[u]; j--) {
            for (int k = 0; k <= j - w[u]; k++) {
                dp[u][j] = max(dp[u][j], dp[u][j - k] + dp[vv][k]);
            }
        }
    }
}
```

## 五、复杂度与正确性分析

| 问题 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 树的直径 | $O(n)$ | $O(n)$ |
| 树的重心 | $O(n)$ | $O(n)$ |
| 树上背包 | $O(nW^2)$ | $O(nW)$ |

**正确性保证**：

1. **状态定义正确**：每个 $dp$ 值都有明确的含义，且能覆盖所有可能的情况。

2. **状态转移正确**：利用树的递归性质，自底向上地计算最优解。

3. **遍历顺序正确**：DFS 遍历保证子节点的信息在父节点之前计算。

## 七、小结

本文介绍了三个经典的树形 DP 问题：**树的直径**、**树的重心** 和 **树上背包**。

四个核心要点：

1. 树形 DP 利用树的递归性质，自底向上地计算每个节点的最优解。

2. 树的直径可以通过 DFS 在 $O(n)$ 时间内求解，关键是在更新直径之后再更新 $d[u]$。

3. 树的重心是使删除后最大子树最小的节点，可以通过一次 DFS 求解。

4. 树上背包是背包问题在树结构上的扩展，需要在合并子节点时注意遍历顺序。
