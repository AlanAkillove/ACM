# 状压 DP

> Made by Aki.
> 最后更新于 2026.06.03

状压 DP（状态压缩动态规划）是一类用二进制表示状态的动态规划方法。它的核心思想是用一个整数的二进制位来表示集合的状态，从而实现状态的快速转移。

---

## 一、引言

来看一个经典问题：

> 有 $n$ 个城市和 $n$ 个商人，每个商人要去一个不同的城市。从城市 $i$ 到城市 $j$ 的费用为 $c_{ij}$。求所有商人旅行的最小总费用。

这就是**旅行商问题（TSP）**——NP 难问题，但当 $n$ 较小（比如 $n \le 20$）时，可以用状压 DP 解决。

为什么状压 DP 适合这个问题？因为我们需要记录「哪些城市已经去过」，而城市数量 $n$ 很小，可以用一个 $n$ 位的二进制数来表示。

## 三、核心推导

### 3.1 旅行商问题（TSP）

#### 问题定义

有 $n$ 个城市，从城市 0 出发，经过所有城市恰好一次，最后回到城市 0。从城市 $i$ 到城市 $j$ 的费用为 $c_{ij}$。求最小总费用。

#### 状态定义

我们定义 $dp[S][i]$ 为：**已经访问过的城市集合为 $S$，当前在城市 $i$ 时，访问剩余城市的最小费用**。

其中 $S$ 是一个 $n$ 位的二进制数，第 $j$ 位为 1 表示城市 $j$ 已经访问过。

#### 状态转移方程

从城市 $i$ 出发，选择下一个未访问的城市 $j$：

$$
dp[S][i] = \min_{j \notin S} \{dp[S \cup \{j\}][j] + c_{ij}\}
$$

#### 边界条件

$dp[(1 << n) - 1][i] = c_{i0}$（所有城市都访问过，从 $i$ 返回城市 0 的费用）

#### 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 20;
const int INF = 1e9;
int c[MAXN][MAXN];
int dp[1 << MAXN][MAXN];

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            cin >> c[i][j];
        }
    }
    
    // 初始化
    for (int S = 0; S < (1 << n); S++) {
        for (int i = 0; i < n; i++) {
            dp[S][i] = INF;
        }
    }
    
    // 边界条件：所有城市都访问过
    for (int i = 0; i < n; i++) {
        dp[(1 << n) - 1][i] = c[i][0];
    }
    
    // 状态转移：从后往前计算
    for (int S = (1 << n) - 2; S >= 0; S--) {
        for (int i = 0; i < n; i++) {
            if (!(S & (1 << i))) continue;  // i 必须在集合 S 中
            for (int j = 0; j < n; j++) {
                if (S & (1 << j)) continue;  // j 不能在集合 S 中
                dp[S][i] = min(dp[S][i], dp[S | (1 << j)][j] + c[i][j]);
            }
        }
    }
    
    // 答案：从城市 0 出发，访问所有城市的最小费用
    cout << dp[1][0] << endl;
    return 0;
}
```

**时间复杂度**：$O(2^n \cdot n^2)$
**空间复杂度**：$O(2^n \cdot n)$

**❓ 为什么 $S$ 要从大到小枚举？**

因为 $dp[S][i]$ 依赖于 $dp[S \cup \{j\}][j]$，而 $S \cup \{j\} > S$。从大到小枚举保证依赖的状态已经计算过。

**❓ 为什么 $i$ 必须在集合 $S$ 中？**

因为 $dp[S][i]$ 表示「已经访问过集合 $S$ 中的城市，当前在城市 $i$」。如果 $i$ 不在 $S$ 中，这个状态没有意义。

### 3.2 覆盖问题

#### 问题定义

有一个 $n \times m$ 的网格，用 $1 \times 2$ 的多米诺骨牌覆盖，求有多少种覆盖方式。

#### 状态定义

我们定义 $dp[i][S]$ 为：**处理到第 $i$ 行，第 $i$ 行的覆盖状态为 $S$ 时，前 $i$ 行的覆盖方案数**。

其中 $S$ 是一个 $m$ 位的二进制数，第 $j$ 位为 1 表示第 $j$ 列被覆盖。

#### 状态转移方程

对于第 $i$ 行，我们需要枚举所有可能的覆盖方式：

1. **不放骨牌**：$S = 0$，转移自 $dp[i-1][S']$，其中 $S'$ 的所有位都是 1（上一行全部覆盖）

2. **横放骨牌**：在第 $j$ 列和第 $j+1$ 列横放一个骨牌，$S$ 的第 $j$ 位和第 $j+1$ 位为 1

3. **竖放骨牌**：在第 $j$ 列竖放一个骨牌，$S$ 的第 $j$ 位为 1，但上一行的第 $j$ 位必须是 0

#### 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 12;
long long dp[MAXN][1 << MAXN];
int n, m;

void dfs(int row, int col, int cur, int next) {
    if (col == m) {
        dp[row][cur] += dp[row - 1][next];
        return;
    }
    
    // 如果当前列已被上一行竖放骨牌覆盖，直接跳过
    if (next & (1 << col)) {
        dfs(row, col + 1, cur, next);
    } else {
        // 横放骨牌（占据本行 col 和 col+1 两列）
        if (col + 1 < m && !(next & (1 << (col + 1)))) {
            dfs(row, col + 2, cur, next);
        }
        // 竖放骨牌（占据本行和下一行）
        dfs(row, col + 1, cur | (1 << col), next);
    }
}

int main() {
    cin >> n >> m;
    
    // 初始化
    dp[0][(1 << m) - 1] = 1;
    
    // 状态转移
    for (int i = 1; i <= n; i++) {
        for (int S = 0; S < (1 << m); S++) {
            dfs(i, 0, 0, S);
        }
    }
    
    cout << dp[n][0] << endl;  // 最后一行不向下延伸的方案数
    return 0;
}
```

**时间复杂度**：$O(n \cdot 2^m \cdot 2^m)$
**空间复杂度**：$O(n \cdot 2^m)$

**❓ 为什么用 DFS 枚举覆盖方式？**

因为覆盖方式比较复杂（横放、竖放、不放），用 DFS 可以方便地枚举所有可能的情况。

### 3.3 集合划分问题

#### 问题定义

有 $n$ 个元素，将它们划分为 $k$ 个非空子集，求划分方案数。

#### 状态定义

我们定义 $dp[S][j]$ 为：**将集合 $S$ 划分为 $j$ 个非空子集的方案数**。

#### 状态转移方程

对于集合 $S$，我们枚举它的一个非空子集 $T$：

$$
dp[S][j] = \sum_{T \subseteq S, T \ne \emptyset} dp[S \setminus T][j - 1]
$$

#### 边界条件

$dp[\emptyset][0] = 1$（空集划分为 0 个子集，只有一种方案）

#### 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 20;
long long dp[1 << MAXN][MAXN];

int main() {
    int n, k;
    cin >> n >> k;
    
    // 初始化
    dp[0][0] = 1;
    
    // 状态转移
    for (int S = 1; S < (1 << n); S++) {
        int lowbit = S & -S;  // S 的最小元素
        for (int j = 1; j <= k; j++) {
            for (int T = S; T > 0; T = (T - 1) & S) {
                if (!(T & lowbit)) continue;  // 强制 T 包含最小元素，避免重复计数
                dp[S][j] += dp[S ^ T][j - 1];
            }
        }
    }
    
    cout << dp[(1 << n) - 1][k] << endl;
    return 0;
}
```

**时间复杂度**：$O(3^n \cdot k)$（枚举所有子集的子集）
**空间复杂度**：$O(2^n \cdot k)$

**❓ 如何枚举集合 $S$ 的所有子集？**

```cpp
for (int T = S; T > 0; T = (T - 1) & S) {
    // T 是 S 的一个非空子集
}
```

这是一个经典的位运算技巧，可以在 $O(2^{|S|})$ 时间内枚举 $S$ 的所有子集。

## 四、代码实现

以下是本篇所有核心问题的代码模板。

### 旅行商问题（TSP）

```cpp
const int MAXN = 20;
const int INF = 1e9;
int c[MAXN][MAXN];
int dp[1 << MAXN][MAXN];

int tsp(int n) {
    for (int S = 0; S < (1 << n); S++) {
        for (int i = 0; i < n; i++) {
            dp[S][i] = INF;
        }
    }
    for (int i = 0; i < n; i++) {
        dp[(1 << n) - 1][i] = c[i][0];
    }
    for (int S = (1 << n) - 2; S >= 0; S--) {
        for (int i = 0; i < n; i++) {
            if (!(S & (1 << i))) continue;
            for (int j = 0; j < n; j++) {
                if (S & (1 << j)) continue;
                dp[S][i] = min(dp[S][i], dp[S | (1 << j)][j] + c[i][j]);
            }
        }
    }
    return dp[1][0];
}
```

### 覆盖问题

```cpp
const int MAXN = 12;
int m;  // 列数（全局变量）
long long dp[MAXN][1 << MAXN];

void dfs(int row, int col, int cur, int next) {
    if (col == m) {
        dp[row][cur] += dp[row - 1][next];
        return;
    }
    // 如果上一行在当前列有竖放骨牌，当前格已被占用，直接跳过
    if (next & (1 << col)) {
        dfs(row, col + 1, cur, next);
    } else {
        // 横放：占用当前列和下一列
        if (col + 1 < m && !(next & (1 << (col + 1)))) {
            dfs(row, col + 2, cur, next);
        }
        // 竖放：占用当前格，并在下一行对应列标记占用
        dfs(row, col + 1, cur | (1 << col), next);
    }
}
```

### 集合划分问题

```cpp
const int MAXN = 20;
long long dp[1 << MAXN][MAXN];

long long countSetPartition(int n, int k) {
    dp[0][0] = 1;
    for (int S = 1; S < (1 << n); S++) {
        int lowbit = S & -S;  // 取最低位，强制 T 包含该位以避免重复计数
        for (int j = 1; j <= k; j++) {
            for (int T = S; T > 0; T = (T - 1) & S) {
                if (!(T & lowbit)) continue;  // 保证每个划分只被计数一次
                dp[S][j] += dp[S ^ T][j - 1];
            }
        }
    }
    return dp[(1 << n) - 1][k];
}
```

## 五、复杂度与正确性分析

| 问题 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 旅行商问题 | $O(2^n \cdot n^2)$ | $O(2^n \cdot n)$ |
| 覆盖问题 | $O(n \cdot 2^m \cdot 2^m)$ | $O(n \cdot 2^m)$ |
| 集合划分 | $O(3^n \cdot k)$ | $O(2^n \cdot k)$ |

**正确性保证**：

1. **状态定义正确**：用二进制表示集合状态，覆盖所有可能的情况。

2. **状态转移正确**：枚举所有可能的转移，取最优值或累加方案数。

3. **计算顺序正确**：从后往前计算，保证依赖的状态已经计算过。

## 七、小结

本文介绍了状压 DP 的基本思想和三个经典问题：**旅行商问题**、**覆盖问题** 和 **集合划分问题**。

四个核心要点：

1. 状压 DP 用二进制表示集合状态，适用于元素数量较小（$n \le 20$）的问题。

2. 状态转移时，枚举所有可能的转移，取最优值或累加方案数。

3. 计算顺序从后往前，保证依赖的状态已经计算过。

4. 枚举子集可以用 `for (int T = S; T > 0; T = (T - 1) & S)` 的位运算技巧。
