# ACM 竞赛数学系列博客

面向校 ACM 集训队新人的全数学板块学习材料，涵盖数论、组合数学、博弈论、计算几何、概率与期望、线性代数六大板块。共 103 篇文章，竞赛导向，重推导轻模板。

## 系列风格

- **竞赛导向**：每篇以「为什么学这个」开头，以「什么时候用」收尾
- **重推导轻模板**：思路 > 推导 > 代码；代码只是推导的自然产物
- **新人友好**：引导式语言，手算例子穿插，❓ 标注答疑点
- **练习题不超纲**：推荐题目均带链接，且仅依赖已学知识

## 阅读顺序

推荐按板块顺序阅读：**数论 → 组合数学 → 博弈论 → 概率与期望 → 线性代数 → 计算几何**。

每个板块内部按序号递增阅读——后续篇章依赖前置知识。

## 难度标识

- 🟢 入门：零基础可读
- 🟡 进阶：需要该板块基础篇知识
- 🔴 高阶：需要较完整基础
- 🟣 专题/综合：串联多个知识点
