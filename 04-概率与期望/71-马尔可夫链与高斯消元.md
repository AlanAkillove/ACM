## 引言

在之前的文章中，我们处理过多种带有环的随机过程：
- #67 期望 DP 中，自环通过移项消去
- #68 概率 DP 中，更复杂的环用高斯消元求解
- #69 吸收马尔可夫链中，通过矩阵求逆一步到位

当一个马尔可夫链不是吸收的（或虽然有吸收态但数量很多），并且状态图中有复杂的环结构时，高斯消元是最通用的解法。

但这里有一个关键问题：**对于一般马尔可夫链（不一定有吸收态），我们关心的是什么？**

答案通常是**平稳分布 (Stationary Distribution)**——链运行足够长时间后，处于各个状态的概率分布。

对于有限状态、不可约、非周期的马尔可夫链，平稳分布存在且唯一。它可以通过求解线性方程组得到——这正是高斯消元的用武之地。

此外，对于带环的期望/概率计算，同样需要对每个状态列写出线性方程，然后一并求解。这些方程往往具有稀疏结构，可以用更高效的方法来处理。

## 1. 一般马尔可夫链的基本概念

### 1.1 定义回顾

一个**马尔可夫链**由以下要素定义：

- **状态空间** $S$：有限或可数无限个状态
- **转移矩阵** $P$：$P_{ij} = P(X_{t+1} = j \mid X_t = i)$
- **初始分布** $\pi^{(0)}$：$P(X_0 = i)$

**马尔可夫性**：$P(X_{t+1} = j \mid X_t = i, X_{t-1}, \dots, X_0) = P(X_{t+1} = j \mid X_t = i)$

### 1.2 平稳分布

**平稳分布 (Stationary Distribution)** $\pi$ 是一个概率分布，满足：

$$\pi = \pi P$$

即 $\pi_j = \sum_i \pi_i P_{ij}$ 对所有 $j$ 成立。

**物理意义**：如果链在某一时刻的分布是 $\pi$，那么接下来所有时刻的分布都是 $\pi$——链达到了"统计平衡"。

### 1.3 存在性与唯一性

对于有限状态马尔可夫链，以下条件保证平稳分布存在且唯一：

1. **不可约 (Irreducible)**：从任何状态出发，都能到达任何其他状态
2. **非周期 (Aperiodic)**：每个状态返回自身的步数集合的最大公约数为 1

在这种情况下，链被称为**遍历的 (Ergodic)**，且 $\lim_{t\to\infty} \pi^{(t)} = \pi$ 与初始分布无关。

> **注意**：平稳分布存在时，链不一定是遍历的（例如周期链也有平稳分布，但不收敛到它）。但在竞赛中，我们处理的大多是遍历链。

## 2. 用高斯消元求解平稳分布

### 2.1 方程组

平稳分布满足 $\pi = \pi P$，等价于：

$$\pi_j = \sum_{i} \pi_i P_{ij}, \quad \forall j$$

以及归一化条件 $\sum_i \pi_i = 1$。

写成线性方程组形式：

$$\begin{cases}
\pi_1 = \pi_1 P_{11} + \pi_2 P_{21} + \cdots + \pi_n P_{n1} \\
\pi_2 = \pi_1 P_{12} + \pi_2 P_{22} + \cdots + \pi_n P_{n2} \\
\vdots \\
\pi_n = \pi_1 P_{1n} + \pi_2 P_{2n} + \cdots + \pi_n P_{nn} \\
\pi_1 + \pi_2 + \cdots + \pi_n = 1
\end{cases}$$

移项整理：

$$\begin{cases}
\pi_1(P_{11} - 1) + \pi_2 P_{21} + \cdots + \pi_n P_{n1} = 0 \\
\pi_1 P_{12} + \pi_2(P_{22} - 1) + \cdots + \pi_n P_{n2} = 0 \\
\vdots \\
\pi_1 P_{1n} + \pi_2 P_{2n} + \cdots + \pi_n(P_{nn} - 1) = 0 \\
\pi_1 + \pi_2 + \cdots + \pi_n = 1
\end{cases}$$

前 $n$ 个方程是线性相关的（系数矩阵的秩为 $n-1$），所以我们需要用归一化方程替换掉其中一个方程。

### 2.2 手算小例子

**题目**：考虑一个 3 状态马尔可夫链，转移矩阵为：

$$P = \begin{pmatrix}
0.5 & 0.4 & 0.1 \\
0.2 & 0.6 & 0.2 \\
0.3 & 0.3 & 0.4
\end{pmatrix}$$

求平稳分布。

**解答**：

设 $\pi = (\pi_1, \pi_2, \pi_3)$。方程组 $\pi = \pi P$：

$$\begin{aligned}
\pi_1 &= 0.5\pi_1 + 0.2\pi_2 + 0.3\pi_3 \\
\pi_2 &= 0.4\pi_1 + 0.6\pi_2 + 0.3\pi_3 \\
\pi_3 &= 0.1\pi_1 + 0.2\pi_2 + 0.4\pi_3
\end{aligned}$$

移项（移项后注意符号）：

$$\begin{aligned}
0.5\pi_1 - 0.2\pi_2 - 0.3\pi_3 &= 0 \\
-0.4\pi_1 + 0.4\pi_2 - 0.3\pi_3 &= 0 \\
-0.1\pi_1 - 0.2\pi_2 + 0.6\pi_3 &= 0
\end{aligned}$$

这三个方程线性相关（和为 0），用归一化方程 $\pi_1 + \pi_2 + \pi_3 = 1$ 替换其中一个：

$$\begin{aligned}
-0.4\pi_1 + 0.4\pi_2 - 0.3\pi_3 &= 0 \\
-0.1\pi_1 - 0.2\pi_2 + 0.6\pi_3 &= 0 \\
\pi_1 + \pi_2 + \pi_3 &= 1
\end{aligned}$$

从第一个方程：$\pi_2 = \pi_1 + 0.75\pi_3$

从归一化方程：$\pi_1 + (\pi_1 + 0.75\pi_3) + \pi_3 = 1 \implies 2\pi_1 + 1.75\pi_3 = 1$

代入第二个方程：

$$-0.1\pi_1 - 0.2(\pi_1 + 0.75\pi_3) + 0.6\pi_3 = 0$$
$$-0.1\pi_1 - 0.2\pi_1 - 0.15\pi_3 + 0.6\pi_3 = 0$$
$$-0.3\pi_1 + 0.45\pi_3 = 0$$
$$\pi_3 = \frac{2}{3}\pi_1$$

代入 $2\pi_1 + 1.75\pi_3 = 1$：

$$2\pi_1 + 1.75 \cdot \frac{2}{3}\pi_1 = 1$$
$$2\pi_1 + \frac{3.5}{3}\pi_1 = 1$$
$$\frac{6 + 3.5}{3}\pi_1 = 1$$
$$\frac{9.5}{3}\pi_1 = 1$$
$$\pi_1 = \frac{3}{9.5} = \frac{6}{19} \approx 0.316$$

$$\pi_3 = \frac{2}{3} \cdot \frac{6}{19} = \frac{4}{19} \approx 0.211$$
$$\pi_2 = \pi_1 + \frac{3}{4}\pi_3 = \frac{6}{19} + \frac{3}{4} \cdot \frac{4}{19} = \frac{6}{19} + \frac{3}{19} = \frac{9}{19} \approx 0.474$$

**验证**：$\pi_1 + \pi_2 + \pi_3 = \frac{6+9+4}{19} = 1$。且 $\pi P = \pi$：

$$\begin{pmatrix}\frac{6}{19} & \frac{9}{19} & \frac{4}{19}\end{pmatrix}
\begin{pmatrix}
0.5 & 0.4 & 0.1 \\
0.2 & 0.6 & 0.2 \\
0.3 & 0.3 & 0.4
\end{pmatrix}
= \begin{pmatrix}
\frac{3}{19}+\frac{1.8}{19}+\frac{1.2}{19} & \frac{2.4}{19}+\frac{5.4}{19}+\frac{1.2}{19} & \frac{0.6}{19}+\frac{1.8}{19}+\frac{1.6}{19}
\end{pmatrix}
= \begin{pmatrix}
\frac{6}{19} & \frac{9}{19} & \frac{4}{19}
\end{pmatrix}$$

正确！

❓ **为什么前 n 个方程线性相关？**

> $\pi = \pi P$ 意味着 $\pi(I-P) = 0$。矩阵 $I-P$ 的每列之和为 $1 - \sum_i P_{ij} = 1 - 1 = 0$，所以列向量线性相关。换言之，$I-P$ 是不可逆的（零空间至少一维）。因此我们需要用归一化条件 $\sum \pi_i = 1$ 来"唯一确定"解。这是平稳分布理论的固有特征——方程组本身不唯一，但归一化后唯一。

## 3. 带环的期望 DP 与高斯消元

除了平稳分布，高斯消元更常见的用途是求解带有一般环（不一定是自环）的期望 DP 和概率 DP。

### 3.1 问题模型

对于期望 DP，方程的一般形式是：

$$\mathbb{E}[i] = c_i + \sum_j P_{ij} \cdot \mathbb{E}[j]$$

其中 $c_i$ 是从状态 $i$ 出发一步的代价（可能是 1 或其他值）。

对于概率 DP，方程的一般形式是：

$$P[i] = \sum_j P_{ij} \cdot P[j]$$

加上边界条件 $P[目标] = 1$。

将方程整理成标准形式后，对每个状态 $i$ 列出一个线性方程，然后用高斯消元求解 $n$ 个未知数。

### 3.2 手算小例子

**题目**：在一个 $2 \times 2$ 的网格上，从 $(1,1)$ 出发，每一步等概率选择上下左右四个方向之一移动。如果移动会碰壁（出界），则留在原地。求到达 $(2,2)$ 的期望步数。

**分析**：

状态有 4 个：$(1,1), (1,2), (2,1), (2,2)$。$(2,2)$ 是目标态，$\mathbb{E}[2,2] = 0$。

对于 $(1,1)$：
- 向右到 $(1,2)$：概率 $1/4$
- 向下到 $(2,1)$：概率 $1/4$
- 向上/向左碰壁，留在 $(1,1)$：概率 $2/4$

所以：

$$\mathbb{E}[1,1] = 1 + \frac{1}{4}\mathbb{E}[1,2] + \frac{1}{4}\mathbb{E}[2,1] + \frac{2}{4}\mathbb{E}[1,1]$$

移项：

$$\frac{2}{4}\mathbb{E}[1,1] - \frac{1}{4}\mathbb{E}[1,2] - \frac{1}{4}\mathbb{E}[2,1] = 1$$

乘以 4：

$$2\mathbb{E}[1,1] - \mathbb{E}[1,2] - \mathbb{E}[2,1] = 4$$

对于 $(1,2)$：
- 向左到 $(1,1)$：$1/4$
- 向下到 $(2,2)$：$1/4$（目标！期望为 0）
- 向上碰壁：$1/4$，留在 $(1,2)$
- 向右碰壁：$1/4$，留在 $(1,2)$

$$\mathbb{E}[1,2] = 1 + \frac{1}{4}\mathbb{E}[1,1] + \frac{1}{4}\cdot 0 + \frac{2}{4}\mathbb{E}[1,2]$$

移项：

$$-\frac{1}{4}\mathbb{E}[1,1] + \frac{2}{4}\mathbb{E}[1,2] = 1$$

乘以 4：

$$-\mathbb{E}[1,1] + 2\mathbb{E}[1,2] = 4$$

对于 $(2,1)$：
- 向上到 $(1,1)$：$1/4$
- 向右到 $(2,2)$：$1/4$（目标）
- 向下碰壁：$1/4$，留在 $(2,1)$
- 向左碰壁：$1/4$，留在 $(2,1)$

$$\mathbb{E}[2,1] = 1 + \frac{1}{4}\mathbb{E}[1,1] + \frac{1}{4}\cdot 0 + \frac{2}{4}\mathbb{E}[2,1]$$

和 $(1,2)$ 对称，同样得到：

$$-\mathbb{E}[1,1] + 2\mathbb{E}[2,1] = 4$$

解方程组：

$$\begin{cases}
2\mathbb{E}[1,1] - \mathbb{E}[1,2] - \mathbb{E}[2,1] = 4 \\
-\mathbb{E}[1,1] + 2\mathbb{E}[1,2] = 4 \\
-\mathbb{E}[1,1] + 2\mathbb{E}[2,1] = 4
\end{cases}$$

由后两个方程：$\mathbb{E}[1,2] = \mathbb{E}[2,1] = \frac{4 + \mathbb{E}[1,1]}{2}$

代入第一个方程：

$$2\mathbb{E}[1,1] - \frac{4 + \mathbb{E}[1,1]}{2} - \frac{4 + \mathbb{E}[1,1]}{2} = 4$$

$$2\mathbb{E}[1,1] - (4 + \mathbb{E}[1,1]) = 4$$

$$\mathbb{E}[1,1] - 4 = 4$$

$$\mathbb{E}[1,1] = 8$$

因此从 $(1,1)$ 出发到达 $(2,2)$ 的期望步数是 8。

$$\mathbb{E}[1,2] = \mathbb{E}[2,1] = \frac{4 + 8}{2} = 6$$

直观上合理——角上的状态更容易被"卡住"，所以期望步数最大。

❓ **为什么不用吸收马尔可夫链的公式？**

> 这个例子本质上就是一个吸收马尔可夫链（$(2,2)$ 是吸收态）。用标准型方法同样可以求解。但由于 $n=4$ 很小，手动列方程和高斯消元更直接。当 $n$ 很大（几百）时，矩阵求逆 $O(n^3)$ 和高斯消元 $O(n^3)$ 复杂度相同。关键是要根据问题特征选择合适的方法——对于结构化的链（如三对角），用 Thomas 算法；对于一般的稠密链，用高斯消元或 LU 分解。

## 4. 🏆 洛谷 P6128 题解

### 4.1 题目描述

**[JSOI2012] 蓝桥杯迷宫**：有一个 $n$ 个点 $m$ 条边的有向图，从点 $s$ 出发。每次等概率选择当前点的一条出边移动。求到达点 $t$ 的期望步数。

### 4.2 思路分析

这是最经典的应用——有向图上从起点到终点的期望步数。

对每个点 $i$ 设 $\mathbb{E}[i]$ 为从 $i$ 出发到达 $t$ 的期望步数。转移方程：

$$\mathbb{E}[i] = 1 + \frac{1}{deg_i} \sum_{j \in out(i)} \mathbb{E}[j], \quad i \neq t$$

$$\mathbb{E}[t] = 0$$

整理为：

$$deg_i \cdot \mathbb{E}[i] - \sum_{j \in out(i)} \mathbb{E}[j] = deg_i$$

令 $n$ 个点形成 $n$ 个方程，用高斯消元求解。

### 4.3 代码实现

```cpp
#include <bits/stdc++.h>
using ll = long long;

// 高斯消元求解线性方程组 Ax = b
// A 是 n x n 矩阵，b 是 n 维向量
// 返回 x (n 维向量)
// 采用列主元消去法，支持 double 类型
std::vector<double> gauss(std::vector<std::vector<double>> A, 
                          std::vector<double> b) {
    int n = A.size();
    // 构造增广矩阵
    for (int i = 0; i < n; ++i) {
        A[i].push_back(b[i]);
    }
    
    for (int col = 0; col < n; ++col) {
        // 列主元：找到绝对值最大的元素所在行
        int pivot = col;
        for (int row = col + 1; row < n; ++row) {
            if (std::abs(A[row][col]) > std::abs(A[pivot][col])) {
                pivot = row;
            }
        }
        if (pivot != col) {
            std::swap(A[col], A[pivot]);
        }
        
        // 检查是否奇异（在概率问题中通常不会发生）
        if (std::abs(A[col][col]) < 1e-12) continue;
        
        // 归一化主元行
        double div = A[col][col];
        for (int j = col; j <= n; ++j) {
            A[col][j] /= div;
        }
        
        // 消去其他行的第 col 列
        for (int row = 0; row < n; ++row) {
            if (row == col) continue;
            double factor = A[row][col];
            if (std::abs(factor) < 1e-12) continue;
            for (int j = col; j <= n; ++j) {
                A[row][j] -= factor * A[col][j];
            }
        }
    }
    
    // 提取解
    std::vector<double> x(n);
    for (int i = 0; i < n; ++i) {
        x[i] = A[i][n];
    }
    return x;
}

// 有向图上从起点到终点的期望步数
// n: 顶点数, edges: 有向边列表 (1-based)
// s: 起点, t: 终点
double expected_steps(int n, const std::vector<std::pair<int,int>>& edges, 
                      int s, int t) {
    // 构建图的出边
    std::vector<std::vector<int>> out(n);
    for (auto [u, v] : edges) {
        out[u - 1].push_back(v - 1);  // 转为 0-based
    }
    
    // 构建线性方程组
    // E[i] = 1 + (1/deg[i]) * sum E[neighbor]
    // => deg[i] * E[i] - sum E[neighbor] = deg[i]
    // 边界: E[t] = 0
    
    std::vector<std::vector<double>> A(n, std::vector<double>(n, 0.0));
    std::vector<double> b(n, 0.0);
    
    for (int i = 0; i < n; ++i) {
        if (i == t - 1) {
            // 终点：E[t] = 0
            A[i][i] = 1.0;
            b[i] = 0.0;
        } else {
            int deg = out[i].size();
            if (deg == 0) {
                // 从该点出发没有出边（且不是终点）：无法到达
                // 视作 E[i] = INF 或不会发生
                A[i][i] = 1.0;
                b[i] = 1e18;  // 很大的数
            } else {
                A[i][i] = deg;
                for (int v : out[i]) {
                    A[i][v] -= 1.0;
                }
                b[i] = deg;
            }
        }
    }
    
    auto E = gauss(A, b);
    return E[s - 1];
}

int main() {
    // 示例：3 个点的简单图
    // 1 -> 2, 1 -> 3
    // 2 -> 3 (目标)
    // 3 -> 3 (自环，已经在目标)
    int n = 3;
    std::vector<std::pair<int,int>> edges = {
        {1, 2}, {1, 3}, {2, 3}
    };
    
    double ans = expected_steps(n, edges, 1, 3);
    std::cout << "从节点1到节点3的期望步数: " << ans << std::endl;
    
    // 手动验算：
    // E[3] = 0
    // E[2] = 1 + E[3] = 1
    // E[1] = 1 + (E[2] + E[3]) / 2 = 1 + (1 + 0) / 2 = 1.5
    // 输出应为 1.5
    
    return 0;
}
```

## 5. 高斯消元的优化技巧

### 5.1 稀疏矩阵优化

在竞赛中，转移矩阵 $I-P$ 通常是**稀疏的**（每个状态只连接少数其他状态）。直接用 $O(n^3)$ 的高斯消元可能过慢。

常用优化：

1. **三对角矩阵**（CF 24D 型）：$O(n)$ Thomas 算法
2. **带状矩阵**（状态按某种顺序排列，只有对角附近非零）：带状高斯消元 $O(n \cdot k^2)$，其中 $k$ 是带宽
3. **迭代法**：对于一些特定问题，可以用高斯-赛德尔迭代（Gauss-Seidel）或值迭代（Value Iteration）来近似求解

### 5.2 利用问题结构

很多竞赛问题不会让你直接套高斯消元模板，而是需要你**发现特殊结构**来化简。

常见技巧：
- 将状态分组，组内先消元，再处理组间关系
- 利用对称性减少状态数量
- 找到不需要高斯消元的递推关系（如赌徒破产问题中的二阶递推）

### 5.3 精度问题

高斯消元在 double 类型下会有浮点误差。当 $n$ 较大（几百）时，误差可能累积。

建议：
- 使用 `long double` 增加精度
- 列主元（Partial Pivoting）是必须的
- 如果答案需要输出分数，可以使用有理数类（在模质数意义下用逆元计算）

## 7. 小结

本文介绍了马尔可夫链与高斯消元的核心结合方式：

- **平稳分布** $\pi = \pi P$ 给出了链的长期行为，通过高斯消元求解线性方程组得到
- **带环期望/概率 DP** 同样化为线性方程组，用高斯消元或特殊结构优化求解
- **关键技巧**：利用归一化条件 $\sum \pi_i = 1$ 消除线性相关性；列主元保证数值稳定性；根据矩阵结构选择优化方法

高斯消元是处理一般马尔可夫链的**通用瑞士军刀**——它不假设特殊结构（如三对角），适用于任何有限状态链。但代价是 $O(n^3)$ 的时间复杂度，所以当 $n$ 较大（超过 500）时，需要寻找更巧妙的解法。

下一篇文章将介绍**期望 Min-Max 容斥**——这是处理"所有事件都发生"类期望问题的最强武器。
