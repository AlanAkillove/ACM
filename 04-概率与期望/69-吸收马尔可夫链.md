# 吸收马尔可夫链

> Made by Aki.
> 最后更新于 2026.04.30

## 引言

之前在赌徒破产问题中，我们遇到了"一旦到达某个状态就永远停在那里"的情况——赌徒输光或赢够钱后游戏结束。在概率 DP 中，我们曾提到过"吸收态"的概念。

当一个随机过程中存在**吸收态**（一旦进入就无法离开的状态），并且从任意非吸收态出发最终都会进入某个吸收态，这样的马尔可夫链称为**吸收马尔可夫链 (Absorbing Markov Chain)**。

吸收马尔可夫链是一类极为重要的模型，因为它精确描述了大量竞赛场景：
- 随机游走直到被某个边界吸收
- 回合制游戏直到分出胜负
- 模式匹配问题直到某个模式出现

更棒的是，吸收马尔可夫链有一套**完整的解析框架**，可以一次性求出：
1. 从每个状态出发，最终被某个吸收态吸收的概率
2. 从每个状态出发，被吸收所需的期望步数

这套框架的核心是**标准型分解 (Canonical Form)** 和**基本矩阵 (Fundamental Matrix)**。

---

## 1. 吸收马尔可夫链的定义

### 1.1 什么是吸收马尔可夫链？

一个马尔可夫链称为**吸收的**，当且仅当：

1. 存在至少一个**吸收态**：一旦进入该状态，就永远停留在那里（即 $p_{ii} = 1$）
2. 从**任意非吸收态**出发，最终都能到达某个吸收态（概率为 1）

> 注意第二个条件排除了"永远在非吸收态之间来回游荡"的情况。在实际竞赛问题中，这通常自动成立。

### 1.2 标准型 (Canonical Form)

将 $m$ 个吸收态和 $n$ 个非吸收态（暂态）分别编号。通常我们把暂态排在前面，吸收态排在后面。那么转移概率矩阵 $P$ 可以写成如下**标准型**：

$$P = \begin{pmatrix}
Q & R \\
0 & I
\end{pmatrix}$$

其中：
- $Q$：$n \times n$ 矩阵，暂态到暂态的转移概率
- $R$：$n \times m$ 矩阵，暂态到吸收态的转移概率
- $0$：$m \times n$ 零矩阵（吸收态不回暂态）
- $I$：$m \times m$ 单位矩阵（吸收态的自环）

这个分块形式极其优雅——非吸收态之间的转移放在左上角，被吸收的过程放在右上角，吸收态本身放在右下角。

> **直观理解**：$Q$ 描述的是"在暂态之间如何游荡"，$R$ 描述的是"从哪个暂态进入哪个吸收态"。

## 2. 基本矩阵 N = (I - Q)^(-1)

### 2.1 基本矩阵的定义

在吸收马尔可夫链中，**基本矩阵 (Fundamental Matrix)** 定义为：

$$N = (I - Q)^{-1}$$

**物理意义**：$N_{ij}$ 表示从暂态 $i$ 出发，在进入吸收态之前**期望访问暂态 $j$ 的次数**。

为什么是 $(I-Q)^{-1}$？我们推导一下。

设 $E_{ij}$ 为从 $i$ 出发到达吸收态前访问 $j$ 的期望次数。首次转移后：
- 以概率 $Q_{ik}$ 进入暂态 $k$，然后访问 $j$ 的次数为 $E_{kj}$（加上当前可能在 $j$ 的情况）
- 以概率 $R_{ik}$ 进入吸收态，访问 $j$ 的期望次数取决于是否 $i=j$

规整一下：

$$E_{ij} = \delta_{ij} + \sum_{k} Q_{ik} \cdot E_{kj}$$

其中 $\delta_{ij}$ 是克罗内克 $\delta$（如果 $i=j$ 则 $\delta_{ij}=1$，否则为 $0$）。

写成矩阵形式：

$$E = I + QE$$

移项：

$$E - QE = I$$

$$(I - Q)E = I$$

$$E = (I - Q)^{-1} = N$$

### 2.2 手算小例子

**题目**：考虑一个简单的抽卡游戏。有普通卡（状态 1）和稀有卡（状态 2）两种暂态，以及"抽到 SSR"（吸收态 A）和"弃坑"（吸收态 B）两种吸收态。转移概率如下：

从普通卡出发：
- 继续抽到普通卡：概率 0.5
- 抽到稀有卡：概率 0.3
- 抽到 SSR（吸收）：概率 0.1
- 弃坑（吸收）：概率 0.1

从稀有卡出发：
- 回到普通卡：概率 0.2
- 继续抽到稀有卡：概率 0.4
- 抽到 SSR（吸收）：概率 0.3
- 弃坑（吸收）：概率 0.1

**解答**：

暂态有 2 个（普通卡=1，稀有卡=2），吸收态有 2 个（SSR=A，弃坑=B）。

$$Q = \begin{pmatrix}
0.5 & 0.3 \\
0.2 & 0.4
\end{pmatrix}, \quad
R = \begin{pmatrix}
0.1 & 0.1 \\
0.3 & 0.1
\end{pmatrix}$$

计算基本矩阵：

$$I - Q = \begin{pmatrix}
0.5 & -0.3 \\
-0.2 & 0.6
\end{pmatrix}$$

求逆（对于 $2\times2$ 矩阵，$\begin{pmatrix}a & b \\ c & d\end{pmatrix}^{-1} = \frac{1}{ad-bc}\begin{pmatrix}d & -b \\ -c & a\end{pmatrix}$）：

$$\det(I-Q) = 0.5 \times 0.6 - (-0.3) \times (-0.2) = 0.3 - 0.06 = 0.24$$

$$N = (I-Q)^{-1} = \frac{1}{0.24} \begin{pmatrix}
0.6 & 0.3 \\
0.2 & 0.5
\end{pmatrix} = \begin{pmatrix}
2.5 & 1.25 \\
0.833 & 2.083
\end{pmatrix}$$

**验证直观**：从普通卡出发，在吸收前期望访问普通卡 2.5 次，访问稀有卡 1.25 次。这合理——因为经常抽到普通卡，所以会反复回到普通卡状态。

❓ **为什么基本矩阵要求逆而不是其他运算？**

> 基本矩阵的物理意义是"期望访问次数"，它满足 $E = I + QE$ 这个递推关系。$I$ 项表示"当前这一步就在这个状态"，$QE$ 项表示"从下一步开始期望访问的次数"。解这个矩阵方程自然得到 $E = (I-Q)^{-1}$。你可以把它看作期望 DP 方程的矩阵版本。

## 3. 期望吸收时间 t = N * 1

### 3.1 公式推导

设 $t_i$ 为从暂态 $i$ 出发，被某个吸收态吸收所需的期望步数。令 $\mathbf{t}$ 为 $n$ 维列向量。

从 $i$ 出发走一步，然后进入下一步的状态：
- 以概率 $Q_{ij}$ 进入暂态 $j$，还需期望 $t_j$ 步
- 以概率 $R_{ik}$ 进入吸收态，还需 0 步

所以：

$$t_i = 1 + \sum_{j=1}^n Q_{ij} \cdot t_j$$

注意那个"1"——从 $i$ 出发走一步就要计数一步。

写成矩阵形式：

$$\mathbf{t} = \mathbf{1} + Q\mathbf{t}$$

其中 $\mathbf{1}$ 是全部为 1 的 $n$ 维列向量。

移项：

$$(I - Q)\mathbf{t} = \mathbf{1}$$

$$\mathbf{t} = (I - Q)^{-1} \mathbf{1} = N \cdot \mathbf{1}$$

**简洁优美**：基本矩阵 $N$ 的第 $i$ 行和，就是从状态 $i$ 出发的期望吸收步数。

### 3.2 手算小例子

接上面的抽卡例子：

$$\mathbf{t} = N \cdot \mathbf{1} = \begin{pmatrix}
2.5 & 1.25 \\
0.833 & 2.083
\end{pmatrix} \begin{pmatrix}
1 \\ 1
\end{pmatrix} = \begin{pmatrix}
3.75 \\ 2.917
\end{pmatrix}$$

所以从普通卡出发，期望需要 3.75 步进入吸收态（要么抽到 SSR，要么弃坑）；从稀有卡出发，期望需要 2.917 步。

**验算**：从稀有卡出发期望步数更少——因为稀有卡抽到 SSR 的概率（0.3）比普通卡（0.1）大得多。

❓ **为什么 $N \cdot \mathbf{1}$ 刚好是期望步数，而不是别的什么？**

> $N_{ij}$ 是从 $i$ 出发期望访问 $j$ 的次数。每次访问 $j$ 都会产生一步（除了第一次从 $i$ 出发不计访问 $i$ 的第一次——但 $N_{ii}$ 已经把这一步算进去了，因为基本矩阵的公式中包含了 $\delta_{ij}$ 项）。所以对 $j$ 求和就是期望的总步数。这个解释更直观——你在各暂态之间跳来跳去，每次跳都是一步，总步数就是所有访问次数的总和。

## 4. 吸收概率 B = N * R

### 4.1 公式推导

设 $B_{ik}$ 为从暂态 $i$ 出发，最终被吸收态 $k$ 吸收的概率。令 $B$ 为 $n \times m$ 矩阵。

首次转移后：
- 以概率 $Q_{ij}$ 进入暂态 $j$，然后被吸收态 $k$ 吸收的概率为 $B_{jk}$
- 以概率 $R_{ik}$ **直接**进入吸收态 $k$

所以：

$$B_{ik} = R_{ik} + \sum_{j} Q_{ij} \cdot B_{jk}$$

写成矩阵形式：

$$B = R + Q B$$

移项：

$$B - Q B = R$$

$$(I - Q) B = R$$

$$B = (I - Q)^{-1} R = N \cdot R$$

**物理意义也很清晰**：$N_{ij}$ 是从 $i$ 出发期望访问 $j$ 的次数，每次访问 $j$ 时有 $R_{jk}$ 的概率从 $j$ 进入吸收态 $k$。所以对 $j$ 求和就是总吸收概率。

### 4.2 手算小例子

继续抽卡例子：

$$B = N \cdot R = \begin{pmatrix}
2.5 & 1.25 \\
0.833 & 2.083
\end{pmatrix} \begin{pmatrix}
0.1 & 0.1 \\
0.3 & 0.1
\end{pmatrix}$$

先算第一行：
- $B_{1A} = 2.5 \times 0.1 + 1.25 \times 0.3 = 0.25 + 0.375 = 0.625$
- $B_{1B} = 2.5 \times 0.1 + 1.25 \times 0.1 = 0.25 + 0.125 = 0.375$

第二行：
- $B_{2A} = 0.833 \times 0.1 + 2.083 \times 0.3 = 0.0833 + 0.625 = 0.708$
- $B_{2B} = 0.833 \times 0.1 + 2.083 \times 0.1 = 0.0833 + 0.2083 = 0.292$

所以：

$$B = \begin{pmatrix}
0.625 & 0.375 \\
0.708 & 0.292
\end{pmatrix}$$

验证每行和为 1：$0.625 + 0.375 = 1$，$0.708 + 0.292 = 1$——从任何暂态出发，最终必然进入某个吸收态。

从普通卡出发，有 62.5% 的概率抽到 SSR，37.5% 的概率弃坑。从稀有卡出发，有 70.8% 的概率抽到 SSR。

❓ **每行之和一定为 1 吗？**

> 是的！吸收马尔可夫链的定义要求从任何暂态出发，最终必然被某个吸收态吸收。所以 $B$ 矩阵的每一行之和都是 1。这是一个很好的正确性检查——如果计算发现某行之和不等于 1（在浮点误差范围内），说明要么 $Q$ 某行之和不等于 $1-R$ 的行和（概率没归一化），要么链不是吸收的。

## 5. 完整的三步公式总结

吸收马尔可夫链的三个核心公式：

| 公式 | 含义 | 维度 |
|------|------|------|
| $N = (I - Q)^{-1}$ | 基本矩阵，期望访问次数 | $n \times n$ |
| $\mathbf{t} = N \cdot \mathbf{1}$ | 期望吸收时间 | $n \times 1$ |
| $B = N \cdot R$ | 吸收概率 | $n \times m$ |

其中 $n$ 是暂态数，$m$ 是吸收态数。

> **记忆方法**：先求基本矩阵 $N$，然后 $\mathbf{t}$ 是 $N$ 的行和，$B$ 是 $N$ 乘以 $R$。

## 6. 竞赛中的应用

### 6.1 赌徒破产问题回顾

在 #66 中我们用递推公式求解了赌徒破产问题。现在用吸收马尔可夫链的框架再来看它。

赌徒有 $0,1,2,\dots,n$ 共 $n+1$ 个状态，其中 $0$ 和 $n$ 是吸收态（输光 / 赢够）。对于 $1 \le i \le n-1$：

$$P_{i,i+1} = p, \quad P_{i,i-1} = 1-p$$

标准型中，暂态为 $1,2,\dots,n-1$，吸收态为 $0$ 和 $n$。

$Q$ 是 $(n-1) \times (n-1)$ 三对角矩阵：

$$Q = \begin{pmatrix}
0 & p & 0 & \dots & 0 \\
1-p & 0 & p & \dots & 0 \\
0 & 1-p & 0 & \dots & 0 \\
\vdots & \vdots & \vdots & \ddots & \vdots \\
0 & 0 & \dots & 1-p & 0
\end{pmatrix}$$

$R$ 是 $(n-1) \times 2$ 矩阵：
- 从状态 $1$ 到吸收态 $0$ 的概率为 $1-p$
- 从状态 $n-1$ 到吸收态 $n$ 的概率为 $p$
- 其余为 0

然后 $B = NR$ 给出的吸收概率和在 #66 中推导的 $P_i = \frac{1 - (q/p)^i}{1 - (q/p)^n}$ 完全一致。

不过对于一维随机游走这种结构良好的问题，用递推公式更高效（$O(n)$）；对于一般图上的吸收马尔可夫链，才需要用矩阵求逆（$O(n^3)$）。

### 6.2 🏆 Codeforces 24D - Broken Robot

**题目**：一个 $n \times m$ 的棋盘，一个机器人在 $(x, y)$。每一步等概率选择：向左、向右、向下、不动（如果在边界上则不能向边界外移动）。问机器人到达最后一行的期望步数。

**分析**：

每一行可以建模为一个吸收马尔可夫链——当前行的下一行是吸收态。但巧妙的是，由于机器人只能向下（不能向上），行之间是单向的，所以我们可以逐行处理。

对于第 $i$ 行（$1 \le i < n$），$m$ 个列位置是暂态，第 $i+1$ 行是吸收态。

对于中间列 $2 \le j \le m-1$：

$$E_{i,j} = 1 + \frac{1}{4}(E_{i,j-1} + E_{i,j} + E_{i,j+1} + E_{i+1,j})$$

移项：

$$4E_{i,j} - E_{i,j-1} - E_{i,j} - E_{i,j+1} = 4 + E_{i+1,j}$$

$$3E_{i,j} - E_{i,j-1} - E_{i,j+1} = 4 + E_{i+1,j}$$

这个方程对 $j$ 形成三对角线性方程组，可以用 Thomas 算法（追赶法）在 $O(m)$ 时间内求解。

对于边界列需要单独处理（能动方向更少），但整体思路一致。

这个问题虽然不直接调用矩阵求逆，但其本质就是吸收马尔可夫链的框架——每行是一个子链，$Q$ 矩阵对应三对角结构，用追赶法代替通用求逆来达到 $O(m)$ 的效率。

**代码实现**：

```cpp
#include <bits/stdc++.h>
using ll = long long;

// CF 24D: Broken Robot
// n 行 m 列的棋盘，机器人从 (x, y) 出发，求到达第 n 行的期望步数
// 使用逐行高斯消元（三对角矩阵的 Thomas 算法）

void solve() {
    int n, m, x, y;
    std::cin >> n >> m >> x >> y;
    
    if (m == 1) {
        // 特殊处理：只有一列时，只能向下或不动
        // 需要期望 (n - x) * 2 步（因为每次向下的概率是 1/2）
        double ans = 2.0 * (n - x);
        std::cout << std::fixed << std::setprecision(10) << ans << std::endl;
        return;
    }
    
    // dp[i][j] = 从 (i, j) 出发到达第 n 行的期望步数
    // dp[n][j] = 0（已经在最后一行）
    std::vector<double> dp(m + 2, 0.0);
    
    // 从倒数第二行向上递推
    for (int i = n - 1; i >= x; --i) {
        // 需要解线性方程组:
        // 对于 j=1:   2*dp[1] - dp[2] = 3 + dp_next[1]
        // 对于 2<=j<=m-1: 3*dp[j] - dp[j-1] - dp[j+1] = 4 + dp_next[j]
        // 对于 j=m:   2*dp[m] - dp[m-1] = 3 + dp_next[m]
        //
        // 这里 dp_next 是下一行的期望（已知）
        // dp 是当前行的期望（未知）
        // 使用 Thomas 算法求解三对角系统
        
        // 构建三对角矩阵系数
        std::vector<double> a(m + 1, 0.0);  // 下对角线
        std::vector<double> b(m + 1, 0.0);  // 主对角线
        std::vector<double> c(m + 1, 0.0);  // 上对角线
        std::vector<double> d(m + 1, 0.0);  // 右端项
        
        // 设置系数
        b[1] = 2.0;  c[1] = -1.0;  
        d[1] = 3.0 + dp[1];  // 这里 dp 向量存储的是下一行的值
        
        for (int j = 2; j < m; ++j) {
            a[j] = -1.0;  b[j] = 3.0;  c[j] = -1.0;
            d[j] = 4.0 + dp[j];
        }
        
        a[m] = -1.0;  b[m] = 2.0;
        d[m] = 3.0 + dp[m];
        
        // Thomas 算法（追赶法）求解
        // 第一步：消去下对角线
        for (int j = 2; j <= m; ++j) {
            double w = a[j] / b[j-1];
            b[j] -= w * c[j-1];
            d[j] -= w * d[j-1];
        }
        
        // 第二步：回代
        dp[m] = d[m] / b[m];
        for (int j = m - 1; j >= 1; --j) {
            dp[j] = (d[j] - c[j] * dp[j+1]) / b[j];
        }
    }
    
    std::cout << std::fixed << std::setprecision(10) << dp[y] << std::endl;
}

int main() {
    solve();
    return 0;
}
```

## 8. 小结

吸收马尔可夫链提供了分析"随机过程直到终止"这类问题的完整工具箱：

- **标准型分解** $P = \begin{pmatrix} Q & R \\ 0 & I \end{pmatrix}$ 清晰分离了暂态和吸收态
- **基本矩阵** $N = (I-Q)^{-1}$ 是理解一切的关键桥梁
- **期望吸收时间** $\mathbf{t} = N \cdot \mathbf{1}$ 只需一行求和
- **吸收概率** $B = N \cdot R$ 给出被每个吸收态吸收的概率

这套框架的美妙之处在于：无论链有多复杂（只要它是吸收的），三个公式一次到位，无需逐个状态推导。实际竞赛中，如果 $n$ 较小（$\le 200$）可以直接矩阵求逆；如果结构特殊（如三对角），可以用追赶法优化到 $O(n)$；如果 $n$ 很大，则往往需要挖掘问题的特殊结构来简化解法。

吸收马尔可夫链也是理解更一般马尔可夫链的重要跳板。下一篇文章中，我们将介绍**概率生成函数**——另一个强大的解析工具，可以用来推导各种分布的矩和概率。
