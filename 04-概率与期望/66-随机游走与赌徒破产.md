# 随机游走与赌徒破产

> Made by Aki.
> 最后更新于 2026.04.30

## 引言

一个醉汉站在数轴的原点，每步以概率 $p$ 向右走 1 单位，概率 $q = 1-p$ 向左走 1 单位。他这样漫无目的地走了 $n$ 步之后，会在哪里？

另一个场景：两个赌徒 Alice 和 Bob 在赌场对赌。Alice 有 $a$ 元，Bob 有 $b$ 元，每局 Alice 赢的概率为 $p$，输的概率为 $q = 1-p$，输家给赢家 1 元。游戏一直进行，直到一方输光所有钱为止。Alice 最终获胜（赢光 Bob 所有钱）的概率是多少？

这两个问题看似不同，背后却指向同一个数学模型——**随机游走 (Random Walk)**。它是概率论中最优美也最实用的模型之一，在竞赛中频繁出现。

---

## 1. 一维无限制随机游走

### 1.1 基本模型

设 $S_n$ 为 $n$ 步之后的位置，$S_0 = 0$，每步：

$$
X_i = \begin{cases}
+1, & \text{概率 } p \\
-1, & \text{概率 } q = 1-p
\end{cases}
$$

$$S_n = \sum_{i=1}^n X_i$$

### 1.2 期望位置

由线性性质：

$$\mathbb{E}[S_n] = n \cdot \mathbb{E}[X_1] = n(p - q) = n(2p - 1)$$

- 公平情况（$p = 1/2$）：$\mathbb{E}[S_n] = 0$，醉汉平均还在原点附近。
- 向右偏（$p > 1/2$）：期望随 $n$ 线性增长。

### 1.3 方差

$$\text{Var}(S_n) = n \cdot \text{Var}(X_1)$$

$$\text{Var}(X_1) = \mathbb{E}[X_1^2] - (\mathbb{E}[X_1])^2 = 1 - (2p-1)^2 = 4p(1-p)$$

对公平情况：$\text{Var}(S_n) = n$，标准差为 $\sqrt{n}$。

> 这意味着醉汉虽然"平均"在原点，但他实际位置的标准差是 $\sqrt{n}$ 量级的。走 100 步，他大致在 $[-10, 10]$ 范围内。

### 1.4 返回原点的概率

一个经典结果是：公平随机游走在无限步后**几乎必然 (with probability 1)** 会返回原点。但如果是偏斜的（$p \neq q$），则有正概率永远不回来。

❓ **公平随机游走一定会返回原点吗？"

> "几乎必然"返回，但**期望返回时间是无穷大**！也就是说，虽然返回几乎肯定发生，但平均要等待无限长时间。这个反直观的结论是概率论中的经典——"必然发生但期望无穷"。技术上，这用到了 Borel-Cantelli 引理和重对数律 (Law of the Iterated Logarithm)。

## 2. 赌徒破产问题

### 2.1 问题设定

赌徒有初始资金 $i$ 元，目标是到 $N$ 元（赢光庄家或达到目标）。每局赢 $1$ 元的概率为 $p$，输 $1$ 元的概率为 $q=1-p$。问最终破产（资金为 0）的概率是多少？

令 $P_i$ 为从资金 $i$ 出发最终破产的概率。边界条件：$P_0 = 1$（已经破产），$P_N = 0$（达到目标，不破产）。

### 2.2 方程建立

考虑第一局的结果，用全概率公式：

$$P_i = p \cdot P_{i+1} + q \cdot P_{i-1}$$

这是一个二阶线性递推。怎么解？

### 2.3 求解递推

将方程改写为：

$$p(P_{i+1} - P_i) = q(P_i - P_{i-1})$$

令 $d_i = P_i - P_{i-1}$，则：

$$d_{i+1} = \frac{q}{p} d_i$$

所以 $d_i$ 是等比数列：$d_i = \left(\frac{q}{p}\right)^{i-1} d_1$。

现在利用 $P_N = 0$ 和 $P_0 = 1$ 来确定 $d_1$：

$$
P_N - P_0 = \sum_{i=1}^N d_i = d_1 \sum_{i=1}^N \left(\frac{q}{p}\right)^{i-1}
$$

### 2.4 破产概率闭公式

当 $p \neq q$（不公平硬币）时：

$$P_i = \frac{\left(\frac{q}{p}\right)^i - \left(\frac{q}{p}\right)^N}{1 - \left(\frac{q}{p}\right)^N}$$

当 $p = q = 1/2$（公平硬币）时：

$$P_i = 1 - \frac{i}{N}$$

### 2.5 手算小例子

**问题**：Alice 有 5 元，Bob 有 10 元，每局 Alice 赢的概率 $p = 0.4$，游戏到一方输光为止。问 Alice 赢光 Bob 的概率是多少？

**解答**：

这里 Alice 初始资金 $i = 5$，总资金 $N = 15$，$q/p = 0.6/0.4 = 1.5$。

$$
P_{\text{Alice破产}} = \frac{1.5^5 - 1.5^{15}}{1 - 1.5^{15}}
$$

计算一下：
- $1.5^5 \approx 7.59$
- $1.5^{15} \approx 437.89$

$$
P_{\text{Alice破产}} \approx \frac{7.59 - 437.89}{1 - 437.89} = \frac{-430.30}{-436.89} \approx 0.985
$$

所以 Alice 破产的概率约为 98.5%，获胜概率仅 1.5%。

**直观分析**：Alice 每局赢的概率只有 0.4，本金（5）也比 Bob（10）少得多。两个不利因素叠加，获胜希望渺茫。

### 2.6 无限赌本的情况

如果对方有无限赌本（$N \to \infty$），即赌场模式：

- 若 $p \le q$：破产概率为 $1$（必然破产）
- 若 $p > q$：破产概率为 $\left(\frac{q}{p}\right)^i$

这就是"久赌必输"的数学原理——即使游戏公平（$p = 0.5$），只要对方有无限资金，你的破产概率也是 1。

❓ **为什么公平游戏也会必输？**

> 公平游戏中，每次对赌的期望收益为 0。但因为有有限赌本而对手无限，长期来看随机波动迟早会让你碰到"0"这个吸收壁。这是一个随机游走的自然结果——公平游走在两个吸收壁之间最终必然碰到其中一个。

## 3. 二维随机游走

### 3.1 定义

在二维整数格点 $\mathbb{Z}^2$ 上，每步从当前位置等概率地向四个方向之一移动。

### 3.2 Polya 定理

Polya 在 1921 年证明了关于格点随机游走的常返性的著名定理：

- **一维**：公平随机游走是常返的（返回概率 = 1）
- **二维**：公平随机游走是常返的（返回概率 = 1）  
- **三维及以上**：公平随机游走是暂态的（返回概率 < 1）

换句话说，醉汉在一维和二维的街上走，最终总能晃回家；但在三维空间中，他可能永远回不来了。

> 这背后的直觉：维度越高，空间越"大"，游走就越容易漂远不归。

### 3.3 二维返回概率的估计

二维公平游走中，$2n$ 步后返回原点的概率近似为：

$$P(S_{2n} = 0) \approx \frac{1}{\pi n}$$

这和一维的 $\Theta(1/\sqrt{n})$ 不同。由于 $\sum \frac{1}{n}$ 发散，由 Borel-Cantelli 引理可知返回必然发生。

## 4. 竞赛中的相关应用

### 4.1 赌徒破产的直接应用

很多竞赛题可以直接套用赌徒破产的公式。比如：

- 两个玩家对战，每局胜率固定，问一方的获胜概率。
- 股价（或信用分）的上下波动到 0 或目标值。

### 4.2 随机游走与卡特兰数

公平随机游走在 $2n$ 步后回到原点且**从未进入负半轴**的路径数为卡特兰数：

$$C_n = \frac{1}{n+1}\binom{2n}{n}$$

这在括号序列匹配等组合问题中大量出现。

### 4.3 反射原理 (Reflection Principle)

计算随机游走在某个时间点前触及特定边界的路径数时，反射原理是核心工具：

从 $(0, a)$ 到 $(n, b)$ 且**曾触及** $y = 0$ 的路径数，等于从 $(0, -a)$ 到 $(n, b)$ 的路径数。

这个简明的结论可以用来计算各种"首次达到"的概率。

## 5. 代码实现

```cpp
#include <bits/stdc++.h>
using ll = long long;

// 模拟一维随机游走，返回 n 步后的位置
int simulate_random_walk(int n, double p) {
    std::random_device rd;
    std::mt19937 gen(rd());
    std::uniform_real_distribution<double> dis(0.0, 1.0);
    
    int pos = 0;
    for (int i = 0; i < n; ++i) {
        if (dis(gen) < p) pos++;
        else pos--;
    }
    return pos;
}

// 计算赌徒破产概率（闭公式）
// i: 初始资金, N: 目标总资金（i <= N）
// p: 每局赢的概率
double gamblers_ruin_probability(int i, int N, double p) {
    if (p == 0.5) {
        // 公平情况
        return 1.0 - (double)i / N;
    }
    double r = (1.0 - p) / p;  // q/p
    double num = std::pow(r, i) - std::pow(r, N);
    double den = 1.0 - std::pow(r, N);
    return num / den;
}

// 无限赌本情况：从资金 i 出发，对方无限资金
double gamblers_ruin_infinite(int i, double p) {
    if (p <= 0.5) return 1.0;  // 必然破产
    double r = (1.0 - p) / p;
    return std::pow(r, i);
}

int main() {
    // 测试赌徒破产
    int a = 5, b = 10;
    double p = 0.4;
    
    double ruin_prob = gamblers_ruin_probability(a, a + b, p);
    std::cout << "Alice 初始资金: " << a << "元" << std::endl;
    std::cout << "Bob 初始资金: " << b << "元" << std::endl;
    std::cout << "Alice 每局胜率: " << p << std::endl;
    std::cout << "Alice破产概率: " << ruin_prob << std::endl;
    std::cout << "Alice获胜概率: " << 1.0 - ruin_prob << std::endl;
    
    std::cout << std::endl;
    
    // 模拟验证
    int trials = 100000;
    int alice_wins = 0;
    std::random_device rd;
    std::mt19937 gen(rd());
    std::uniform_real_distribution<double> dis(0.0, 1.0);
    
    for (int t = 0; t < trials; ++t) {
        int alice = a, bob = b;
        while (alice > 0 && bob > 0) {
            if (dis(gen) < p) {
                alice++; bob--;
            } else {
                alice--; bob++;
            }
        }
        if (bob == 0) alice_wins++;
    }
    
    std::cout << "模拟 " << trials << " 次:" << std::endl;
    std::cout << "Alice获胜频率: " << (double)alice_wins / trials << std::endl;
    std::cout << "理论概率: " << 1.0 - ruin_prob << std::endl;
    
    return 0;
}
```

## 7. 小结

本文从一维随机游走出发，推导了赌徒破产问题的闭公式，并讨论了二维游走和 Polya 定理：

- **一维随机游走**是概率建模的基础，期望和方差可以轻松得到
- **赌徒破产公式**给出了有限/无限赌本下的必胜/必败概率闭式解
- **Polya 定理**揭示了随维度变化的常返性——维度越高越难回家
- 随机游走与**卡特兰数**、**反射原理**等组合工具交织，形成丰富的解题技巧

赌徒破产公式在竞赛中经常可以直接套用，记下公式能节省大量时间。但理解推导过程更重要——二阶递推的求解技巧在期望 DP 中也会反复出现。
