# Nim 游戏与 Bouton 定理

> Made by Aki.
> 最后更新于 2026.04.30

## 引言

如果说博弈论中只能选一个最重要的模型，那一定是 Nim 游戏。

想象这样的场景：桌上有三堆石子，数量分别为 $3, 4, 5$。你和对手轮流操作，每次可以选择**任意一堆**，从中取走**任意正整数颗**石子（至少 1 颗，也可以把整堆取走）。取走最后一颗的人获胜。

直觉上，这个游戏比巴什博弈和威佐夫博弈都复杂——不仅是多堆，而且每堆可取的数目没有上限只有下限。但令人惊叹的是，这个看似复杂的游戏有一个简洁优美的解法：把所有堆的石子数按二进制做**异或（XOR）**，结果为 0 则先手必败，否则先手必胜。

这个结论被称为 **Bouton 定理**，发表于 1901 年，是博弈论历史上里程碑式的成果。

## 前置知识

- 博弈论基础：N/P 状态判定（#52）
- 二进制与异或运算（XOR）
- 基本的数学归纳法

## Nim 游戏的规则

Nim 游戏的定义非常简单：

- 有 $n$ 堆石子，第 $i$ 堆有 $a_i$ 颗石子
- 两名玩家轮流操作
- 每次操作：选择一堆石子（$a_i > 0$），从中取走任意正整数颗（可全部取走）
- 取走最后一颗石子者获胜

> ❓ 这个游戏跟巴什博弈有什么本质区别？
> 
> 巴什博弈只有一堆，且每次取子数量有上限；Nim 游戏有多堆，每堆的取子数量无上限（但你不能跨堆取）。多堆的存在让游戏变得极为丰富。

## 从特例到一般

### 情况 1：只有一堆石子

$n=1$ 时，显然先手可以把整堆取走，必胜。所以 $a_1 > 0$ 时都是 N 态。

### 情况 2：两堆石子

$n=2$ 时，先手什么时候必败？我们之前在第 52 篇的「对称策略」中提到过：当两堆相等时，后手可以采用模仿策略。所以 $(k,k)$ 是 P 态。

用异或表示就是：$a_1 \oplus a_2 = k \oplus k = 0$ 时必败。

### 情况 3：三堆石子

我们来手算几个小例子。

设 $(a_1,a_2,a_3)$ 分别为三堆的数量。

**例 1**: $(1,1,1)$
- 先手可以取走一整堆（比如取走第一堆），得到 $(0,1,1)$。$(0,1,1)$ 两堆相同，是 P 态。所以 $(1,1,1)$ 是 N 态。
- $1 \oplus 1 \oplus 1 = 1 \ne 0$，预测先手必胜 ✓

**例 2**: $(0,2,2)$
- 两堆相等，是 P 态（对称策略）。
- $0 \oplus 2 \oplus 2 = 0$ ✓

**例 3**: $(1,2,3)$
- 我们来分析：先手可以怎么走？
  - 从第一堆取 1→$(0,2,3)$
  - 从第二堆取 1→$(1,1,3)$，取 2→$(1,0,3)$
  - 从第三堆取 1→$(1,2,2)$，取 2→$(1,2,1)$，取 3→$(1,2,0)$
- 计算异或：$1 \oplus 2 \oplus 3 = 01_2 \oplus 10_2 \oplus 11_2 = 00_2 = 0$。预测是 P 态。
- 验证一下：$(0,2,3)$ 有 $0 \oplus 2 \oplus 3 = 1 \ne 0$（N），$(1,1,3)$ 有 $1 \oplus 1 \oplus 3 = 3 \ne 0$（N），$(1,2,2)$ 有 $1 \oplus 2 \oplus 2 = 1 \ne 0$（N），$(1,2,1)$ 有 $1 \oplus 2 \oplus 1 = 2 \ne 0$（N）。所有后继都是 N，所以 $(1,2,3)$ 是 P 态 ✓

这些例子强烈暗示：**Nim 游戏的 P 态等价于所有堆石子数的异或为 0**。

## Bouton 定理

**定理（Bouton, 1901）**：在 Nim 游戏中，记 $S = a_1 \oplus a_2 \oplus \cdots \oplus a_n$（异或和，也称为 **Nim-sum**）。则当前状态为 P 态（先手必败）当且仅当 $S = 0$。

### 证明

我们需要证明两件事：
1. 从 $S = 0$ 的状态出发，无论如何操作，都会到达 $S \ne 0$ 的状态
2. 从 $S \ne 0$ 的状态出发，**存在**一种操作，可以到达 $S = 0$ 的状态

再结合终止状态 $(0,0,\dots,0)$ 的 $S = 0$ 是 P 态，由归纳法即可得证。

#### 第一部分：证明 $S = 0 \rightarrow$ 所有后继的 $S \ne 0$

假设当前 Nim-sum $S = 0$。我们选择第 $i$ 堆，取走若干石子，使其从 $a_i$ 变为 $a_i'$（$0 \le a_i' < a_i$）。

新的 Nim-sum 为：$S' = S \oplus a_i \oplus a_i' = 0 \oplus a_i \oplus a_i' = a_i \oplus a_i'$。

> ❓ 为什么 $S' = S \oplus a_i \oplus a_i'$？
> 
> 因为 $S = a_1 \oplus \cdots \oplus a_i \oplus \cdots \oplus a_n$，而 $S' = a_1 \oplus \cdots \oplus a_i' \oplus \cdots \oplus a_n$。两式异或得 $S \oplus S' = a_i \oplus a_i'$。又 $S=0$，所以 $S' = a_i \oplus a_i'$。

由于 $a_i' \ne a_i$，所以 $a_i \oplus a_i' \ne 0$，即 $S' \ne 0$。证毕。

这部分的证明很简单。关键在第二部分。

#### 第二部分：证明 $S \ne 0 \rightarrow$ 存在后继使 $S = 0$

假设 $S \ne 0$。设 $d$ 为 $S$ 二进制表示中**最高位** 1 的位置（从低位开始编号为 0）。

比如 $S = 10110_2$，最高位的 1 在第 4 位（$2^4 = 16$）。

接下来我们声称：**存在某一堆 $a_i$，其二进制在第 $d$ 位上也是 1**。

为什么？因为如果所有 $a_i$ 在第 $d$ 位上都是 0，那么 $S$ 在第 $d$ 位上也会是 0（因为 $0 \oplus 0 \oplus \cdots \oplus 0 = 0$），与 $d$ 是 $S$ 的最高位矛盾。

所以我们可以选择这样的一堆 $a_i$，令
$$a_i' = a_i \oplus S$$

我们需要验证 $a_i' < a_i$（即操作合法——只能取走，不能增加）：

由于 $a_i$ 和 $S$ 在第 $d$ 位上都是 1，异或后 $a_i \oplus S$ 在第 $d$ 位上为 0。而 $a_i$ 在第 $d$ 位上是 1，更高位不变（因为 $d$ 是 $S$ 的最高位，$S$ 在 $> d$ 的位置上全是 0），所以 $a_i \oplus S$ 小于 $a_i$（在第 $d$ 位上从 1 变成 0，更高位相同）。

因此操作合法——我们从第 $i$ 堆取走 $a_i - (a_i \oplus S)$ 颗石子即可。

操作后的 Nim-sum：
$$S' = S \oplus a_i \oplus (a_i \oplus S) = (S \oplus a_i \oplus a_i) \oplus S = 0 \oplus S \oplus S = 0$$

证毕。

### 证明的直观理解

第二部分的构造可以理解为这样一句话：

> **从 Nim-sum 最高位为 1 的那一堆中，取走恰好使该堆变成「原石子数异或 Nim-sum」的数量。**

举个例子：
- $(3,4,5)$：$3 \oplus 4 \oplus 5 = 010_2 \oplus 100_2 \oplus 101_2 = 011_2 = 3 \ne 0$，先手必胜
- Nim-sum $S = 3$，二进制为 $011_2$，最高位在第 1 位（$2^1 = 2$）
- 哪些堆在第 1 位上是 1？$3 = 011_2$ 在第 1 位是 1✓，$4 = 100_2$ 在第 1 位是 0✗，$5 = 101_2$ 在第 1 位是 0✗
- 所以选择第一堆 $(3)$：$a_i' = 3 \oplus 3 = 0$，即取走全部 3 颗
- 操作后状态为 $(0,4,5)$：$0 \oplus 4 \oplus 5 = 1 \ne 0$……等等，不对吗？

⚠️ **这里有一个常见的计算错误**：$3 \oplus 4 \oplus 5$ 不是 3！来重新算一下：

$3 = 011_2, 4 = 100_2, 5 = 101_2$。
$4 \oplus 5 = 100_2 \oplus 101_2 = 001_2 = 1$。
$3 \oplus 1 = 011_2 \oplus 001_2 = 010_2 = 2$。

所以 $S = 2$，二进制 $010_2$，最高位在第 1 位（$2^1 = 2$，第 0 位是 $2^0=1$，第 1 位是 $2^1=2$）。

哪些堆在第 1 位是 1？$3=011_2$ 第 1 位是 1✓，$4=100_2$ 第 1 位是 0✗，$5=101_2$ 第 1 位是 0✗。

选第一堆：$a_i' = 3 \oplus 2 = 011_2 \oplus 010_2 = 001_2 = 1$。即从第一堆取 $3-1=2$ 颗，得到 $(1,4,5)$。
验证：$1 \oplus 4 \oplus 5 = 001_2 \oplus 100_2 \oplus 101_2 = 000_2 = 0$ ✓

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    int n;
    cin >> n;
    ll xor_sum = 0;
    for (int i = 0; i < n; i++) {
        ll x;
        cin >> x;
        xor_sum ^= x;
    }
    if (xor_sum == 0) {
        cout << "Second wins!" << endl;
    } else {
        cout << "First wins!" << endl;
    }
    return 0;
}
```

### 构造必胜策略

如果我们需要输出第一步的具体操作（选择哪一堆，取多少颗），可以这样实现：

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    int n;
    cin >> n;
    vector<ll> a(n);
    ll xor_sum = 0;
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        xor_sum ^= a[i];
    }

    if (xor_sum == 0) {
        cout << "Second wins!" << endl;
        return 0;
    }

    cout << "First wins!" << endl;
    // 构造第一步策略
    for (int i = 0; i < n; i++) {
        ll take = a[i] - (a[i] ^ xor_sum);
        if (take > 0 && (a[i] ^ xor_sum) < a[i]) {
            cout << "Take " << take << " from pile " << i + 1 << endl;
            cout << "New state: ";
            for (int j = 0; j < n; j++) {
                if (j == i) cout << (a[j] ^ xor_sum) << " ";
                else cout << a[j] << " ";
            }
            cout << endl;
            break;
        }
    }
    return 0;
}
```

### 手算例子：完整推演

**问题**：三堆石子 $(2, 7, 5)$，先手是否必胜？如果是，请给出第一步策略。

**解**：

**Step 1**: 计算 Nim-sum
$$2 \oplus 7 \oplus 5 = 010_2 \oplus 111_2 \oplus 101_2 = 000_2 = 0$$

Nim-sum 为 0，所以先手必败。

验证一下：我们能否从某堆取石子使新 Nim-sum 为 0？按照定理证明，如果能取到 $a_i' = a_i \oplus S$，其中 $S=0$，则 $a_i' = a_i \oplus 0 = a_i$，即不取——但规则要求至少取 1 颗，所以确实无合法操作可以使 Nim-sum 保持 0。

**问题 2**：四堆石子 $(3, 5, 6, 8)$，先手是否必胜？

**Step 1**: 计算 Nim-sum
$$3 \oplus 5 \oplus 6 \oplus 8 = 011_2 \oplus 101_2 \oplus 110_2 \oplus 1000_2 = 1000_2 = 8 \ne 0$$

先手必胜。

**Step 2**: 找到一种必胜走法
$S = 8 = 1000_2$，最高位在第 3 位。

哪些堆在第 3 位上是 1？$3=0011_2$——第 3 位是 0✗，$5=0101_2$——第 3 位是 0✗，$6=0110_2$——第 3 位是 0✗，$8=1000_2$——第 3 位是 1✓。

选择第 4 堆 $(8)$：$a_i' = 8 \oplus 8 = 0$。即取走全部 8 颗。

操作后：$(3,5,6,0)$，Nim-sum 为 $3 \oplus 5 \oplus 6 = 0$ ✓

之后无论对手怎么走，我们都可以继续按照 Bouton 定理的策略应对。

## Nim 游戏的变体

### 取走最后一颗者输（Misère Nim）

在正常的 Nim 中，取走最后一颗胜。如果改为取走最后一颗输呢？我们会在第 59 篇详细讨论。这里提前剧透结论：除了所有堆都是 1 的情况外，其他情况与正常 Nim 的判定相同。

### Nim 游戏的实际意义

Nim 游戏之所以重要，不仅仅是因为它本身常考，更因为它是 **Sprague-Grundy 定理**的基础。SG 定理告诉我们：**任何 impartial combinatorial game 都可以等价地转化为 Nim 游戏**。每个游戏局面都可以映射为一个 Nim 堆的大小（SG 值），多个游戏的和就是这些 Nim 堆的异或。

这就是为什么 Nim 游戏被称为「博弈论中的牛顿定律」。

## 小结

- **Nim 游戏**是博弈论的基石，规则简单而结论优美
- **Bouton 定理**：Nim-sum（所有堆石子数的异或）为 0 时先手必败，否则先手必胜
- 必胜策略的构造：找到 Nim-sum 最高位为 1 的堆，将该堆石子数变为 $a_i \oplus S$
- 证明的核心：从 $S=0$ 走不到 $S=0$，从 $S\ne 0$ 可以走到 $S=0$
- Nim 游戏的推广奠定了 SG 定理的基础
