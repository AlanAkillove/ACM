# 翻硬币游戏与 Nim 等价性

> Made by Aki.
> 最后更新于 2026.04.30

## 引言

想象一个长条形的桌面，上面摆着一排硬币。每枚硬币要么正面朝上（H），要么反面朝上（T）。每次操作你可以翻转一枚或多枚硬币——但规则对「哪些硬币可以翻」有特定限制。这就是**翻硬币游戏（Coin Turning Game）**，它在组合博弈论中占据着特殊的地位。

为什么重要？因为翻硬币游戏与 Nim 游戏之间存在着惊人的**等价性**：每一个翻硬币游戏都可以转化为一个等效的 Nim 游戏。这意味着你不需要为每种翻硬币规则重新发明一套理论——直接用 Nim 的结论就可以。

这种等价性不仅深刻，而且在竞赛中非常实用：很多看似与硬币无关的问题，经过抽象后都可以归约为翻硬币模型，进而转化为 Nim。

## Turning Turtles 规则

### 基本规则

首先来看一种具体的翻硬币规则——**Turning Turtles**。

> **Turning Turtles 规则**：
> 一排硬币从位置 1 开始编号。每次操作可以选择一枚**正面朝上**的硬币，将其翻转为反面，同时可以（可选地）翻转它左边任意一枚硬币（翻正或翻反都可以）。
>
> 通俗地说：每次操作必须把一枚正面的硬币翻反，然后可以再任意改变它左边某一枚硬币的状态。

游戏结束时，所有硬币都是反面朝上，此时当前玩家无法操作（因为没有正面硬币可以翻），所以输掉游戏。

### 问题：这跟 Nim 有什么关系？

表面上看，这是翻硬币，跟 Nim 的「取石子」八竿子打不着。但我们要揭示的是：**每个正面硬币等价于一个 Nim 堆**。

## 核心推导：等价性定理

### 定理陈述

**翻硬币游戏与 Nim 等价性定理**：
在 Turning Turtles 规则下，将位置 $i$ 上的一枚正面硬币视为一个大小为 $i$ 的 Nim 堆。那么整个游戏局面的 SG 值就等于所有正面硬币对应 Nim 堆大小的异或和。

$$
SG(\text{局面}) = \bigoplus_{\text{位置 } i \text{ 的硬币为正面}} i
$$

当且仅当这个异或和为 $0$ 时，当前局面是必败态。

### 证明思路

证明的核心是：我们需要验证这种「正面→Nim堆」的映射满足 SG 函数的定义。

更确切地说，定义函数 $f$ 将翻硬币局面映射为 Nim 局面：

$$
f(\text{局面}) = \bigoplus_{\text{位置 } i \text{ 为正面}} i
$$

我们需要验证：翻硬币游戏中的一个合法操作，恰好对应 Nim 游戏中的一个合法操作，且两者 SG 值的变化一致。

我们来看一次操作：
1. 选择位置 $i$ 的一枚正面硬币，翻为反面 → 这相当于从异或和中「移除」了 $i$（因为 $x \oplus x = 0$）。
2. 选择左边某个位置 $j < i$，翻转它的状态：
   - 如果 $j$ 原来是正面，翻为反面 → 从异或和中移除 $j$
   - 如果 $j$ 原来是反面，翻为正面 → 向异或和中加入 $j$

所以一次操作对异或和的影响是：$S \to S \oplus i \oplus (\text{如果翻转了 j 则异或 } j)$。

在 Nim 中，从大小为 $i$ 的堆中取石子相当于减少它的值。但这里 $S \oplus i$ 并不是「从堆 $i$ 取石子」的标准形式啊？

等一下——关键来了：

在 Nim 中，从大小为 $i$ 的堆中取到大小为 $j$（其中 $0 \le j < i$），等价于将异或和从 $S$ 变为 $S \oplus i \oplus j$。

而翻硬币游戏中，从 $S$ 变为 $S \oplus i \oplus j$：翻转位置 $i$（正面→反面，移除 $i$），再翻转位置 $j$（状态取反，相当于 $S \oplus i \oplus j$）。

看！变换形式完全一致。所以翻硬币游戏中的操作一一对应于 Nim 中的操作。

> ❓ 那如果在左边不翻转任何硬币呢？
>
> 这就相当于 $j = 0$。在 Nim 中，把一堆石子全部取走（大小为 $i \to 0$），变化为 $S \oplus i \oplus 0 = S \oplus i$。在翻硬币中，只翻转位置 $i$（正面→反面），异或和变化也是 $S \oplus i$。完美对应。

### 手算例子验证

**例子 1**：局面「H T T」（位置 1 正面，位置 2、3 反面）

异或和 = $1$，非零 → 先手必胜。

先手应该怎么走？翻转位置 1 的正面硬币，可以选择：
- 不在左边翻转任何硬币 → 局面变为全反面（终止态），先手获胜。

确实必胜。

**例子 2**：局面「H H T」（位置 1 和 2 正面）

异或和 = $1 \oplus 2 = 3$，非零 → 先手必胜。

先手可以：
- 翻转位置 2（正面→反面），再翻转位置 1（正面→反面）→ 全反面，获胜。
- 或者翻转位置 2（正面→反面），不翻位置 1 → 剩位置 1 正面，异或和 = 1，非零，但对手走一步就赢了？等一下——剩位置 1 正面对手直接翻掉就赢了。所以我们应该选择第一种走法。

验证一下：翻转位置 2 再翻转位置 1，相当于 $S = 1 \oplus 2 = 3 \to 3 \oplus 2 \oplus 1 = 3 \oplus 3 = 0$，完美。

**例子 3**：局面「H T H」（位置 1 和 3 正面）

异或和 = $1 \oplus 3 = 2$，非零 → 先手必胜。

先手应该怎么走？我们需要找一个操作使异或和变为 0：

目标：$S \oplus i \oplus j = 0$，即 $2 \oplus i \oplus j = 0$，$i \oplus j = 2$。

我们可以选 $i=3$（位置 3 是正面），则 $j = i \oplus 2 = 3 \oplus 2 = 1$。

所以操作是：翻转位置 3（正面→反面），再翻转位置 1（正面→反面）。局面变为全反面，获胜。

**例子 4**：局面「H H H」（位置 1、2、3 正面）

异或和 = $1 \oplus 2 \oplus 3 = 0$ → 先手必败。

验证一下：先手可以选择：
- 翻位置 3，再翻位置 1 → 剩位置 2 正面，异或和 = 2
- 翻位置 3，再翻位置 2 → 剩位置 1 正面，异或和 = 1
- 翻位置 3，不翻其他 → 剩位置 1、2 正面，异或和 = 1 ⊕ 2 = 3
- 翻位置 2，再翻位置 1 → 剩位置 3 正面，异或和 = 3
- 翻位置 2，不翻其他 → 剩位置 1、3 正面，异或和 = 1 ⊕ 3 = 2
- 翻位置 1，不翻其他 → 剩位置 2、3 正面，异或和 = 2 ⊕ 3 = 1

无论怎么走，异或和都不为 0，所以对手总有一个必胜策略。

> ❓ 为什么「先手必败」的局面，所有后继都非零？
>
> 这正是 Nim 游戏的性质：从异或和为 0 的局面出发，任何合法操作都会使异或和变为非 0。因为等价性保证了两者完全一致。

### 这个等价性的深刻含义

这个定理的美妙之处在于它提供了一个通用的分析框架：

1. 任何**只涉及翻正面的翻硬币游戏**，都可以先分析每个单独硬币的 SG 值
2. 在 Turning Turtles 中，位置 $i$ 的正面硬币 SG 值正好是 $i$
3. 整个游戏的 SG 值就是各正面硬币 SG 值的异或

但并不是所有翻硬币规则都这么简单——不同规则下硬币的 SG 值不同。不过分析方法是一样的：**先分析单枚硬币，再用异或合并**。

## 更一般的翻硬币模型

### 规则变体：每次只能翻一枚

如果规则是「每次只能翻转一枚正面硬币，不能附带翻转其他硬币」，这就是最简情况：

一枚位置 $i$ 的正面硬币相当于一个大小为 $i$ 的 Nim 堆？不对——在这个规则下，只能翻一枚，相当于每次只能移除一个堆，且必须全部移除（不能只取一部分）。这对应的是「每次只能取走一整堆」的 Nim 变体——这种游戏直接由堆数的奇偶性决定胜负（偶数堆后手胜）。

所以不同的翻硬币规则对应不同的 Nim 变体。

### 翻硬币游戏的通解思路

对于任意的翻硬币规则，分析步骤如下：

1. 确定单枚硬币放在位置 $i$ 时的 SG 值 $g(i)$
2. 整个局面的 SG 值为所有正面硬币 $g(i)$ 的异或
3. 计算 $g(i)$ 的方法是：$g(i) = mex(\{g(j_1) \oplus g(j_2) \oplus \dots \oplus g(j_k) \mid \text{翻转位置 } i \text{ 及 } j_1, j_2, \dots, j_k\})$
   - 注意，翻转操作可以涉及多个硬币，所以后继状态的 SG 值是所有被翻转位置 SG 值的异或（因为单个硬币是独立子游戏）

这种方法给了一种系统性的计算方法。

## 等价性定理的竞赛应用

很多问题表面上跟翻硬币无关，但可以转化为翻硬币模型。例如：

### 应用 1：棋盘上的开关游戏

**问题**（HihoCoder #1172）：一个 $1 \times N$ 的棋盘，每个格子有一盏灯（开/关）。每次可以选一盏亮着的灯，将其熄灭，同时可以（可选地）改变它左边某一盏灯的状态。求先手胜负。

这就是 Turning Turtles 的直译——亮着的灯就是正面朝上的硬币。答案：将所有亮灯位置的编号异或，非零则先手胜。

### 应用 2：区间翻转游戏

**问题**：一个 01 序列，每次可以选择一个位置 $i$（当前为 1），将其变为 0，同时可以选择一个 $j < i$，将 $a_j$ 取反。求先手胜负。

同样是 Turning Turtles——将值为 1 的位置编号异或即可。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    int n;
    string s;
    cin >> n >> s;  // 长度为 n 的 01 串
    int xor_sum = 0;
    for (int i = 0; i < n; i++) {
        if (s[i] == '1') {
            xor_sum ^= (i + 1);  // 位置从 1 开始编号
        }
    }
    cout << (xor_sum ? "First" : "Second") << " wins!" << endl;
    return 0;
}
```

### 应用 3：多堆取石子的另类解释

有些取石子游戏可以映射为翻硬币游戏。比如一堆大小为 $n$ 的石子，每次可以取 $1$ 到 $n$ 颗——这对应一个位置 $n$ 的正面硬币，操作后变成位置 $n-k$ 的正面硬币。

这种视角让问题更直观：一堆石子等价于一枚硬币，「取石子」等价于「把硬币向左移动」然后翻面。

## 基于翻硬币的更复杂博弈

### 规则：不能翻转任意左边硬币，只能翻转特定位置的

比如规则「必须翻转位置 $i$ 和位置 $i-1$ 两枚硬币（如果 $i-1$ 存在）」——这对应了一种不同的 Nim 变体。

分析：$g(1)=mex(\{g(1) \oplus g(0)\})$，但 $g(0)$ 不存在（没有位置 0）——
不对，更准确地说，翻转位置 1 时，只能翻自己，不能翻位置 0（不存在），所以 $g(1)=mex(\{0\})=1$。

位置 2：翻转 2 和 1，后继 SG = $g(2) \oplus g(1)$，但这是递归的！我们需要解方程：
$g(i) = mex(\{g(i) \oplus g(i-1)\})$ ——但这不对，因为 $g(i)$ 出现在右侧，这是循环定义。

正确的理解方式：翻转操作涉及多个硬币时，每个硬币的 SG 值在翻转后取反。但在单硬币分析方法中，我们考虑的是「只放一枚硬币在位置 $i$ 时该游戏的 SG 值」。

对于「必须翻 $i$ 和 $i-1$」的规则，一枚硬币在位置 $i$ 时：
- 可以翻 $i$ 和 $i-1$：后继状态是「位置 $i-1$ 有一枚硬币」（如果原来位置 $i-1$ 没有），或「位置 $i-1$ 的硬币被移除」（如果原来有）

但最坏的情况是，在只有一枚硬币在位置 $i$ 的前提下：
- 翻转后，位置 $i$ 变为反面，位置 $i-1$ 变为正面 → 后继状态的 SG 值就是 $g(i-1)$

所以 $g(i) = mex(\{g(i-1)\})$，递推得 $g(i) = 1$（对所有 $i \ge 1$）！

这意味着局面就是正面硬币数的奇偶性决定胜负——太简单了。

> ❓ 为什么翻不同的硬币会得到不同的 $g(i)$？
>
> 因为规则决定了每次可以操作哪些硬币，这直接影响了状态转移图的结构。不同的规则等价于不同的 Nim 变体——有些等价于普通 Nim（比如 Turning Turtles），有些等价于更简单或更复杂的游戏。

## 小结

翻硬币游戏与 Nim 的等价性揭示了组合博弈论中的一个深层结构：

- **核心思想**：每枚正面硬币对应一个 Nim 堆，大小为该硬币的位置编号
- **SG 值计算**：整个局面的 SG = 所有正面硬币位置编号的异或
- **等价性来源**：翻硬币操作「翻正面硬币 + 可选翻转左边某枚」与 Nim 操作「从大堆取到小堆」在代数上完全同构
- **推广价值**：任何翻硬币游戏的 SG 值都可以通过类似方法分析——先求单枚硬币的 SG 值，再用异或合并

这种等价性不仅仅是一个结论，更是一种**思维范式**：当你面对一个陌生的博弈问题时，试着寻找它与 Nim 或翻硬币游戏之间的联系，往往能让复杂的问题瞬间变得清晰。
