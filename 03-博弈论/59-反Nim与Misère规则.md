# 反 Nim 与 Misère 规则

> Made by Aki.
> 最后更新于 2026.04.30

## 引言

回想标准的 Nim 游戏：取走最后一颗石子的人获胜。现在把规则反过来——**取走最后一颗石子的人输**。

这听起来只是一个小变动，但它让整个游戏的分析变得面目全非。在标准 Nim 中优美简洁的「异或为 0 则必败」结论，在反规则下不再成立。你需要一套全新的分析方法。

这种「最后一步输」的规则被称为 **Misère 规则**（发音 /miːˈzɛər/，来自法语，意为「贫困、悲惨」——因为最后一步不再是胜利而是失败）。与之相对，标准的「最后一步赢」称为 **Normal 规则**。

Misère 规则在实际竞赛中并不少见，掌握它是在博弈论题型中拿全分的关键。

## Misère Nim：策略分析

### 问题定义

**Misère Nim**：有 $k$ 堆石子，每堆 $a_i$ 颗。每次可以从任意一堆中取任意正整数颗（至少 1 颗，至多整堆）。取走最后一颗石子的人**输**。

### 对问题的第一印象

如果你尝试直接用标准 Nim 的思路——用异或和判断——会发现完全不对。

考虑最简单的情况：只有一堆石子，共 1 颗。
- 先手必须取走这 1 颗（因为至少取 1 颗），然后先手输。
- 所以 $[1]$ 是先手必败态。

但用异或和看：$1 \ne 0$，如果是标准 Nim 会判为先手必胜——反了！

再看一堆共 2 颗：
- 先手可以取 1 颗，剩 1 颗给对手 → 对手必须取走最后 1 颗，对手输 → 先手胜。
- 先手也可以取 2 颗，取走所有石子 → 先手取走了最后一颗，先手输（Misère 规则）。所以先手当然会选取 1 颗。
- $[2]$ 是先手必胜态。

用异或和看：$2 \ne 0$，在标准 Nim 中是先手必胜——这次碰巧一致。但只要 $n \ge 2$ 的一堆都是先手必胜（因为可以取到只剩 1 颗），而异或和只判断非零——好像还是不一样。

> ❓ 所以 Misère Nim 到底怎么分析？直接用标准 Nim 的方法会出什么问题？
>
> 问题出在「所有堆都只有 1 颗」的边界情况。当所有堆的大小都 $\le 1$ 时，正常 Nim 和 Misère Nim 的结论完全相反。而当存在大小 $\ge 2$ 的堆时，两者的结论一致——还是用异或和判断。

### Misère Nim 的完整策略

**定理（Misère Nim 的必胜/必败判定）**：

1. 如果所有堆的大小都 $\le 1$（即全是大小为 1 的堆或空堆）：
   - 堆数为奇数 → 先手必败
   - 堆数为偶数 → 先手必胜

2. 如果存在至少一堆大小 $\ge 2$：
   - 异或和为 0 → 先手必败
   - 异或和非 0 → 先手必胜

这被称为 **Misère Nim 的「全一堆例外」规则**。

### 手算例子验证

**例 1**：$[1, 1, 1]$（三堆，每堆 1 颗）

全为 1，堆数为奇数（3），先手必败。

验证：先手取走一堆（1 颗），剩 $[1, 1]$，轮到对手。对手取走一堆，剩 $[1]$，回到先手。先手必须取走最后一颗 → 先手输。所以先手确实必败。

**例 2**：$[1, 1]$（两堆，每堆 1 颗）

全为 1，堆数为偶数（2），先手必胜。

验证：先手取走一堆，剩 $[1]$，轮到对手。对手必须取走最后一颗 → 对手输。先手胜。

**例 3**：$[1, 2, 3]$

存在大小为 2 的堆（$\ge 2$），所以用异或和判断：$1 \oplus 2 \oplus 3 = 0$ → 先手必败。

验证：先手可以怎么走？
- 从堆 3 取 1 → $[1, 2, 2]$，异或和 $1 \oplus 2 \oplus 2 = 1 \ne 0$
- 从堆 3 取 2 → $[1, 2, 1]$ = $[1, 1, 2]$，异或和 $1 \oplus 1 \oplus 2 = 2 \ne 0$
- 从堆 3 取 3 → $[1, 2, 0]$ = $[1, 2]$，异或和 $1 \oplus 2 = 3 \ne 0$
- 从堆 2 取 1 → $[1, 1, 3]$，异或和 $1 \oplus 1 \oplus 3 = 3 \ne 0$
- 从堆 2 取 2 → $[1, 0, 3]$ = $[1, 3]$，异或和 $1 \oplus 3 = 2 \ne 0$
- 从堆 1 取 1 → $[0, 2, 3]$ = $[2, 3]$，异或和 $2 \oplus 3 = 1 \ne 0$

所有后继的异或和都不为 0，说明对手总有一个必胜回应。

**例 4**：$[2, 3]$

存在大小为 2、3 的堆（$\ge 2$），异或和 $2 \oplus 3 = 1 \ne 0$ → 先手必胜。

先手需要找到一步使异或和为 0：从堆 3 取 2 颗 → $[2, 1]$，异或和 $2 \oplus 1 = 3 \ne 0$？不对，我们从堆 3 取到多少呢？

在标准 Nim 中，从大小为 $x$ 的堆取到 $y < x$，异或和变化为 $old \oplus x \oplus y$。我们需要 $1 \oplus 3 \oplus y = 0$，即 $y = 1 \oplus 3 = 2$。

所以从堆 3 取到剩 2 颗（即取 1 颗），得到 $[2, 2]$，异或和 $2 \oplus 2 = 0$。

验证一下：$[2, 2]$，有大小为 $\ge 2$ 的堆，异或和为 0 → 对手处于必败态。对手怎么走？
- 从一堆取 1 → $[1, 2]$，异或和 $1 \oplus 2 = 3 \ne 0$，我们可以回到异或 0
- 从一堆取 2 → $[0, 2]$ = $[2]$，异或和 $2 \ne 0$，但一堆石子（大小 $\ge 2$）在 Misère 中是必胜态！

等等——这里要注意！$[2]$ 是一堆且 $\ge 2$，按照规则先手（我们）必胜。所以对手从 $[2,2]$ 取走一整堆到 $[2]$，轮到我们——这是不是必胜呢？

$[2]$ 是大小为 2 的一堆，轮到我们：我们可以取 1 颗，留 $[1]$ 给对手，对手必须取走最后一颗 → 对手输。所以 $[2]$ 是必胜态！

所以我们留下的 $[2, 2]$ 并没有让对手必败——因为 $[2]$ 作为后继是必胜的。

啊，这说明一个问题：在 Misère Nim 中，虽然「存在大堆时异或为 0 必败」的结论成立，但具体构造必胜策略时需要注意——不能把对手送入「一堆且大小 $\ge 2$」的必胜态。

> ❓ 那在 $[2, 3]$ 局面下，先手到底怎么走才能必胜？
>
> 答案是正确的，但具体走法要小心。从 $[2,3]$ 出发，目标是将对手送入「异或为 0 且不是『一堆大小 $\ge 2$』」或「全 1 且偶数堆」的状态。
>
> 我们试着：从堆 3 取 1 → $[2, 2]$，异或为 0。此时如果对手从一堆取 2（全取），得到 $[2]$——轮到我们。$[2]$ 取 1 留 1 给对手，对手输。所以在我们正确应的情况下，$[2, 2]$ 确实是必败态（因为 $[2]$ 是必胜态意味着从 $[2,2]$ 到 $[2]$ 这一步不是好棋）。
>
> 对手在 $[2,2]$ 的所有走法：
> - 从一堆取 1 → $[1, 2]$，异或 = 3，我们回应：从堆 2 取 1 到 $[1, 1]$（全 1 偶数堆，对手必败）
> - 从一堆取 2 → $[0, 2] = [2]$，我们取 1 留 1 → $[1]$，对手输
> - 合并起来：$[2,2]$ 确实对手必败。

### 进一步理解「全一例外」

为什么当所有堆都只有 1 颗时，结论会反转？

因为在标准 Nim 中，$[1,1,1,\dots]$ 的异或和就是堆数的奇偶性决定的——奇数个 1 异或为 1（必胜），偶数个为 0（必败）。

但在 Misère 下：奇数个 1 意味着最后一颗必然被先手取走（因为每次只能取 1 颗，交替取），所以先手输；偶数个则先手可以迫使对手取最后一颗，所以先手赢。

## Misère 规则下的 SG 函数

### 推广：任意游戏在 Misère 规则下的分析

对于任意 impartial 组合博弈，Misère 规则下的 SG 函数与 Normal 规则下有本质区别。**SG 定理在 Misère 规则下不成立**——你不能简单地把子游戏的 Misère-SG 值异或起来得到整体的 Misère-SG 值。

这是一个深刻的结论：SG 定理严重依赖于 Normal 规则（最后一步走的人赢）。对于 Misère 规则，一般没有这么简洁的通用理论。

### 例外：Misère Nim 为什么有简洁解？

Misère Nim 之所以有简洁解，是因为 Nim 的特殊结构。事实上，**除了 Nim 和少数特例，大多数游戏的 Misère 版本都非常复杂**。

### SJ 定理（Sprague-Grundy for Misère）

但是有一个重要的特例——**SJ 定理（Shoelace-Jagannathan 定理）**，也称为 **Misère SG 定理**，它指出：

> 对于某些特殊的游戏类（称为「几乎 N-游戏」或「tame games」），Misère 规则下的 SG 值可以从 Normal 规则的结果推导出来。

具体来说，SJ 定理适用于所有「每个子游戏的 SG 值 $\le 1$ 的情形」。实际上，SJ 定理是处理反 Nim（Anti-Nim）以及其他反博弈问题的强大工具。

**SJ 定理（简化版）**：
对于 impartial 组合博弈，在 Misère 规则下：

如果所有子游戏的 SG 值都 $\le 1$，则整体的 Misère SG 值为子游戏 SG 值异或和的「取反」——即异或和为 0 时必胜，不为 0 时必败（恰好与 Normal 相反）。

如果存在子游戏的 SG 值 $\ge 2$，则整体的 Misère SG 值与 Normal SG 值相同——即异或和为 0 时必败。

看，这与 Misère Nim 的结论完全一致！因为大小为 1 的堆的 SG 值恰好是 1（在 Normal 规则下），大小为 $\ge 2$ 的堆的 SG 值 $\ge 2$。

> ❓ SJ 定理只适用于 Nim 吗？
>
> 不是。SJ 定理适用于**任何 impartial 组合博弈**。比如你可以把一个复杂的取石子游戏（每次可以取几种不同数量）的每个堆的 SG 值算出来，然后用 SJ 定理判断 Misère 规则下的胜负。只不过更复杂的游戏在计算每个子游戏的 SG 值时就已经不简单了。

### 代码示例：Misère Nim 判定

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    int n;
    cin >> n;
    vector<ll> a(n);
    ll xor_sum = 0;
    bool all_one = true;
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        xor_sum ^= a[i];
        if (a[i] > 1) all_one = false;
    }

    bool first_win;
    if (all_one) {
        // 全为 1：奇数堆则先手输
        first_win = (n % 2 == 0);
    } else {
        // 存在大堆：与标准 Nim 一致
        first_win = (xor_sum != 0);
    }

    cout << (first_win ? "First" : "Second") << " wins!" << endl;
    return 0;
}
```

## 反 Nim 的推广：其他游戏的 Misère 版本

### 巴什博弈的 Misère 版本

**问题**：$n$ 颗石子，每次取 $1 \sim m$ 颗，取走最后一颗的人输。

分析：
- 如果 $n \bmod (m+1) = 0$：先手可以按标准策略走（凑 $m+1$ 的倍数），最后留下 1 颗给对手取走，对手输。所以先手必胜。
- 如果 $n \bmod (m+1) = 1$：先手只能留下 $n-1, n-2, \dots, n-m$ 颗——这些都不是 $(m+1)$ 的倍数，所以对手可以凑 $(m+1)$ 倍数，最终留给先手 1 颗 → 先手输。
- 其他情况：先手可以取到剩 1 颗（$n \bmod (m+1) = 1$ 的状态），先手胜。

这个结论跟标准巴什博弈正好相反：

| $n \bmod (m+1)$ | 标准巴什 | Misère 巴什 |
|:---:|:---:|:---:|
| 0 | 先手输 | 先手赢 |
| 1 | 先手赢 | 先手输 |
| 其他 | 先手赢 | 先手赢 |

### 威佐夫博弈的 Misère 版本

威佐夫博弈的 Misère 版本比 Nim 复杂得多，没有完全一致的简洁公式。但有一个已知结论：对于大部分局面的胜负判定与 Normal 规则相同，只有在 $(0, k)$、$(1, 1)$、$(2, 2)$ 等少数「小」局面时结果会反转。

竞赛中很少要求 Misère 威佐夫这种冷门变体，但了解存在这种差异是好的。

## 反 Nim 的实战策略构造

### 从必胜态构造具体策略

对于 Misère Nim，当判定为先手必胜时，具体应该怎么走？

我们需要分情况：

**情况 1**：全为 1 且堆数为偶数 → 先手必胜。
策略：任意取走一整堆（1 颗），留下奇数个 1 → 对手必败。

**情况 2**：存在大小 $\ge 2$ 的堆且异或和非 0 → 先手必胜。
策略：按照标准 Nim 的策略，找一步使异或和为 0。同时要特别注意——如果走完后所有堆都变成 $\le 1$，需要确保留下的不是「全 1 奇数堆」。

换句话说，标准 Nim 的策略在大部分情况下有效，但需要避开「将局面变为全 1 奇数堆」的走法。

### 一个需要避开的陷阱

假设局面为 $[1, 2, 2]$，异或和 $1 \oplus 2 \oplus 2 = 1 \ne 0$。

标准 Nim 策略：找大小为 $x$ 的堆，使 $x \oplus 1 < x$。$2 \oplus 1 = 3 > 2$，不行（没有减少）。所以无法用标准策略走到异或 0？

等等，重新算：我们需要 $1 \oplus x \oplus y = 0$，即 $y = 1 \oplus x$。

从堆 2 出发：$y = 1 \oplus 2 = 3 > 2$，不行（不能增加大小）。
从堆 1 出发：$y = 1 \oplus 1 = 0$，即取走全部 → 局面变为 $[2, 2]$，异或为 0。

但 $[2, 2]$ 中对手怎么走？前面已经分析过，$[2,2]$ 在 Misère 下是必败态。所以先手直接取走那堆 1，留下 $[2,2]$ 即可。

> ❓ 总是可以通过取走「单独的那个 1」来解决问题吗？
>
> 不总是。比如 $[1, 1, 2]$，异或和 = $1 \oplus 1 \oplus 2 = 2 \ne 0$。需要找 $y = 2 \oplus x$ 且 $y < x$。
>
> 从堆 2 出发：$y = 2 \oplus 2 = 0$，即取走全部 → $[1, 1]$，全 1 偶数堆（对手必败）✓
> 从堆 1 出发：$y = 2 \oplus 1 = 3 > 1$，不行。
>
> 所以正确的策略是取走整堆大小为 2 的石子，留下 $[1, 1]$。这一步在标准 Nim 中也是正确的（$2 \oplus 0 = 2$，恰好需要把堆 2 归零，使异或从 2 变为 0）。

但考虑 $[2, 3, 5]$，异或和 $2 \oplus 3 \oplus 5 = 4 \ne 0$。

标准 Nim 策略：找 $y = 4 \oplus 5 = 1 < 5$，从堆 5 取到剩 1 → $[2, 3, 1]$，异或 $2 \oplus 3 \oplus 1 = 0$。这没问题，因为 $[1, 2, 3]$ 不是全 1。

那如果走完后恰好变成全 1 呢？考虑 $[1, 2, 3]$，异或为 0 → 本身是必败态，不需要考虑从它出发。

我们再找一个例子：$[2, 3]$，异或 = 1。标准策略：从堆 3 取到 2（取 1 颗）→ $[2, 2]$，异或 = 0。没问题，$[2, 2]$ 不是全 1。

$[1, 3]$，异或 = 2。标准策略：从堆 3 取到 1（取 2 颗）→ $[1, 1]$，全 1 偶数堆——这对 Misère 来说是必败态（对手输），所以这个走法可行。但如果留下来的恰好是 $[1]$（全 1 奇数堆）呢？

$[1, 2]$，异或 = 3。标准策略：从堆 2 取到 1（取 1 颗）→ $[1, 1]$，全 1 偶数堆 ✓

看起来标准策略在存在大堆时总是安全的。能证明吗？

**定理**：在 Misère Nim 中，如果存在大小 $\ge 2$ 的堆且异或和非 0，那么存在一步标准 Nim 策略使异或和为 0，且走完后不会出现「全 1 奇数堆」的情况。

**思路**：如果标准策略走完后变成了全 1 奇数堆，这意味着原来只有一堆 $\ge 2$，其他都是 1。可以证明这种情况不可能同时满足异或和非 0 以及 $x \oplus S < x$（取石子的合法性）。证明略，但结论可靠。

## 小结

Misère 规则是博弈论中一个微妙但重要的变体：

- **Misère Nim 的判定**：分两情况——全为 1 时看堆数奇偶，存在大堆时看异或和
- **SJ 定理**：给出了 Misère 规则下 SG 函数的一般分析框架——所有子游戏 SG $\le 1$ 时结论反转，否则与 Normal 一致
- **策略构造**：在存在大堆时，标准 Nim 的取法基本适用，只需避免意外送入「全 1 奇数堆」
- **竞赛要点**：Misère Nim 是竞赛中出现最多的 Misère 规则问题，掌握它即可应对大多数情况

最后记得：竞赛中看到「取走最后一颗者输」的表述，立即切换到 Misère 的分析框架。
