# 巴什博弈与威佐夫博弈

> Made by Aki.
> 最后更新于 2026.04.30

## 引言

假设有两个游戏：

**游戏 A**：一堆石子共 $n$ 颗，两人轮流取，每次可以取 $1 \sim m$ 颗，取走最后一颗的人获胜。

**游戏 B**：两堆石子，数量分别为 $(a,b)$，两人轮流取，每次可以从一堆中取任意数量（至少 1 颗），或者从两堆中取相同数量（至少 1 颗），取走最后一颗的人获胜。

游戏 A 是小学奥数就见过的问题，游戏 B 是著名的「威佐夫博弈」。前者有简单的模数公式，后者的优美程度令人惊叹——它和黄金分割比 $\phi = \frac{1+\sqrt5}{2}$ 有着深刻的联系。

这两类博弈是 ACM 竞赛中最基础、最高频出现的模型。理解它们之后，你会发现很多看似复杂的博弈题，最终都归约为这两个模型之一。

## 巴什博弈（Bash Game）

### 问题描述

一堆石子共 $n$ 颗，两人轮流取，每次可取 $1,2,\dots,m$ 颗（至少 1 颗，至多 $m$ 颗），取走最后一颗者胜。

### 从直觉到公式

我们在上一篇文章的手算部分其实已经得到了答案。让我们重新梳理一遍。

从 $n=0$ 开始：
- $0$：P（必败，无棋可走）
- $1$：N（取 1，走到 P）
- $2$：N（取 2，走到 P）
- $\dots$
- $m$：N（取 $m$，走到 P）
- $m+1$：P（无论取 $1\sim m$，都走到 N 态）
- $m+2$：N（取 1，走到 $m+1$ 这个 P 态）
- $\dots$

模式很清晰了：**$n$ 是 $m+1$ 的倍数时是 P 态，否则是 N 态**。

> ❓ 为什么会以 $m+1$ 为周期？
> 
> 我们来想一下：如果我从一个 P 态出发，无论怎么走，都会走到 N 态（因为每次至少取 1，最多取 $m$，不可能一次跨过 $m+1$ 这个区间）。而从 N 态出发，只要不是 $m+1$ 的倍数，就可以恰好取 $n \bmod (m+1)$ 颗，使剩余石子数为 $m+1$ 的倍数，从而将 P 态送给对手。

### 严谨证明

**定理（巴什博弈）**：在巴什博弈中，先手必胜当且仅当 $n \bmod (m+1) \neq 0$。

**证明**：
我们证明更强的结论：$f(n) = \text{N} \iff n \bmod (m+1) \neq 0$。

用数学归纳法。

- 奠基：$n=0$ 时，$0 \bmod (m+1) = 0$，$f(0)=\text{P}$，成立。
- 归纳假设：对所有小于 $n$ 的石子数结论成立。
- 考虑 $n$：
  - 若 $n \bmod (m+1) = 0$，即 $n = k(m+1)$。对于任意取子 $x \in [1,m]$，有 $n-x = k(m+1)-x$，其模 $m+1$ 余 $m+1-x \in [1,m]$，不为 0。由归纳假设，所有后继都是 N。因此 $n$ 是 P。
  - 若 $n \bmod (m+1) \neq 0$，设 $n = k(m+1) + r$，其中 $1 \le r \le m$。取 $r$ 颗石子，则 $n - r = k(m+1)$，模 $m+1$ 余 0，由归纳假设该后继为 P。因此 $n$ 是 N。

证毕。

### 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    int n, m;
    cin >> n >> m;
    if (n % (m + 1) == 0) {
        cout << "Second wins!" << endl;
    } else {
        cout << "First wins!" << endl;
    }
    return 0;
}
```

### 巴什博弈的变种

**变种 1：取走最后一颗者输**
这时候思路类似。从 $n=1$ 开始手算：
- $1$：取 1 颗就输，所以是 P（必败）
- $2$：取 1 颗，留给对手 1 颗——对手必败，所以 N
- $\dots$

最终规律是：$n \bmod (m+1) = 1$ 时必败。你可以自行尝试推导。

**变种 2：每次可取 $[l, r]$ 颗（不是 $[1,m]$）**
同样可以推导，先手必胜当且仅当 $n \bmod (l+r) \in [l, l+r-1]$（假设 $l \le r$）。我们留作思考题。

## Beatty 定理

在进入威佐夫博弈之前，我们需要一个重要的数论工具——Beatty 定理。

**定理（Beatty）**：设 $\alpha > 1$，$\beta > 1$，且 $\frac{1}{\alpha} + \frac{1}{\beta} = 1$。则数列 $\lfloor \alpha n \rfloor$ 和 $\lfloor \beta n \rfloor$（$n=1,2,3,\dots$）**不交且覆盖所有正整数**。

直觉上：两个数列就像两个人分蛋糕，一个取 $\lfloor \alpha n \rfloor$，另一个取 $\lfloor \beta n \rfloor$，刚好把正整数不重复不遗漏地分完。

**证明**：
记 $A = \{\lfloor \alpha \cdot 1 \rfloor, \lfloor \alpha \cdot 2 \rfloor, \dots\}$，$B = \{\lfloor \beta \cdot 1 \rfloor, \lfloor \beta \cdot 2 \rfloor, \dots\}$。

首先证明 $A \cap B = \varnothing$。假设存在 $k = \lfloor \alpha m \rfloor = \lfloor \beta n \rfloor$。则
$$\alpha m < k+1, \quad \beta n < k+1$$
相加得 $\alpha m + \beta n < 2(k+1)$。

另一方面，$\alpha m \ge k$，$\beta n \ge k$，相加得 $\alpha m + \beta n \ge 2k$。

结合 $\frac{1}{\alpha} + \frac{1}{\beta} = 1$ 可以推出矛盾（具体推导从略，核心在于两个数列不可能有重合）。

再证明覆盖性：对任意正整数 $N$，考虑不超过 $N$ 的数在 $A$ 中出现的次数。不超过 $N$ 的 $\lfloor \alpha n \rfloor$ 的个数为 $\lfloor N/\alpha \rfloor$，不超过 $N$ 的 $\lfloor \beta n \rfloor$ 的个数为 $\lfloor N/\beta \rfloor$。由于 $\frac{1}{\alpha} + \frac{1}{\beta} = 1$，有 $\lfloor N/\alpha \rfloor + \lfloor N/\beta \rfloor = N-1$ 或 $N$（取决于 $N$ 是否在 $A$ 或 $B$ 中）。通过数学归纳即可证明覆盖性。

> ❓ 为什么要学 Beatty 定理才能理解威佐夫博弈？
> 
> 因为威佐夫博弈的奇异局势恰好对应了 Beatty 定理中 $\alpha = \phi$（黄金分割比）的情况。两个数列 $\lfloor n\phi \rfloor$ 和 $\lfloor n\phi^2 \rfloor$ 正好是威佐夫博弈中必败态的两堆石子数。

## 威佐夫博弈（Wythoff Game）

### 问题描述

有两堆石子，数量分别为 $(a,b)$。两人轮流操作，每次可以选择：
1. 从一堆中取任意正整数颗石子
2. 从两堆中取相同正整数颗石子

取走最后一颗石子者胜。问先手是否必胜？

### 直觉探索

我们从小的 $(a,b)$ 开始手算，标记所有 P 态（必败态）。

由于游戏是对称的，我们只考虑 $a \le b$ 的情况。

- $(0,0)$：P，已经结束了。
- $(0,1)$：N，可以取走 1 颗到达 P。
- $(0,2)$：N。
- $\dots$
- $(0,k)$：N（$k>0$），直接取完即可。
- $(1,1)$：N，可以从两堆同时取 1 到 $(0,0)$。
- $(1,2)$：我们来分析一下。从 $(1,2)$ 可以走到：
  - $(0,2)$ N, $(1,1)$ N, $(1,0)$ N（从一堆取）
  - $(0,1)$ N（从两堆同时取 1）
  所有后继都是 N！所以 $(1,2)$ 是 P。

继续手算：
- $(1,3)$：取 1 从第二堆到 $(1,2)$（P），所以是 N。
- $(2,2)$：从两堆各取 2 到 $(0,0)$（P），所以是 N。
- $(2,3)$：从第一堆取 2 到 $(0,3)$（N），从第二堆取 3 到 $(2,0)$（N），取 1 从第二堆到 $(2,2)$（N），从两堆同时取 2 到 $(0,1)$（N），取 1 从两堆到 $(1,2)$（P）——等等，从两堆同时取 1 到 $(1,2)$，这是 P。所以 $(2,3)$ 是 N。
- $(3,5)$：手算发现是 P。
- $(4,7)$：P。
- $(6,10)$：P。

P 态序列（$a\le b$）：$(0,0), (1,2), (3,5), (4,7), (6,10), (8,13), (9,15), (11,18), \dots$

有没有发现规律？令第 $k$ 个 P 态为 $(a_k, b_k)$，其中 $k$ 从 0 开始：
- $a_0 = 0, b_0 = 0$
- $a_1 = 1, b_1 = 2$
- $a_2 = 3, b_2 = 5$
- $a_3 = 4, b_3 = 7$
- $a_4 = 6, b_4 = 10$
- $a_5 = 8, b_5 = 13$

观察到：
- $b_k - a_k = k$（差值刚好等于序号 $k$）
- $a_k$ 是所有「未在之前 P 态中出现过的最小正整数」

这看起来像 Beatty 定理！如果我们能找到一个无理数 $\alpha$，使得 $\lfloor \alpha k \rfloor$ 恰好等于 $a_k$，且 $\lfloor \beta k \rfloor$ 恰好等于 $b_k$，且 $\frac{1}{\alpha} + \frac{1}{\beta} = 1$，那么 Beatty 定理就能保证这两个数列不交且覆盖所有正整数——完美符合 $a_k$ 的「未出现过的最小整数」定义。

### 威佐夫博弈定理

**定理（威佐夫博弈）**：设 $\phi = \frac{1+\sqrt5}{2} \approx 1.618$（黄金分割比），$\phi^2 = \phi + 1 = \frac{3+\sqrt5}{2} \approx 2.618$。则所有必败态（奇异局势）为：
$$( \lfloor k\phi \rfloor,\ \lfloor k\phi^2 \rfloor ), \quad k = 0,1,2,\dots$$

其中 $\lfloor x \rfloor$ 表示向下取整。

**验证**：
- $k=0$：$(0,0)$ ✓
- $k=1$：$\lfloor 1.618 \rfloor = 1$，$\lfloor 2.618 \rfloor = 2$ → $(1,2)$ ✓
- $k=2$：$\lfloor 3.236 \rfloor = 3$，$\lfloor 5.236 \rfloor = 5$ → $(3,5)$ ✓
- $k=3$：$\lfloor 4.854 \rfloor = 4$，$\lfloor 7.854 \rfloor = 7$ → $(4,7)$ ✓
- $k=4$：$\lfloor 6.472 \rfloor = 6$，$\lfloor 10.472 \rfloor = 10$ → $(6,10)$ ✓

完美匹配！

> ❓ 为什么 $\phi$ 能满足 Beatty 定理的条件？
> 
> 因为 $\frac{1}{\phi} + \frac{1}{\phi^2} = \frac{1}{\phi} + \frac{1}{\phi+1}$，注意到 $\phi$ 满足 $\phi^2 = \phi + 1$，所以 $\frac{1}{\phi} + \frac{1}{\phi^2} = \frac{\phi+1}{\phi(\phi+1)} + \frac{\phi}{\phi(\phi+1)} = \frac{2\phi+1}{\phi^2+\phi} = \frac{2\phi+1}{2\phi+1} = 1$。实际上这来自 $\phi$ 的性质：$\phi$ 是方程 $x^2 = x+1$ 的正根。

### 必败态判定

给定 $(a,b)$（假设 $a \le b$），如何判断它是不是必败态？

先计算差值 $d = b - a$，然后看 $a$ 是否等于 $\lfloor d \cdot \phi \rfloor$。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    ll a, b;
    cin >> a >> b;
    if (a > b) swap(a, b);  // 保证 a <= b

    double phi = (1 + sqrt(5)) / 2;
    ll d = b - a;
    ll ak = (ll)(d * phi);  // floor(d * phi)

    if (ak == a) {
        cout << "Second wins!" << endl;  // 奇异局势，先手必败
    } else {
        cout << "First wins!" << endl;   // 非奇异局势，先手必胜
    }
    return 0;
}
```

> ❓ 浮点数精度会不会有问题？
> 
> 当 $a, b$ 很大（比如 $10^{18}$ 级别）时，浮点数精度可能不够。竞赛中的一种常见优化是使用整数运算：因为 $\phi = \frac{1+\sqrt5}{2}$，可以预计算 $\frac{\sqrt5+1}{2}$ 的近似值并配合高精度整数，或者使用 `sqrtl`（long double 版本）。多数情况下 `double` 能处理 $10^9$ 以内的数据。对于更大的数据，可以用黄金分割比的连分数近似来做整数判定。

### 构造必胜策略

如果 $(a,b)$ 不是奇异局势（即先手必胜），我们应该怎么走？

思路很简单：找到唯一的一个奇异局势，使得可以一步到达。

具体来说，有三种方式：
1. 从多的一堆中取石子，使两堆的差值 $d$ 不变，但 $a$ 变为 $\lfloor d \cdot \phi \rfloor$
2. 从少的一堆中取石子，调整差值到某个合适的 $d$
3. 从两堆同时取相同数量，调整差值

对应的实现需要一些分类讨论。我们来看一个完整的裁判代码：

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    ll a, b;
    cin >> a >> b;
    if (a > b) swap(a, b);

    double phi = (1 + sqrt(5)) / 2;
    ll d = b - a;
    ll ak = (ll)(d * phi);

    if (ak == a) {
        cout << 0 << endl;  // 先手必败
    } else {
        cout << 1 << endl;  // 先手必胜
        // 构造第一步：尝试从两堆同时取
        for (ll i = 0; i <= a; i++) {
            ll na = a - i, nb = b - i;
            ll nd = nb - na;
            if ((ll)(nd * phi) == na) {
                cout << na << " " << nb << endl;
                return 0;
            }
        }
        // 否则，从多的一堆取
        for (ll i = 0; i <= b; i++) {
            ll nb = b - i;
            if (nb < a) break;
            ll nd = nb - a;
            if ((ll)(nd * phi) == a) {
                cout << a << " " << nb << endl;
                return 0;
            }
        }
        // 从少的一堆取
        for (ll i = 0; i <= a; i++) {
            ll na = a - i;
            if (na > b) break;
            ll nd = b - na;
            if ((ll)(nd * phi) == na) {
                cout << na << " " << b << endl;
                return 0;
            }
        }
    }
    return 0;
}
```

### 手算例子

**例**：$(a,b) = (10, 15)$。先手是否必胜？如果必胜，第一步怎么走？

**解**：
$d = 15 - 10 = 5$，$\lfloor 5\phi \rfloor = \lfloor 8.09 \rfloor = 8 \ne 10$，所以不是奇异局势，先手必胜。

第一步怎么走？从两堆同时取 $x$ 颗，使得剩余的两堆之差为某个 $d$ 且满足奇异条件。或者从一堆中取。

我们尝试从两堆同时取 $2$ 颗：得到 $(8,13)$，差为 $5$，$\lfloor 5\phi \rfloor = 8$，刚好匹配！所以第一步从两堆各取 2 颗。

验证：$(8,13)$ 确实是奇异局势（$k=5$ 时，$\lfloor 5\phi \rfloor = 8$，$\lfloor 5\phi^2 \rfloor = 13$）。

## 巴什博弈 vs 威佐夫博弈 — 对比与总结

| 特性 | 巴什博弈 | 威佐夫博弈 |
|------|---------|-----------|
| 堆数 | 1 堆 | 2 堆 |
| 操作 | 取 $1\sim m$ 颗 | 取任意颗（一堆或两堆同取） |
| 必败态 | $n \bmod (m+1) = 0$ | $(\lfloor k\phi\rfloor, \lfloor k\phi^2\rfloor)$ |
| 核心工具 | 模运算 | Beatty 定理 + 黄金分割 |
| 复杂度 | $O(1)$ | $O(1)$ |

两种博弈是竞赛中最基础的双人博弈模型。如果你在赛场上看到一个取石子问题，先问自己：

**这是几堆石子？操作有什么限制？**

- 一堆 + 取固定范围 → 巴什博弈
- 两堆 + 可以取任意颗或等量 → 威佐夫博弈
- 多堆 + 取任意颗 → Nim 游戏（下一篇）

## 小结

- **巴什博弈**：$n \bmod (m+1) = 0$ 为必败态，推导思路是构造模 $m+1$ 的对称策略
- **威佐夫博弈**：必败态为 $(\lfloor k\phi \rfloor, \lfloor k\phi^2 \rfloor)$，与黄金分割比深刻绑定
- 两类博弈的共同思想：**找到所有 P 态的规律，然后用公式 $O(1)$ 判断**
- Beatty 定理作为数论工具，在威佐夫博弈的证明中扮演了核心角色
