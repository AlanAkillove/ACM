# 不平等博弈与 Surreal Numbers 入门

> Made by Aki.
> 最后更新于 2026.04.30

## 引言

至此，我们讨论的所有博弈都是 **Impartial（公平）博弈**——在任一局面下，双方可走的操作完全相同。Nim、巴什、SG 函数等全部建立在这个设定上。

但如果游戏的规则对双方**不对称**呢？比如：先手只能移动白色棋子，后手只能移动黑色棋子；或者一方可以取 1 或 2 颗石子，另一方只能取 1 或 3 颗石子。

这就是 **Partisan（不平等）博弈**——双方的「武器库」不同。分析不平等博弈需要的数学工具不再只是 SG 函数，而是一个更深邃的概念：**Surreal Numbers（超现实数）**。

超现实数由 John Conway 在 1970 年代提出，它不仅是一种数的构造方式，更是不平等博弈的「通用语言」——**每个不平等博弈局面都可以映射为一个超现实数，其正负和大小直接对应博弈的胜负**。

## Partisan Game 的概念

### 公平博弈 vs 不平等博弈

**公平博弈（Impartial Game）**：双方可走操作完全相同。我们用 SG 函数分析。

**不平等博弈（Partisan Game）**：双方可走操作不同。需要更复杂的工具。

一个经典的不平等博弈例子是 **Col 游戏**：在黑白棋盘上，先手（Left）只能删除黑色的棋子及与之相连的区域，后手（Right）只能删除白色的棋子及与之相连的区域。

另一个更简单的例子是 **红蓝 Hackenbush**：丛林中有红（先手专用）和蓝（后手专用）两种颜色的边，以及绿色（双方共用）的边。先手只能砍红边和绿边，后手只能砍蓝边和绿边。

### 符号约定

在不平等博弈中，通常用：
- $L$（Left）：先手，控制左操作
- $R$（Right）：后手，控制右操作
- 一个局面记作 $G = \{G^L \mid G^R\}$，其中 $G^L$ 是 Left 可走到的后继局面集合，$G^R$ 是 Right 可走到的后继局面集合

这与公平博弈中 $SG(x) = mex(\{SG(y)\})$ 的表示形式有异曲同工之妙——都是通过「后继局面」定义当前局面。

### 胜负的四种可能

在不平等博弈中，给定一个局面 $G$，双方都采取最优策略：

- **$G > 0$**：Left 先手时 Left 胜，Right 先手时 Left 也胜。称「Left 必胜」。
- **$G < 0$**：Left 先手时 Right 胜，Right 先手时 Right 也胜。称「Right 必胜」。
- **$G = 0$**：先手者负（谁先手谁输）。相当于公平博弈中的 P-position。
- **$G \parallel 0$**（$G$ 与 0 不可比）：先手者胜（谁先手谁赢）。相当于公平博弈中的 N-position。

## 从博弈到超现实数

### 构造性定义

超现实数的定义与博弈的表示完全一致。一个超现实数由左集合和右集合构成：

$$
x = \{X_L \mid X_R\}
$$

其中 $X_L$ 和 $X_R$ 是超现实数的集合，且满足**任意左集合中的数 < 任意右集合中的数**。

这个条件看似简单，但覆盖了所有实数甚至更多！

### 从 0 开始构造

最小的超现实数是：$0 = \{\ \mid\ \}$（左右集合都为空）。

任何左集合为空、右集合为空的博弈局面也正好是「终止态」——谁先走谁输。这正是 $G = 0$ 的含义。

### 构造 1 和 -1

$1 = \{0 \mid\ \}$（左集合包含 0，右集合为空）
$-1 = \{\ \mid 0\}$（左集合为空，右集合包含 0）

这意味着：
- 在博弈中，1 表示 Left 可以一步走到终止态（0），但 Right 无法操作 → Left 必胜
- -1 表示 Right 可以一步走到终止态，但 Left 无法操作 → Right 必胜

### 构造 2

$2 = \{1 \mid\ \}$

### 构造 1/2

$\frac{1}{2} = \{0 \mid 1\}$

这意味着：Left 可以走到 0，Right 可以走到 1。

这个博弈的直觉：Left 走一步到必败态（0），所以 Left 赢；Right 走一步到 Left 必胜态（1），所以 Right 输。因此 $\frac{1}{2} > 0$（Left 必胜）。但它只有 1 的一半优势——Left 先手赢，但优势比 1 小。

> ❓ 1/2 这个「数」怎么在博弈中体现？
>
> 想象两堆石子：第一堆有 1 颗（Left 只能取这堆），第二堆有 2 颗（Right 只能取这堆）。Left 每次可以取走第一堆的，Right 每次可以取走第二堆的。这其实不是一个「一人取一堆」的简单模型——但超现实数正是用来量化这种「优势程度」的。
>
> 实际上，$\{0 \mid 1\}$ 最常见的体现是：Left 可以一步赢（到 0），Right 如果走一步则给了 Left 一个必胜局面（1）。Left 的优势恰好是「半步」。

## 超现实数的运算

### 加法

两个超现实数的加法定义为：

$$
x + y = \{X_L + y, x + Y_L \mid X_R + y, x + Y_R\}
$$

即：轮到 Left 时，可以在 $x$ 的左集合中走一步（$X_L + y$），或在 $y$ 的左集合中走一步（$x + Y_L$）。轮到 Right 时同理。

这对应于博弈的「和」——两个独立游戏放在一起玩，每回合只能操作其中一个。

### 例子：计算 1 + 1

$1 = \{0 \mid\ \}$，所以：
$$
1 + 1 = \{0 + 1, 1 + 0 \mid \ \} = \{1 \mid\ \} = 2
$$

### 例子：计算 1/2 + 1/2

$$
\frac{1}{2} + \frac{1}{2} = \{0 \mid 1\} + \{0 \mid 1\} = \{0+\frac{1}{2}, \frac{1}{2}+0 \mid 1+\frac{1}{2}, \frac{1}{2}+1\} = \{\frac{1}{2} \mid \frac{3}{2}\}
$$

在超现实数中，$\{\frac{1}{2} \mid \frac{3}{2}\} = 1$，所以 $\frac{1}{2} + \frac{1}{2} = 1$。这验证了分数加法的一致性。

### 相反数

$$
-x = \{-X_R \mid -X_L\}
$$

即左右集合交换并取反。

### 比较

$x \le y$ 当且仅当**不存在 $x_L \in X_L$ 使得 $y \le x_L$，且不存在 $y_R \in Y_R$ 使得 $y_R \le x$**。

直观理解：如果 Left 可以走一步让 $x$ 变得 $\ge y$，或者 Right 可以走一步让 $y$ 变得 $\le x$ 则……

这个递归定义看起来复杂，但计算时通常不需要每次都回溯定义——可以直接用数值判断。

## 例子：用超现实数分析不平等博弈

### 红蓝 Hackenbush

在红蓝 Hackenbush 中：
- Left 可以砍红边或绿边
- Right 可以砍蓝边或绿边

每个局面可以映射为一个超现实数，数值等于该局面的**值**（value）：
- 正值 → Left 优势
- 负值 → Right 优势
- 0 → 后手胜

映射规则（简化版）：
- 一条单独的红边（只有 Left 能砍）= $1$
- 一条单独的蓝边 = $-1$
- 一条绿边（双方都能砍）= $0$（但这样过于简化——实际上绿边不是 0）

更精确的：一条有 $m$ 层（红边构成一条链）的 Hackenbush 局面，其值等于一个正数，具体值取决于树的形状。如果链上有 $n$ 条红边，其值为 $n$；如果混杂了蓝边，则用超现实数加法计算。

具体的计算需要用到 Colon Principle 的思想——但与公平博弈的 Colon Principle 不同，Partisan 版本的 Colon Principle 要复杂得多。

### 一个简单的例子

考虑一个 Hackenbush 局面：
- 一条红边（Left 专属），值 = 1
- 一条蓝边（Right 专属），值 = -1

两棵树独立，所以整体值 = $1 + (-1) = 0$ → 后手胜。

验证：先手走一步（砍自己的边），剩下对方的一条边，对方砍掉获胜。确实后手胜 ✓

## 更多超现实数

### 小数和分数

所有有理数都可以用超现实数表示。例如：
- $\frac{1}{4} = \{0 \mid \frac{1}{2}\}$
- $\frac{3}{4} = \{\frac{1}{2} \mid 1\}$
- $\frac{1}{8} = \{0 \mid \frac{1}{4}\}$

事实上，用二分法可以构造出所有分母为 $2^n$ 的分数，进而构造出所有实数的超现实数表示。

### 无穷大和无穷小

超现实数的真正力量在于可以表示无穷大和无穷小：

- $\omega = \{0, 1, 2, 3, \dots \mid\ \}$（大于所有整数）
- $\epsilon = \{0 \mid 1, \frac{1}{2}, \frac{1}{4}, \dots\} = 1/\omega$（小于所有正实数，但大于 0）

在博弈中，$\omega$ 表示 Left 有「无限步」的优势；$\epsilon$ 表示 Left 有「小于任何正实数但大于 0」的优势——即超级微小的领先。

### 星号（*）和「模糊」关系

$* = \{0 \mid 0\}$——这不是一个超现实数（因为左集合中的 0 = 右集合中的 0，不满足「左 < 右」条件），但它是一个**博弈值**。

在公平博弈中，一个 SG 值为 1 的博弈状态的值正是 $*$。$*$ 与 0 不可比（$* \parallel 0$），这正是 N-position 的含义——先手胜。

更一般地，$\{a \mid b\}$ 中如果 $a < b$，结果是超现实数；如果 $a \ge b$，结果是一个**游戏**（game），不是数。

### 常见值的总结

| 表示 | 含义 |
|:---|:---|
| $0$ | 后手胜（谁先走谁输） |
| $n$（正整数） | Left 有 n 步的优势 |
| $-n$ | Right 有 n 步的优势 |
| $1/2$ | Left 有半步优势 |
| $*$ | 先手胜 |
| $\uparrow = \{0 \mid *\}$ | 很小的 Left 优势（大于 0 但小于任意正实数） |
| $\downarrow = \{\* \mid 0\}$ | 很小的 Right 优势 |

> ❓ $\uparrow$（读作「up」）是什么？它有什么用？
>
> $\uparrow = \{0 \mid *\}$ 是另一个重要的博弈值。它不是超现实数（因为 $0 < *$ ？不，$* \parallel 0$，它们不可比——但这是游戏层面的判断）。
>
> 实际计算中，$\uparrow$ 表示「Left 有极微小的优势」——如果你在 $\uparrow$ 和 $1/2$ 之间选择，应该选 $1/2$；但 $\uparrow > 0$（Left 先手必胜），且 $\uparrow + \uparrow > 0$。
>
> 在复杂的 Partisan 博弈中，$\uparrow$ 和 $\downarrow$ 经常作为微调值出现，用于处理那些「赢不太多」的局面。

## 在竞赛中应用超现实数

坦率地说，超现实数在 ACM/ICPC 竞赛中的直接考察非常少——它更常出现在数学竞赛（特别是信息学竞赛中的选拔题）中。绝大多数博弈问题仍然可以通过 SG 函数或简单的 Partisan 分析解决。

但如果你遇到以下特征的问题，可能需要用到超现实数：

1. **双方操作不对称**——明显是 Partisan 游戏
2. **局面可以分解为子局面之和**——加法运算是核心
3. **需要精确量化优势大小**——而不只是判断胜负

### 典型问题：不平等取石子

**问题**（简化）：有两堆石子。Left 每次可以从第一堆取 1 或 2 颗，或从第二堆取 1 颗。Right 每次可以从第一堆取 1 颗，或从第二堆取 1 或 2 颗。取走最后一颗者胜。

这显然是不平等博弈——双方能操作的堆和数量不同。可以用超现实数分析每个堆的值。

### 典型问题：Procrastination

**问题**（原题 POJ 2931 - "Procrastination"）：这是一个经典的 Partisan 博弈问题，涉及一个不平等取石子游戏，需要分析局面的值。

解法思路：将游戏分解为若干独立子游戏，计算每个子游戏的值（超现实数），然后相加判断总和的正负。

### LOJ 2327

**问题**（LOJ 2327 - 「清橙 A1366」简单的博弈）：给出了一个不平等的博弈树，双方策略不同。

这类问题通常需要递归地计算局面的值。虽然可以用超现实数理论系统分析，但在实际竞赛中，往往只需要用到「比较两个局面的大小关系」——而这可以通过递归定义直接判断，不需要显式计算超现实数的数值。

### 一个判断胜负的简化方法

对于简单的 Partisan 博弈，有时不需要计算确切的超现实数值，而是直接按照博弈树的递归定义判断：

```
function evaluate(G):
    if G 是终止态: return 0
    if Left 可走到值 <= 0 的局面:
        return positive  // Left 胜
    if Right 可走到值 >= 0 的局面:
        return negative  // Right 胜
    // 否则取决于谁先手
```

但这个简化判断只对「双方只有胜负两个结果」的简单情况有效。对于需要**量化优势**（如追及问题、资源累积问题）的情况，仍然需要超现实数。

## 方展鹏论文导读

如果要深入学习超现实数在竞赛中的应用，推荐阅读**方展鹏**的论文《浅谈如何解决不平等博弈问题》（发表于 2008 年国家集训队论文）。

这篇论文的主要内容包括：

1. **超现实数的构造与运算**：从基础定义开始，系统地介绍超现实数的代数结构
2. **与博弈的对应关系**：详细阐述如何将 Partisan 博弈局面映射为超现实数
3. **运算封闭性**：证明超现实数在加法、乘法下封闭（乘法很少在博弈中使用）
4. **案例分析**：用多个经典例题展示超现实数法的优越性
5. **与其他方法的对比**：与 SG 函数、Minimax 搜索等方法对比

论文的关键结论可以总结为：

> 对于任何 Partisan 博弈，如果其状态可以用超现实数表示，那么整体博弈的值等于各子博弈值的和。判断胜负只需看最终值的正负——正的 Left 胜，负的 Right 胜，零则后手胜。

### 论文中的经典例题

**例题一**：不平等取石子游戏，双方每次可取的石子集合不同。通过递推计算每个石子数对应的超现实数值，得到 SG 函数的一般化形式。

**例题二**：一个复杂的 Hackenbush 变体，需要用 Colon Principle 的 Partisan 版本来计算树的值。

**例题三**：双方操作不对称的棋盘游戏，利用超现实数加法拆解为独立行的值的和。

## 小结

不平等博弈和超现实数是组合博弈论的「高阶领地」：

- **不平等博弈（Partisan Game）**：双方操作不对称，记作 $G = \{G^L \mid G^R\}$
- **超现实数（Surreal Numbers）**：每个数形如 $\{X_L \mid X_R\}$，其中 $X_L < X_R$
- **值与胜负**：正值 → Left 胜，负值 → Right 胜，0 → 后手胜，$*$ → 先手胜
- **加法**：独立子游戏的和，值直接相加
- **无穷大与无穷小**：$\omega$ 和 $\epsilon$ 可以表示「无限大优势」和「无穷小优势」
- **$\uparrow$ 与 $\downarrow$**：代表极微小的优势

在竞赛中，超现实数的直接考察很少，但了解其思想对于解决需要量化优势的 Partisan 博弈问题极有帮助。方展鹏的论文是这方面的必读材料。
