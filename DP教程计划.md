# DP（动态规划）算法教程设计计划

## 背景

用户希望基于其 GitHub 仓库 https://github.com/AlanAkillove/ACM 中的 ACM 数学算法教程风格，设计一个针对 DP（动态规划）算法的完整教程。需要保持相同的教学风格、结构组织和内容深度。

## 现有教程风格分析

通过分析用户的数学算法教程，发现以下关键特点：

### 结构组织
1. **标题格式**：`# 主题名称`
2. **作者信息**：`> Made by Aki.`
3. **更新日期**：`> 最后更新于 YYYY.MM.DD`
4. **引言**：以实际竞赛问题开始，说明为什么需要学习这个概念
5. **前置知识**：列出需要的先决条件
6. **核心推导**：详细的数学推导和算法原理
7. **代码实现**：提供 C++ 代码模板
8. **复杂度与正确性分析**：分析时间空间复杂度
9. **典型应用场景**：竞赛中的实际应用
10. **小结**：总结核心要点
11. **系列索引**：指向上一篇和下一篇

### 写作风格
- **竞赛导向**：每篇以「为什么学这个」开篇，以「什么时候用」收尾
- **重推导轻模板**：思路 > 推导 > 代码；代码是推导的自然产物而非终点
- **新人友好**：引导式语言，手算例子穿插，❓ 标注答疑与易错点
- **做题友好**：推荐题目均带链接（Codeforces、洛谷等）

### 内容深度
- 详细的数学推导和算法原理
- 实际竞赛问题示例
- 代码实现和优化技巧
- 复杂度分析
- 推荐练习题（带链接）
