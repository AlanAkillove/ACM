# 代码目录

本目录存放 [ACM 竞赛数学系列](..) 中各篇文档的例题与推荐练习题参考代码。

## 目录结构

```
codes/
├── cpp/          # C++ 参考代码
├── python/       # Python 参考代码
└── README.md     # 本文件
```

## 命名规范

所有代码文件按 **板块-文章-例题** 三级结构存放：

```
{板块编号}/{文章编号}/{板块编号}-{文章编号}-{例题编号}.{ext}
```

例如 `python/01/01-01/01-01-01.py`：

| 层级 | 值 | 含义 |
|------|-----|------|
| 板块编号 | `01` | 数论板块 |
| 文章编号 | `01-01` | 第 1 篇（模运算与同余基础） |
| 例题编号 | `01-01-01` | 该篇第 1 道例题 |
| 扩展名 | `.py` | Python 实现 |

## 行首注释

每份代码的第一行以注释形式标注原题链接，方便对照查阅：

```python
# https://codeforces.com/problemset/problem/1328/A
```

## 语言说明

| 语言 | 目录 | 用途 |
|------|------|------|
| **Python 3** | `python/` | 快速验证算法思路，简洁易读 |
| **C++17** | `cpp/` | 竞赛标准实现，兼顾性能与可读性 |

> 代码仅为参考实现，推荐在理解推导过程后独立完成代码编写。同一道题在 `cpp/` 和 `python/` 中共享相同的路径和文件名（仅扩展名不同）。
