# 图上游走的蚂蚁：从三角形到多面体，从禁止停留到允许停留的计数问题

> Made by Aki.
> 最后更新于 2026.06.03

一只蚂蚁在连通图的顶点间游走，每步沿一条边移动。给定起点和步数 $n$，恰好 $n$ 步后回到起点的方案数是多少？如果允许蚂蚁「原地踏步」又会如何？本文从邻接矩阵的代数结构出发，利用对称性将高维状态压缩到低维递推，系统求解从三角形到立方体、从禁止停留到允许停留的各类情形。

---

## 一、基础知识

### 1.1 问题符号

| 符号 | 含义 |
|------|------|
| $G = (V, E)$ | 无向连通图，$|V| = m$ |
| $A$ | $G$ 的邻接矩阵：$A_{ij} = 1$ 当 $i,j$ 有边，否则 $0$ |
| $I$ | $m \times m$ 单位矩阵 |
| $f_n(v)$ | 从起点 $s$ 出发，恰好 $n$ 步后到达顶点 $v$ 的方案数 |
| $\text{MOD} = 10^9 + 7$ | 常用模数 |

### 1.2 邻接矩阵与路径计数

**定理**：禁止停留时，$(A^n)_{ij}$ 等于从 $i$ 到 $j$ 恰好 $n$ 步的路径数。允许停留（每步可加自环）时，方案数为 $((A+I)^n)_{ij}$。

**证明**（归纳）：$n=1$ 时 $A_{ij} = 1$ 当且仅当 $ij$ 有边，即有恰好 1 条长为 1 的路。假设 $n=k$ 时成立，则：

$$
(A^{k+1})_{ij} = \sum_{t} (A^k)_{it} \cdot A_{tj}
$$

从 $i$ 到 $t$ 的 $k$ 步路径 $(A^k)_{it}$ 再接上边 $t \to j$，求和遍历所有中间顶点 $t$，恰为 $k+1$ 步路径总数。允许停留时，每一步多一个「自环」选项，等价于将邻接矩阵替换为 $A+I$。

### 1.3 对称性与状态约简

当图具有对称性时，许多顶点的路径计数相同。将这些等价顶点归入同一个**轨道**（orbit），以轨道为状态建立低维递推，将 $O(m^3 \log n)$ 的矩阵快速幂降为 $O(k^3 \log n)$（$k$ 为轨道数，通常远小于 $m$）。

轨道划分原则：若存在图的自同构将顶点 $u$ 映射到 $v$，且在递推过程中始终保持关于起点的距离对称性，则 $f_n(u) = f_n(v)$。

### 1.4 动态规划与矩阵快速幂

- **滚动数组 DP**：$O(k^2 n)$，适用于 $n \le 10^7$，状态数 $k$ 较小。
- **矩阵快速幂**：$O(k^3 \log n)$，适用于 $n \le 10^{18}$。
- **线性递推 + Bostan-Mori**：当递推阶数很小且 $n$ 极大（$10^{10^5}$ 级别）时可用，本文不展开。

以下为本文所有代码共用的矩阵快速幂模板。更多细节见 [第 74 篇：矩阵基本运算与快速幂](../05-线性代数/74-矩阵基本运算与快速幂.md)。

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const ll MOD = 1e9 + 7;

// 普通快速幂（用于数值公式）
ll qpow(ll a, ll b, ll m = MOD) {
    ll r = 1; a %= m;
    while (b) {
        if (b & 1) r = r * a % m;
        a = a * a % m; b >>= 1;
    }
    return r;
}

// 矩阵结构（动态大小）
struct Mat {
    int n;
    vector<vector<ll>> a;
    Mat(int _n) : n(_n), a(_n, vector<ll>(_n, 0)) {}
    Mat operator*(const Mat& o) const {
        Mat r(n);
        for (int i = 0; i < n; ++i)
            for (int k = 0; k < n; ++k)
                if (a[i][k])
                    for (int j = 0; j < n; ++j)
                        r.a[i][j] = (r.a[i][j] + a[i][k] * o.a[k][j]) % MOD;
        return r;
    }
};

Mat identity(int n) {
    Mat r(n);
    for (int i = 0; i < n; ++i) r.a[i][i] = 1;
    return r;
}

Mat mat_qpow(Mat a, ll b) {
    Mat r = identity(a.n);
    while (b) {
        if (b & 1) r = r * a;
        a = a * a; b >>= 1;
    }
    return r;
}
```

## 二、禁止停留——从三角形到立方体

本节讨论**禁止停留**（每步必须移动）的情况。

### 2.1 三角形（$K_3$）

**结构**：3 个顶点两两相连，每个顶点度数 $2$。

**状态划分**：起点为 $S$，其余两个顶点由对称性等价，记为 $A$。状态：$a_n = f_n(S)$，$b_n = f_n(A)$（每个 $A$ 类顶点的方案数）。

**转移方程**：

- $a_n = 2 \cdot b_{n-1}$：从每个 $A$ 顶点各有一条边通向 $S$，共 2 个 $A$ 顶点，每个计数 $b_{n-1}$。
- $b_n = a_{n-1} + b_{n-1}$：从 $S$ 有一条边到该 $A$ 顶点；从另一个 $A$ 顶点也有一条边到该 $A$ 顶点。两个来源的计数分别是 $a_{n-1}$ 和 $b_{n-1}$。

**初始条件**：$a_0 = 1$，$b_0 = 0$。

**矩阵形式**：

$$
\begin{bmatrix} a_n \\ b_n \end{bmatrix}
=
\begin{bmatrix} 0 & 2 \\ 1 & 1 \end{bmatrix}^n
\begin{bmatrix} 1 \\ 0 \end{bmatrix}
$$

**闭式推导**：消去 $b$ 得 $a_n = 2(a_{n-2} + b_{n-2}) = 2a_{n-2} + a_{n-1}$，即 $a_n - a_{n-1} - 2a_{n-2} = 0$。特征方程 $r^2 - r - 2 = 0$ 的根为 $r = 2, -1$，通解 $a_n = \alpha \cdot 2^n + \beta \cdot (-1)^n$。代入 $a_0 = 1$，$a_1 = 0$ 解得：

$$
\boxed{a_n = \tfrac{2^n + 2 \cdot (-1)^n}{3}}
$$

推广到完全图 $K_m$：

$$
\boxed{a_n = \frac{(m-1)^n + (m-1) \cdot (-1)^n}{m}}
$$

**验证表**：

| $n$ | $a_n$（公式） | 手动枚举 |
|-----|-------------|---------|
| 1 | $(2 - 2)/3 = 0$ | 从 $S$ 出发 1 步只能到邻居，无法回到 $S$ |
| 2 | $(4 + 2)/3 = 2$ | $S \to A \to S$，$S \to B \to S$ |
| 3 | $(8 - 2)/3 = 2$ | $S \to A \to B \to S$，$S \to B \to A \to S$ |

```cpp
// K3 禁止停留 —— 2×2 矩阵快速幂
ll solve_k3(ll n) {
    Mat M(2);
    M.a[0][0] = 0; M.a[0][1] = 2;
    M.a[1][0] = 1; M.a[1][1] = 1;
    Mat P = mat_qpow(M, n);
    return P.a[0][0];   // a_n = (M^n)_{0,0} · a_0，a_0 = 1
}
```

### 2.2 正方形环（$C_4$）

**结构**：顶点 $0, 1, 2, 3$ 依次成环，每个顶点度数 $2$。起点为 $0$。

**状态划分**：距离起点的最短路径长度将顶点划分为三个轨道：

| 轨道 | 距离 | 顶点 |
|------|------|------|
| $S$ | 0 | $\{0\}$ |
| $P$ | 1 | $\{1, 3\}$ |
| $Q$ | 2 | $\{2\}$ |

状态：$a_n = f_n(0)$，$b_n = f_n(1) = f_n(3)$，$c_n = f_n(2)$。

**转移方程**：

- $a_n = 2 \cdot b_{n-1}$：顶点 $1$ 和 $3$ 各有一条边通往 $0$。
- $b_n = a_{n-1} + c_{n-1}$：顶点 $1$ 的邻居是 $0$ 和 $2$，各贡献一条边。
- $c_n = 2 \cdot b_{n-1} = a_n$：由对称性，$c_n$ 的递推与 $a_n$ 相同（注意 $c_0 = 0 \neq a_0$）。

**闭式**：联立得 $a_n = 4a_{n-2}$（$n \ge 3$）。结合 $a_0 = 1$，$a_1 = 0$，$a_2 = 2$：

$$
\boxed{a_n = \begin{cases}
1, & n = 0 \\
0, & n \text{ 为奇数} \\
2^{\,n-1}, & n \ge 2 \text{ 且为偶数}
\end{cases}}
$$

**验证**：$n=2$：$a_2 = 2^1 = 2$（$0 \to 1 \to 0$ 或 $0 \to 3 \to 0$），$n=4$：$a_4 = 8$。

```cpp
// C4 禁止停留 —— 3×3 矩阵快速幂
ll solve_c4(ll n) {
    Mat M(3);
    M.a[0][1] = 2;                      // S <- 2*P
    M.a[1][0] = 1; M.a[1][2] = 1;       // P <- S + Q
    M.a[2][1] = 2;                      // Q <- 2*P
    Mat P = mat_qpow(M, n);
    return P.a[0][0];                    // a_0 = 1
}
```

> 注：$C_4$ 是二部图，奇偶步交替，所以奇数步 $a_n = 0$。

### 2.3 正四面体（$K_4$）

**结构**：4 个顶点两两相连，每个顶点度数 $3$。

**状态划分**：起点 $S$，其余 3 个顶点等价，每个记为 $A$。完全图 $K_4$ 的对称性给出两个状态。

**转移方程**：

- $a_n = 3 \cdot b_{n-1}$
- $b_n = a_{n-1} + 2 \cdot b_{n-1}$（从 $S$ 有一条边到该 $A$；从另外 2 个 $A$ 各有一条边到该 $A$）

**闭式**（代入 $K_m$ 公式 $m=4$）：

$$
\boxed{a_n = \frac{3^n + 3 \cdot (-1)^n}{4}}
$$

**验证**：$a_1 = 0$，$a_2 = (9 + 3)/4 = 3$，$a_3 = (27 - 3)/4 = 6$。直观理解：$n=2$ 时有 3 条长度为 2 的回路（从 $S$ 出发经任一邻居返回），$n=3$ 时每条 $S \to A \to B \to S$ 路径有 $3 \times 2 = 6$ 种排列。

```cpp
// K4 禁止停留 —— 直接套用 K_m 公式
ll solve_k4(ll n) {
    ll m = 4;
    ll p = qpow(m - 1, n);                    // 3^n
    ll sign = (n & 1) ? MOD - 1 : 1;           // (-1)^n
    ll inv_m = qpow(m, MOD - 2);               // 费马小定理求逆元
    return (p + (m - 1) * sign) % MOD * inv_m % MOD;
}
```

> 这本质上就是 [CF 166E - Tetrahedron](https://codeforces.com/problemset/problem/166/E)：正四面体上走 $n$ 步回到起点的方案数。

### 2.4 立方体

**结构**：8 个顶点，每个顶点度数 $3$。将顶点编号为 $(x,y,z) \in \{0,1\}^3$，起点为 $000$。

**状态划分**：按与起点的汉明距离分为四个轨道：

| 轨道 | 距离 | 顶点数 | 示例 |
|------|------|--------|------|
| $S$ | 0 | 1 | 000 |
| $P$ | 1 | 3 | 100, 010, 001 |
| $Q$ | 2 | 3 | 110, 101, 011 |
| $R$ | 3 | 1 | 111 |

**转移分析**（以单个顶点的视角）：每个距离-1 顶点有 1 个距离-0 邻居和 2 个距离-2 邻居，每个距离-2 顶点有 2 个距离-1 邻居和 1 个距离-3 邻居。

**转移矩阵**：

$$
\begin{bmatrix} a_n \\ b_n \\ c_n \\ d_n \end{bmatrix}
=
\begin{bmatrix}
0 & 3 & 0 & 0 \\
1 & 0 & 2 & 0 \\
0 & 2 & 0 & 1 \\
0 & 0 & 3 & 0
\end{bmatrix}
\begin{bmatrix} a_{n-1} \\ b_{n-1} \\ c_{n-1} \\ d_{n-1} \end{bmatrix}
$$

其中 $a_n = f_n(000)$，$b_n = f_n(100)$（每个 $P$ 类顶点），$c_n = f_n(110)$（每个 $Q$ 类顶点），$d_n = f_n(111)$。

**推导理由**：
- 第一行：到 $000$ 只能从 3 个距离-1 顶点来，每个各贡献 1 条边。
- 第二行：到某个特定距离-1 顶点，邻居有 1 个 $000$ 和 2 个距离-2 顶点。
- 第三行：到某个特定距离-2 顶点，邻居有 2 个距离-1 顶点和 1 个 $111$。
- 第四行：到 $111$ 只能从 3 个距离-2 顶点来。

立方体的 $a_n$ 无简单闭式，但矩阵 $M$ 的特征值为 $\pm 3, \pm 1$，可导出 $a_n$ 满足四阶线性递推。

**验证数据**：

| $n$ | $a_n$ | 说明 |
|-----|-------|------|
| 1 | 0 | 奇数步不可能回到起点（立方体是二部图） |
| 2 | 3 | 从 $000$ 出发沿一条边再原路返回，共 3 条边 |
| 3 | 0 | 奇数步 |
| 4 | 21 | 长度 4 的回路 |
| 5 | 0 | 奇数步 |
| 6 | 183 | |

```cpp
// 立方体 4×4 矩阵快速幂
ll solve_cube(ll n) {
    Mat M(4);
    M.a[0][1] = 3;
    M.a[1][0] = 1; M.a[1][2] = 2;
    M.a[2][1] = 2; M.a[2][3] = 1;
    M.a[3][2] = 3;
    Mat P = mat_qpow(M, n);
    return P.a[0][0];   // a_0 = 1
}
```

## 三、允许停留——自环的引入

允许蚂蚁在每步选择留在原地，等价于为每个顶点添加自环，即矩阵变为 $A + I$。本节展示此举如何从根本上改变问题结构。

### 3.1 基本观察

在禁止停留下，完全图 $K_m$ 的 $a_n$ 呈现 $(m-1)^n \pm \cdots$ 的形式，奇偶步交替震荡。添加自环后：

- 每一步从每个顶点出发均有 $m$ 个选择（$m-1$ 条出边 + 1 个自环）。
- 若图的邻接矩阵变为 $J$（全 1 矩阵，因为任意两顶点均相邻且带自环），则 $(J^n)_{ij} = m^{\,n-1}$ 对所有 $i,j$ 成立。

### 3.2 三角形（$K_3$，允许停留）

**转移方程**（每个顶点有 3 个选择：去往两个邻居或停留）：

- $a_n = a_{n-1} + 2 \cdot b_{n-1}$：从 $S$ 可停留（贡献 $a_{n-1}$）；从两个 $A$ 各可走一步到 $S$（贡献 $2 \cdot b_{n-1}$）。
- $b_n = a_{n-1} + 2 \cdot b_{n-1}$：从此 $A$ 可停留（$b_{n-1}$）；从 $S$ 可走到此 $A$（$a_{n-1}$）；从另一 $A$ 可走到此 $A$（$b_{n-1}$），合计 $a_{n-1} + 2b_{n-1}$。

**矩阵**：

$$
\begin{bmatrix} a_n \\ b_n \end{bmatrix}
=
\begin{bmatrix} 1 & 2 \\ 1 & 2 \end{bmatrix}
\begin{bmatrix} a_{n-1} \\ b_{n-1} \end{bmatrix}
$$

**闭式**：由转移方程可见 $a_n = a_{n-1} + 2b_{n-1}$ 且 $b_n = a_{n-1} + 2b_{n-1}$，因此对 $n \ge 1$ 恒有 $a_n = b_n$（注意 $a_0 = 1 \neq b_0 = 0$）。由 $a_n = a_{n-1} + 2a_{n-1} = 3a_{n-1}$（$n \ge 2$）及 $a_1 = 1$ 得：

$$
\boxed{a_n = 3^{\,n-1} \quad (n \ge 1)}
$$

**验证**：$n=1$：$a_1 = 1$（停留在 $S$）。$n=2$：$a_2 = 3$（$S \to S \to S$；$S \to A \to S$；$S \to B \to S$）。$n=3$：$a_3 = 9$，总方案数随 $3^n$ 增长。

### 3.3 正方形环（$C_4$，允许停留）

每个顶点度数由 $2$ 变为 $3$（加上自环）。转移方程：

- $a_n = a_{n-1} + 2 \cdot b_{n-1}$（在 $S$ 停留，或从两个 $P$ 类顶点走一步来到 $S$）
- $b_n = a_{n-1} + b_{n-1} + c_{n-1}$（从此 $P$ 停留；从 $S$ 走来；从对应的 $Q$ 走来）
- $c_n = c_{n-1} + 2 \cdot b_{n-1}$（在 $Q$ 停留，或从两个 $P$ 类顶点走一步来到 $Q$）

初始：$a_0 = 1$，$b_0 = c_0 = 0$。无简单闭式，使用矩阵快速幂。

**验证**：$n=1$：$a_1 = 1$（停在 $S$），$b_1 = 1$，$c_1 = 0$。$n=2$：$a_2 = 1 + 2 \cdot 1 = 3$（停留-停留、$0 \to 1 \to 0$、$0 \to 3 \to 0$）。

### 3.4 完全图 $K_m$ 的通用公式

**允许停留**时每个顶点有 $m$ 个选择（含自环），邻接矩阵为全 1 矩阵 $J$：

$$
(J^n)_{ij} = m^{\,n-1} \quad (\forall\, i,j)
$$

因此：

$$
\boxed{a_n = \begin{cases}
1, & n = 0 \\
m^{\,n-1}, & n \ge 1
\end{cases}}
$$

**禁止停留**时需区分两种情况：

$$
\boxed{a_n = \begin{cases}
1, & n = 0 \\
0, & n = 1 \\
\displaystyle\frac{(m-1)^n + (m-1) \cdot (-1)^n}{m}, & n \ge 2
\end{cases}}
$$

这两个公式的对比直观地展现了「自环」对图的连通结构的深刻改变：允许停留抹平了奇偶步的差异。

```cpp
// K_m 通用求解（允许/禁止停留）
ll solve_km(ll m, ll n, bool allow_stay) {
    if (n == 0) return 1;
    if (allow_stay) return qpow(m % MOD, n - 1, MOD);
    if (n == 1) return 0;
    ll p = qpow((m - 1) % MOD, n, MOD);
    ll sign = (n & 1) ? MOD - 1 : 1;
    ll num = (p + (m - 1) * sign) % MOD;
    return num * qpow(m % MOD, MOD - 2, MOD) % MOD;
}
```

## 四、进阶讨论

### 4.1 无对称性时怎么办

当图不具有可利用的对称性时，无法约简为少数轨道。两种处理方式：

1. **直接矩阵快速幂**：对 $m \times m$ 的邻接矩阵 $A$（或 $A+I$）求幂，$O(m^3 \log n)$。适合 $m \le 100$ 且 $n$ 极大。
2. **DP + 滚动数组**：$O(m^2 n)$，适合 $n \le 10^7$ 且 $m$ 不大。

```cpp
// 通用求解：对任意邻接矩阵做矩阵快速幂
ll solve_general(const vector<vector<ll>>& adj, int s, ll n, bool stay) {
    int m = adj.size();
    Mat M(m);
    for (int i = 0; i < m; ++i) {
        for (int j = 0; j < m; ++j) {
            M.a[i][j] = adj[i][j];
        }
        if (stay) M.a[i][i] = (M.a[i][i] + 1) % MOD;
    }
    Mat P = mat_qpow(M, n);
    return P.a[s][s];
}
```

### 4.2 有向图推广

若有向图每条边是单向的，邻接矩阵 $A$ 不再对称。但 $(A^n)_{ij}$ 的路径计数公式不变，矩阵快速幂同样适用。需注意有向图中「回到起点」可能远比无向图困难（必须存在有向回路覆盖起点）。

### 4.3 与随机游走的关系

若蚂蚁每一步**等概率**选择出边（或含自环），则：
- 邻接矩阵需归一化为转移概率矩阵 $P$，$P_{ij} = 1 / \deg(i)$（或 $(1 + \deg(i))^{-1}$ 含自环）。
- $n$ 步后位于各顶点的概率分布 = $v_0 \cdot P^n$。
- 平均返回时间、平稳分布等与马尔可夫链理论紧密相连，详见 [第 69 篇：吸收马尔可夫链](../04-概率与期望/69-吸收马尔可夫链.md) 和 [第 71 篇：马尔可夫链与高斯消元](../04-概率与期望/71-马尔可夫链与高斯消元.md)。

## 五、常见误区

1. **「$n=1$ 时禁止停留回到起点的方案数为 $0$」——仅当起点无自环时成立。** 若图本身有自环或允许停留，$n=1$ 可回到起点。

2. **「$A^n$ 的对角线就是答案」——需注意起点。** $(A^n)_{ii}$ 是顶点 $i$ 到 $i$ 的方案数。若起点非 $i$，应取 $(A^n)_{si}$。

3. **「图连通等价于必有回到起点的方案」——不一定。** $C_4$ 中奇数步回到起点的方案数为 $0$（二部图性质）。一般地，若图是二部图，奇数步不可能回到起点。

4. **「允许停留等价于 $A+I$」——仅当每步都允许停留。** 若只在某些顶点允许停留，需构造不同的矩阵。

5. **「对称性约简总是有效」——依赖于起点。** 若起点破坏了某自同构（例如在非对称位置），某些轨道可能分裂。

6. **「闭式公式总是存在」——仅对高度对称的图。** 一般图只能通过矩阵快速幂或 DP 数值求解，闭式依赖于图的谱分解的简洁性。

## 六、总结

本文从邻接矩阵的代数性质出发，系统地解决了一类图上游走计数问题：

| 图 | 禁止停留 $a_n$ | 允许停留 $a_n$（$n\ge 1$） |
|----|---------------|--------------------------|
| $K_3$ | $(2^n + 2(-1)^n)/3$ | $3^{\,n-1}$ |
| $C_4$ | $0$（奇）/ $2^{\,n-1}$（偶） | 矩阵 $3 \times 3$ DP |
| $K_4$ | $(3^n + 3(-1)^n)/4$ | $4^{\,n-1}$ |
| 立方体 | 4 状态矩阵 DP | 4 状态矩阵 DP |
| $K_m$ | $\frac{(m-1)^n + (m-1)(-1)^n}{m}$ | $m^{\,n-1}$ |

核心工具链：邻接矩阵 → 对称性约简 → 低维递推 → 滚动数组 / 矩阵快速幂。这套方法论可应用于任意具有对称性的图。
