# 杜教筛

> Made by Aki.
> 最后更新于 2026.04.30

第 22 篇的线性筛可以在 $O(n)$ 时间内预处理积性函数。但如果 $n = 10^{11}$ 呢？数组开不下。杜教筛（Du Sieve）以 $O(n^{2/3})$ 的亚线性复杂度计算积性函数前缀和——不需要完整数组。

---

## 一、引言

求 $\sum_{i=1}^{n} \varphi(i)$，$n = 10^{11}$。

线性筛需要 $O(n)$ 的空间和时间，$10^{11}$ 的数组在任何机器上都放不下。但杜教筛的解法是这样的：通过构造适当的 Dirichlet 卷积，将大范围求和递归分解为小范围——每次递归将 $n$ 缩小到 $\sqrt{n}$。递归树的记忆化搜索配合整除分块，总复杂度 $O(n^{2/3})$。

## 三、核心推导

### 3.1 问题设定

给定积性函数 $f$，求前缀和 $S(n) = \sum_{i=1}^{n} f(i)$。

杜教筛适用的前提：能找到一个 $g$ 使得 $h = f * g$（Dirichlet 卷积）的前缀和容易计算。

---

### 3.2 核心恒等式

由 Dirichlet 卷积展开：

$$
\sum_{i=1}^{n} h(i) = \sum_{i=1}^{n} \sum_{d \mid i} f(d) \cdot g\left(\frac{i}{d}\right)
$$

交换求和次序（先枚举 $d$，再枚举 $d$ 的倍数 $i = d \cdot j$）：

$$
\sum_{i=1}^{n} h(i) = \sum_{d=1}^{n} g(d) \sum_{j=1}^{\lfloor n/d \rfloor} f(j) = \sum_{d=1}^{n} g(d) \cdot S\left(\left\lfloor \frac{n}{d} \right\rfloor\right)
$$

分离 $d=1$ 的项：

$$
\sum_{i=1}^{n} h(i) = g(1) \cdot S(n) + \sum_{d=2}^{n} g(d) \cdot S\left(\left\lfloor \frac{n}{d} \right\rfloor\right)
$$

解出 $S(n)$（假设 $g(1) \neq 0$）：

$$
\boxed{S(n) = \frac{\displaystyle\sum_{i=1}^{n} h(i) - \sum_{d=2}^{n} g(d) \cdot S\left(\left\lfloor \frac{n}{d} \right\rfloor\right)}{g(1)}}
$$

---

### 3.3 具体公式

**$\mu$ 的前缀和**：取 $g = 1$（常数函数），$h = \mu * 1 = \epsilon$。$\sum h(i) = 1$。$g(1) = 1$。

$$
S_\mu(n) = 1 - \sum_{d=2}^{n} S_\mu\left(\left\lfloor \frac{n}{d} \right\rfloor\right)
$$

**$\varphi$ 的前缀和**：取 $g = 1$，$h = \varphi * 1 = I$（恒等函数）。$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$。$g(1) = 1$。

$$
S_\varphi(n) = \frac{n(n+1)}{2} - \sum_{d=2}^{n} S_\varphi\left(\left\lfloor \frac{n}{d} \right\rfloor\right)
$$

### 3.4 算法实现思路

直接递归计算 $S(n)$ 仍然需要 $O(n)$ 时间。关键优化是**记忆化 + 整除分块**：

1. 用线性筛预处理 $S(1)$ 到 $S(K)$（$K$ 取 $n^{2/3}$ 左右）。
2. 对 $> K$ 的 $n'$，递归计算并用 `unordered_map` 记忆化。
3. 每次递归中用整除分块（第 27 篇）将 $\sum_{d=2}^{n}$ 划分为 $O(\sqrt{n})$ 段。

总复杂度：$O(K + n/\sqrt{K})$。取 $K = n^{2/3}$ 时最优，总复杂度 $O(n^{2/3})$。

## 四、代码实现

```cpp
unordered_map<ll, ll> mem_phi, mem_mu;
const int K = 5000000;  // 预处理上限（n^{2/3} ≈ 5e6 for n=1e10）
vector<ll> phi_sum, mu_sum;

void precompute(int n) {
    // 线性筛 φ 和 μ（第 22 篇），然后前缀和
    phi_sum.resize(n + 1); mu_sum.resize(n + 1);
    phi_sum[1] = mu_sum[1] = 1;
    // ...（省略筛的过程，见第 22 篇完整代码）
    for (int i = 2; i <= n; i++) {
        phi_sum[i] = phi_sum[i-1] + phi[i];
        mu_sum[i] = mu_sum[i-1] + mu[i];
    }
}

ll S_phi(ll n) {
    if (n <= K) return phi_sum[n];
    if (mem_phi.count(n)) return mem_phi[n];
    ll ans = n * (n + 1) / 2;   // Σ h = Σ id
    for (ll l = 2, r; l <= n; l = r + 1) {
        r = n / (n / l);
        ans -= (r - l + 1) * S_phi(n / l);
    }
    return mem_phi[n] = ans;
}

ll S_mu(ll n) {
    if (n <= K) return mu_sum[n];
    if (mem_mu.count(n)) return mem_mu[n];
    ll ans = 1;                  // Σ h = Σ ε = 1
    for (ll l = 2, r; l <= n; l = r + 1) {
        r = n / (n / l);
        ans -= (r - l + 1) * S_mu(n / l);
    }
    return mem_mu[n] = ans;
}
```
