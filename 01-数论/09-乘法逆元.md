# 乘法逆元

> Made by Aki.
> 最后更新于 2026.04.29

模运算的世界里，加法、减法、乘法都顺畅无阻——但除法是禁区。乘法逆元就是为"模意义下的除法"发放的通行证。

---

## 一、引言

看一道题：

> 求 $\binom{1000}{500} \bmod 10^9+7$。

组合数公式 $\binom{n}{k} = \frac{n!}{k!(n-k)!}$ 里有一个除法。但第 1 篇我们强调过：**模除法不能直接把分子分母分别取模再除**。

那怎么办？答案是将「除以 $a$」转化为「乘以 $a$ 的**逆元**」：

$$
\frac{A}{B} \bmod m = A \times B^{-1} \bmod m
$$

其中 $B^{-1}$ 是满足 $B \times B^{-1} \equiv 1 \pmod{m}$ 的数——这就是乘法逆元。本文将介绍求逆元的三种方法及其适用场景。

## 三、核心推导

### 3.1 逆元的定义与存在条件

> 在模 $m$ 意义下，如果存在整数 $x$ 使得
>
> $$a \times x \equiv 1 \pmod{m}$$
>
> 则 $x$ 称为 $a$ 在模 $m$ 下的**乘法逆元**，记作 $a^{-1} \pmod{m}$。

**存在条件**：$a^{-1} \pmod{m}$ 存在当且仅当 $\gcd(a, m) = 1$。

这个条件可以等价地理解——同余方程 $ax \equiv 1 \pmod{m}$ 等价于不定方程 $ax + my = 1$。由裴蜀定理（第 3 篇），它有解的充要条件是 $\gcd(a, m) \mid 1$，也就是 $\gcd(a, m) = 1$。

这就解释了为什么竞赛中常选用 $10^9+7$ 或 $998244353$ 作为模数——它们都是素数，只要 $a$ 不是模数的倍数（即 $a \bmod p \neq 0$），逆元就必然存在。

---

### 3.2 法一：费马小定理（模数为素数）

当模数 $p$ 是素数且 $\gcd(a, p) = 1$（即 $a$ 不是 $p$ 的倍数）时，费马小定理告诉我们：

$$
a^{p-1} \equiv 1 \pmod{p}
$$

于是：

$$
a^{p-2} \times a \equiv a^{p-1} \equiv 1 \pmod{p}
$$

这说明 $a^{p-2}$ 就是 $a$ 的逆元：

$$
a^{-1} \equiv a^{p-2} \pmod{p}
$$

**代码**：直接调用第 7 篇的快速幂，一行搞定。

```cpp
ll inv_fermat(ll a, ll p) {
    return qpow(a, p - 2, p);
}
```

**适用场景**：模数为素数时的单次求逆。时间复杂度 $O(\log p)$，代码量是所有方法中最少的。

**局限性**：模数必须为素数。当模数不是素数时（如组合数取模中的 exLucas 场景），需要换用 exgcd 方法。

---

### 3.3 法二：扩展欧几里得（模数任意，只需 $\gcd = 1$）

逆元方程 $ax \equiv 1 \pmod{m}$ 改写为：

$$
ax + my = 1
$$

这正是 exgcd 的标准形式。当 $\gcd(a, m) = 1$ 时，exgcd 解出的 $x$ 就是 $a$ 的逆元。

```cpp
ll inv_exgcd(ll a, ll m) {
    ll x, y;
    exgcd(a, m, x, y);
    return (x % m + m) % m;  // 调整到 [0, m)
}
```

**适用场景**：模数不一定是素数，只要 $\gcd(a, m) = 1$ 就能求逆。时间复杂度同样是 $O(\log m)$。

对比费马法：费马法只需一行代码、不引入额外函数，但限制模数为素数。exgcd 法通用性更强，但代码稍长。在 $p$ 为素数的竞赛题目中二者等价，按习惯选用即可。

---

### 3.4 法三：线性递推——批量求 $1$ 到 $n$ 的所有逆元

洛谷 P3811 是一个经典场景：$n$ 可达 $3 \times 10^6$，如果对每个 $i$ 单独调用一次 $O(\log p)$ 的方法，总时间 $O(n \log p)$ 不可接受。有没有 $O(n)$ 的方法？

有——而且推导过程非常漂亮。

**推导**：

设模数为 $p$。对任意 $i \ge 2$，写出带余除法：

$$
p = \left\lfloor \frac{p}{i} \right\rfloor \times i + (p \bmod i)
$$

记 $k = \lfloor p/i \rfloor$，$r = p \bmod i$。则在模 $p$ 下：

$$
k \cdot i + r \equiv 0 \pmod{p}
$$

两边同时乘以 $i^{-1} \cdot r^{-1}$：

$$
k \cdot r^{-1} + i^{-1} \equiv 0 \pmod{p}
$$

移项：

$$
i^{-1} \equiv -k \cdot r^{-1} \pmod{p}
$$

代回 $k$ 和 $r$：

$$
\boxed{inv[i] \equiv -\left\lfloor \frac{p}{i} \right\rfloor \times inv[p \bmod i] \pmod{p}}
$$

由于 $p \bmod i < i$，$inv[p \bmod i]$ 已经在之前的循环中计算过了——这正是递推成立的关键。

转换为非负形式（避免负数取模的坑）：

$$
\boxed{inv[i] = (p - p \mathbin{/} i) \times inv[p \mathbin{\%} i] \mathbin{\%} p}
$$

**边界**：$inv[1] = 1$。

**复杂度**：每个 $i$ 只需一次乘法、一次取模，总时间 $O(n)$。

线性递推法利用 $p = ki + r$ 的带余关系实现了对 $1$ 到 $n$ 的 $O(n)$ 求逆，但它依赖「$p \bmod i < i$」这一特殊性质——这意味着它只能处理连续整数 $1, 2, \dots, n$ 的逆元。如果输入是任意 $n$ 个互不相干的数 $a_1, a_2, \dots, a_n$，线性递推就无法直接套用了。

下面介绍一种同样能做到 $O(n + \log p)$ 的「前缀积 + 后缀逆元」法，它不要求 $a_i$ 之间有特定的递推关系，适用于任意 $n$ 个数的批量求逆场景。

### 3.5 法三变体：前缀积 + 后缀逆元——任意 $n$ 个数的批量求逆

#### 问题引入：洛谷 P5431

> 给定 $n, p, k$ 和 $n$ 个正整数 $a_i$（$1 \le i \le n$），求
>
> $$\sum_{i=1}^{n} \frac{k^i}{a_i} \pmod{p}$$
>
> 其中 $p$ 为素数。$1 \le n \le 5 \times 10^6$，单组输入。

这个问题的核心在于对每个 $a_i$ 求逆元 $a_i^{-1}$。最直接的做法是对每个 $a_i$ 单独调用一次费马小定理（$O(\log p)$），总复杂度 $O(n \log p)$。当 $n$ 达到 $5 \times 10^6$ 时，$10^7$ 数量级的快速幂运算不可接受——我们需要一个 $O(n + \log p)$ 的方法。

#### 算法推导

设 $P_i = \prod_{j=1}^{i} a_j$ 为前 $i$ 个数的前缀积（定义 $P_0 = 1$）。核心思想分为两步：

1. 先求出所有 $n$ 个数的总积 $P_n$，用费马小定理求出 $P_n^{-1}$（一次 $O(\log p)$ 的快速幂）。
2. 利用 $P_n^{-1}$ 倒推出每个 $a_i^{-1}$。

推导的关键在于以下观察。记后缀积 $S_i = \prod_{j=i}^{n} a_j$（$S_{n+1} = 1$），那么 $S_1 = P_n$。对任意 $i$，有：

$$
a_i^{-1} \equiv P_{i-1} \times S_i^{-1} \pmod{p}
$$

验证：

$$
\begin{aligned}
a_i \times (P_{i-1} \times S_i^{-1})
&\equiv P_i \times S_i^{-1} \\
&\equiv P_n \times P_n^{-1} \equiv 1 \pmod{p}
\end{aligned}
$$

而 $S_i^{-1}$ 可以通过 $P_n^{-1}$ 递推得到——从 $i = n$ 到 $i = 1$：

$$
S_i^{-1} \equiv P_n^{-1} \times P_{i-1} \pmod{p}, \qquad
S_{i-1}^{-1} \equiv S_i^{-1} \times a_i \pmod{p}
$$

#### 算法流程

1. **前缀积**：$P_i = P_{i-1} \times a_i \bmod p$，$i = 1 \dots n$。
2. **总积的逆元**：$P_n^{-1} \equiv P_n^{p-2} \pmod{p}$，一次 $O(\log p)$ 快速幂。
3. **后缀递推**：从 $i = n$ 递减到 $1$，维护变量 $suf = S_i^{-1}$（初始 $suf = P_n^{-1}$）：
   - $a_i^{-1} = P_{i-1} \times suf \bmod p$
   - $suf = suf \times a_i \bmod p$（得到 $S_{i-1}^{-1}$，供下一轮使用）

#### 复杂度分析

- 时间复杂度：$O(n + \log p)$。前缀积 $O(n)$，一次快速幂 $O(\log p)$，后缀递推 $O(n)$。
- 空间复杂度：$O(n)$，需要存储 $a$ 数组和前缀积数组。

#### 代码实现

参见第四章「代码实现」中的 `inv_any` 函数：

```cpp
vector<ll> inv_any(const vector<ll> &a, ll p) {
    int n = a.size() - 1;  // 假设 a[1..n] 有效
    vector<ll> pre(n + 2, 1), suf_inv(n + 2, 1), inv(n + 1);
    for (int i = 1; i <= n; i++)
        pre[i] = pre[i - 1] * a[i] % p;
    suf_inv[n] = inv_fermat(pre[n], p);   // P_n^{-1}
    for (int i = n; i >= 1; i--) {
        inv[i] = pre[i - 1] * suf_inv[i] % p;
        suf_inv[i - 1] = suf_inv[i] * a[i] % p;
    }
    return inv;
}
```

> **与 3.4 线性递推的联系**：两种方法的共同点在于将 $n$ 次求逆降为 $1$ 次，剩余的工作由递推完成。区别在于：3.4 法利用整数序列 $1 \dots n$ 的特殊性质（$p \bmod i < i$），不占用额外空间；而 3.5 法通过前缀积和后缀逆元的双向构造来「绕过」$a_i$ 之间的无关性，灵活性更高但需要 $O(n)$ 的额外空间。

---

### 3.6 三种方法对比

| 方法 | 时间复杂度 | 模数要求 | 适用场景 |
|------|-----------|----------|----------|
| 费马小定理 | $O(\log p)$ 单次 | $p$ 为素数 | 模素数的单次/少量求逆 |
| exgcd | $O(\log m)$ 单次 | $\gcd(a,m)=1$ | 模数不限的单次求逆 |
| 线性递推 | $O(n)$ 批量 | $\gcd(i,p)=1$ | 求 $1$ 到 $n$ 所有逆元 |
| 前缀积+后缀逆元 | $O(n + \log p)$ 批量 | $p$ 为素数 | 任意 $n$ 个数的批量求逆 |

在绝大多数 ACM 题目中模数是素数，费马法是最简洁的选择。批量求逆时线性递推是唯一可行的方案。exgcd 法的能力最强（模数可以是合数），但代码量稍大。

## 四、代码实现

```cpp
#include <bits/stdc++.h>
using ll = long long;

// 法一：费马小定理求逆元（p 为素数）
ll inv_fermat(ll a, ll p) {
    return qpow(a, p - 2, p);       // qpow 来自第 7 篇
}

// 法二：扩展欧几里得求逆元（只需 gcd(a, m) = 1）
ll inv_exgcd(ll a, ll m) {
    ll x, y;
    exgcd(a, m, x, y);              // exgcd 来自第 8 篇
    return (x % m + m) % m;
}

// 法三：线性递推批量求逆元（p 为素数，求 1..n 的逆）
vector<ll> inv_range(int n, ll p) {
    vector<ll> inv(n + 1);
    inv[1] = 1;
    for (int i = 2; i <= n; i++)
        inv[i] = (p - p / i) * inv[p % i] % p;
    return inv;
}

// 法三变体：任意 n 个数的批量求逆（前缀积 + 后缀逆元，O(n + log p)）
vector<ll> inv_any(const vector<ll> &a, ll p) {
    int n = a.size() - 1;  // 假设 a[1..n] 有效
    vector<ll> pre(n + 2, 1), suf_inv(n + 2, 1), inv(n + 1);
    for (int i = 1; i <= n; i++)
        pre[i] = pre[i - 1] * a[i] % p;
    suf_inv[n] = inv_fermat(pre[n], p);   // 总积的逆元
    for (int i = n; i >= 1; i--) {
        inv[i] = pre[i - 1] * suf_inv[i] % p;
        suf_inv[i - 1] = suf_inv[i] * a[i] % p;
    }
    return inv;
}
```

使用示例：

```cpp
ll p = 1000000007;

// 单次求逆
cout << inv_fermat(123, p) << endl;     // 123 在模 1e9+7 下的逆元

// 批量求 1..10 的逆元
auto inv = inv_range(10, p);
cout << inv[5] << endl;                 // 5^{-1} mod 1e9+7
// inv[1..10] = {1, 500000004, 333333336, ..., 700000005}
```

## 五、复杂度与正确性分析

| 方法 | 复杂度 | 关键依赖 |
|------|--------|----------|
| 费马 | $O(\log p)$ | 费马小定理 $a^{p-1} \equiv 1$ + 快速幂 |
| exgcd | $O(\log m)$ | 裴蜀定理 $ax+my=1$ 有解 + exgcd 回溯构造 |
| 线性递推 | $O(n)$ | $p = ki + r$ 的带余除法 + 递推链 $r < i$ |
| 前缀积+后缀逆元 | $O(n + \log p)$ | 总积 $P_n$ 的逆元 + 后缀递推 $a_i^{-1} = P_{i-1} \times S_i^{-1}$ |

**正确性**：三种方法的正确性分别由费马小定理、裴蜀定理+exgcd、以及带余除法的模意义代数恒等变换保证。线性递推的关键在于 $inv[p \bmod i]$ 在 $inv[i]$ 之前已经算好——因为 $p \bmod i < i$，算术上自动满足递推序。

## 六、典型应用场景

乘法逆元是竞赛中「模意义除法」的入口。以下是它在后续篇章中的主要出场位置：

1. **组合数取模**（第 14 篇）：$\binom{n}{k} \equiv n! \times (k!)^{-1} \times ((n-k)!)^{-1}$，逆元用于处理除法的分母。

2. **中国剩余定理**（第 13 篇）：CRT 合并过程中需要 $M_i^{-1} \pmod{m_i}$，常用 exgcd 求逆。

3. **分数模输出**：很多题目输出形如「$\frac{A}{B} \bmod p$」的结果，本质就是 $A \times B^{-1} \bmod p$。

4. **概率期望题**：期望值以有理数给出，最终输出需要模逆元转化。

**推荐练习题**：

- [洛谷 P3811](https://www.luogu.com.cn/problem/P3811) — 【模板】乘法逆元。$n \le 3 \times 10^6$，必须用线性递推 $O(n)$，单次求逆会超时
- [洛谷 P5431](https://www.luogu.com.cn/problem/P5431) — 【模板】乘法逆元 2。任意 $n$ 个数的批量求逆，前缀积+后缀逆元法（详见 3.5 节）
- [Codeforces 300C](https://codeforces.com/problemset/problem/300/C) — Beautiful Numbers。枚举组合数 $\binom{n}{i}$ 并取模，综合运用组合数公式和乘法逆元

## 七、小结

乘法逆元将模运算的最后一块拼图——除法——补全了。在竞赛中，它通常以四种面孔出现：

- **费马法**：一行 `qpow(a, p-2, p)`，适合模素数的快速单次求逆
- **exgcd 法**：两行，普适性强，模数不必为素数
- **线性递推法**：`inv[i] = (p - p/i) * inv[p%i] % p`，$O(n)$ 批量求解，推导过程本身就是一道优美的数论题
- **前缀积+后缀逆元法**：先求总积逆元，再递推每个 $a_i^{-1}$，$O(n + \log p)$ 处理任意 $n$ 个数的逆元，以空间换时间

下一篇是**欧拉函数与欧拉定理**——我们将揭开 $\varphi(n)$ 的面纱。欧拉函数是费马小定理的推广（从素数走向任意整数），也是后续原根和离散对数篇章的基石。
