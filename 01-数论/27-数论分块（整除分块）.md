# 数论分块（整除分块）

> Made by Aki.
> 最后更新于 2026.04.30

$\lfloor n/i \rfloor$ 当 $i$ 从 $1$ 取到 $n$ 时，看似有 $n$ 个不同值，实际上只有 $O(\sqrt{n})$ 种。这个事实——以及利用它把 $O(n)$ 求和降到 $O(\sqrt{n})$ 的技巧——就是数论分块。

本篇是这个系列唯一的「纯技巧篇」。它本身不引入新的数学定理，但它在莫比乌斯反演、杜教筛、约数和等问题中以「最后一公里」的身份反复出现。本篇将按题型分类，逐一给出模板、推导和例题。

---

## 一、核心原理

### 1.1 为什么只有 $O(\sqrt{n})$ 种取值？

对于 $i \le \sqrt{n}$：最多 $\sqrt{n}$ 种不同的 $i$，自然最多 $\sqrt{n}$ 种 $\lfloor n/i \rfloor$。

对于 $i > \sqrt{n}$：此时 $\lfloor n/i \rfloor < \sqrt{n}$，最多 $\sqrt{n}$ 种不同的值。

所以总共不超过 $2\sqrt{n}$ 种取值。它们连续分布在不同的「块」中。

### 1.2 块边界的计算

对于确定的左端点 $l$，令 $k = \lfloor n/l \rfloor$（当前块的值）。那么 $\lfloor n/i \rfloor = k$ 的最大 $i$ 为：

$$
\boxed{r = \left\lfloor \frac{n}{k} \right\rfloor = \left\lfloor \frac{n}{\lfloor n/l \rfloor} \right\rfloor}
$$

**推导**：由 $k = \lfloor n/l \rfloor \le n/l$ 得 $l \le n/k$。$\lfloor n/i \rfloor = k$ 当且仅当 $k \le n/i < k+1$，即 $n/(k+1) < i \le n/k$。所以最大 $i$ 是 $\lfloor n/k \rfloor$。

## 二、题型全解

### 题型一：基础求和 $\sum \lfloor n/i \rfloor$

**模板题**：[洛谷 P1403 - 约数研究](https://www.luogu.com.cn/problem/P1403)

直接套用块边界公式：

```cpp
ll sum_floor(ll n) {
    ll ans = 0;
    for (ll l = 1, r; l <= n; l = r + 1) {
        ll k = n / l;
        r = n / k;                        // 块的右端点
        ans += k * (r - l + 1);           // 块值 × 块长
    }
    return ans;
}
```

时间复杂度 $O(\sqrt{n})$，$n = 10^{12}$ 时可轻松计算。

### 题型二：余数求和 $\sum k \bmod i$

**模板题**：[洛谷 P2261 [CQOI2007] 余数求和](https://www.luogu.com.cn/problem/P2261)

核心转化：

$$
k \bmod i = k - i \cdot \left\lfloor \frac{k}{i} \right\rfloor
$$

$$
\sum_{i=1}^{n} k \bmod i = n \cdot k - \sum_{i=1}^{n} i \cdot \left\lfloor \frac{k}{i} \right\rfloor
$$

块内 $i$ 用等差数列求和：

$$
\sum_{i=l}^{r} i = \frac{(l + r)(r - l + 1)}{2}
$$

**完整代码**：

```cpp
ll mod_sum(ll n, ll k) {
    ll ans = n * k;
    for (ll l = 1, r; l <= n; l = r + 1) {
        ll val = k / l;
        if (val == 0) break;            // 之后的项都是 0
        r = min(k / val, n);            // 不能超过 n
        ans -= val * (l + r) * (r - l + 1) / 2;
    }
    return ans;
}
```

**变式**：[洛谷 R5E 余数和](https://www.luogu.com.cn/problem/B4013)（2025 年新题）——$N \le 10^9$，同样思路。

### 题型三：约数个数/约数和前缀

**约数个数前缀** —— $\sum_{i=1}^{n} d(i)$

枚举每个可能的因数 $d$：$d$ 在 $1$ 到 $n$ 中作为因数恰好出现了 $\lfloor n/d \rfloor$ 次。

$$
\sum_{i=1}^{n} d(i) = \sum_{d=1}^{n} \left\lfloor \frac{n}{d} \right\rfloor
$$

直接套题型一的模板。**例题**：[洛谷 P1403 [AHOI2005] 约数研究](https://www.luogu.com.cn/problem/P1403)

**约数和前缀** —— $\sum_{i=1}^{n} \sigma(i)$

每个因数 $d$ 贡献 $d \times \lfloor n/d \rfloor$：

$$
\sum_{i=1}^{n} \sigma(i) = \sum_{d=1}^{n} d \cdot \left\lfloor \frac{n}{d} \right\rfloor
$$

```cpp
ll divisor_sum_prefix(ll n) {
    ll ans = 0;
    for (ll l = 1, r; l <= n; l = r + 1) {
        ll k = n / l;
        r = n / k;
        ans += k * (l + r) * (r - l + 1) / 2;  // k * Σ d
    }
    return ans;
}
```

**例题**：[洛谷 P2424 约数和](https://www.luogu.com.cn/problem/P2424) —— 求区间 $[x, y]$ 内各数的约数和之和。`calc(y) - calc(x-1)` 差分即可。

**例题**：[洛谷 P3935 Calculating](https://www.luogu.com.cn/problem/P3935) —— 求 $[l, r]$ 各数因数个数之和，模 $998244353$。

### 题型四：二维整除分块

同时涉及 $\lfloor n/i \rfloor$ 和 $\lfloor m/i \rfloor$ 时，取两者右端点的最小值作为块的右端点。

**模板题**：[洛谷 P3327 [SDOI2015] 约数个数和](https://www.luogu.com.cn/problem/P3327)

求 $\sum_{i=1}^{n} \sum_{j=1}^{m} d(ij)$。经过推导（过程略，见洛谷题解区），可以化为：

$$
\sum_{d=1}^{\min(n,m)} \mu(d) \cdot f\left(\left\lfloor \frac{n}{d} \right\rfloor\right) \cdot f\left(\left\lfloor \frac{m}{d} \right\rfloor\right)
$$

其中 $f$ 涉及整除分块。

**二维分块模板**：

```cpp
for (ll l = 1, r; l <= min(n, m); l = r + 1) {
    ll k1 = n / l, k2 = m / l;
    r = min(n / k1, m / k2);                // 取两块右端点的较小值
    // 块 [l, r] 内 ⌊n/i⌋ 和 ⌊m/i⌋ 均不变
    ans += (sum[r] - sum[l-1]) * f(k1) * f(k2);
}
```

**例题**：[洛谷 P2260 [清华集训2012] 模积和](https://www.luogu.com.cn/problem/P2260) —— 二维余数求和的容斥型分块。

### 题型五：向上取整分块

向上取整 $\lceil n/i \rceil$ 可以通过恒等式转化为向下取整：

$$
\left\lceil \frac{n}{i} \right\rceil = \left\lfloor \frac{n - 1}{i} \right\rfloor + 1
$$

块边界：左端点 $l$，$k = \lceil n/l \rceil$，右端点为 $\lfloor (n-1)/(k-1) \rfloor$（$k > 1$ 时）。$k=1$ 时 $r = n$。

**常考场景**：需要保证 $i$ 至少为某个倍数时的分段，经常出现在与周期性相关的题目中。

---

### 题型六：整除分块嵌套 + 莫比乌斯反演

这是整除分块最常见的「大场面」出场——在莫比乌斯反演化简后的式子上做整除分块优化。

**例题**：[洛谷 P3455 [POI2007] ZAP-Queries](https://www.luogu.com.cn/problem/P3455)

求 $\sum_{i=1}^{a} \sum_{j=1}^{b} [\gcd(i, j) = d]$。

经莫比乌斯反演后化为：

$$
\sum_{t=1}^{\lfloor a/d \rfloor} \mu(t) \cdot \left\lfloor \frac{a}{dt} \right\rfloor \cdot \left\lfloor \frac{b}{dt} \right\rfloor
$$

预处理 $\mu$ 的前缀和，然后 $k_1 = a/(dt)$ 和 $k_2 = b/(dt)$ 分别用整除分块：

```cpp
ll solve(ll a, ll b, ll d) {
    a /= d; b /= d;
    ll ans = 0;
    for (ll l = 1, r; l <= min(a, b); l = r + 1) {
        ll k1 = a / l, k2 = b / l;
        r = min(a / k1, b / k2);
        ans += (sum_mu[r] - sum_mu[l - 1]) * k1 * k2;
    }
    return ans;
}
```

**例题**：[洛谷 P2522 [HAOI2011] Problem B](https://www.luogu.com.cn/problem/P2522) —— 区间版的 ZAP。

---

### 题型七：类欧几里得中的整除分块

类欧几里得算法（第 18 篇）本质上是在 $O(\log n)$ 层次上做参数互换，而整除分块是在 $O(\sqrt{n})$ 层次上分段。两者结合可以处理更复杂的求和。

**例题**：[洛谷 P5170 【模板】类欧几里得算法](https://www.luogu.com.cn/problem/P5170) —— 虽然主要依赖类欧，但在与莫反联动时经常需要分块。

## 三、常见坑点与调试技巧

### 3.1 除零和越界

```cpp
ll k = n / l;
if (k == 0) break;        // 必须 break！否则下一步 n/k 是除零
r = min(n / k, n);        // 确保 r 不超过 n
```

### 3.2 溢出陷阱

$n$ 和 $k$ 乘积可能爆 `int`（即便 $n$ 本身在 `int` 范围，$(l+r)(r-l+1)$ 也可能）。务必全程使用 `long long`。

### 3.3 二维分块的 $r$ 取 min 位置容易写错

当两个下取整同时变化时，哪个先到达下一段决定了块的实际边界。**始终取两个 $r$ 的 min**。

### 3.4 分块循环变量的步进

标准写法 `l = r + 1`，不要写错成 `l = r`（死循环）或 `l = r - 1`（漏元素）。

## 四、进阶：利用数论分块的恒等式

### 4.1 嵌套下取整

$$
\left\lfloor \frac{\lfloor a/b \rfloor}{c} \right\rfloor = \left\lfloor \frac{a}{bc} \right\rfloor
$$

这个恒等式在推导双重分块时非常有用。

### 4.2 向上取整的完整公式

$$
\left\lceil \frac{n}{x} \right\rceil = \left\lfloor \frac{n - 1}{x} \right\rfloor + 1 = \left\lfloor \frac{n + x - 1}{x} \right\rfloor
$$

第三种形式（加上 $x-1$ 再下取整）在代码中最常用。

## 六、小结

数论分块的本质是「同一值连续分布」的利用。它能以 $O(\sqrt{n})$ 替代 $O(n)$ 的原因正在于 $\lfloor n/i \rfloor$ 在大量连续区间上维持不变。分块之后的块内求和可以套用等差数列公式、前缀和差分等组合数学技巧——这使它与莫比乌斯反演、杜教筛、约数求和等高频操作天然兼容。
