# 欧拉函数与欧拉定理

> Made by Aki.
> 最后更新于 2026.04.29

费马小定理说：当 $p$ 是素数时，$a^{p-1} \equiv 1 \pmod{p}$。但如果模数不是素数呢？欧拉函数 $\varphi(n)$ 正是将这条性质推广到任意正整数模数的关键。

---

## 一、引言

回顾第 9 篇，我们用费马小定理求逆元的前提是「模数必须是素数」。但竞赛中有些题目——比如 exLucas 定理（第 14 篇）——模数可能是合数。此时费马小定理失效，需要更强大的工具。

这就是欧拉定理：

$$
a^{\varphi(m)} \equiv 1 \pmod{m} \quad (\gcd(a, m) = 1)
$$

当 $m$ 是素数时 $\varphi(m) = m-1$，退化回费马小定理。本篇将系统推导 $\varphi(n)$ 的定义、计算公式、线性筛法，以及欧拉定理和它的实用扩展形式——指数降幂公式。

## 三、核心推导

### 3.1 欧拉函数的定义与意义

> **欧拉函数** $\varphi(n)$ 表示 $1$ 到 $n$ 中与 $n$ 互质的数的个数。即：
>
> $$
> \varphi(n) = \big|\{k \in [1, n] \mid \gcd(k, n) = 1\}\big|
> $$

例如：$\varphi(1) = 1$，$\varphi(8) = 4$（$\{1,3,5,7\}$），$\varphi(12) = 4$（$\{1,5,7,11\}$）。

对于素数 $p$，$1$ 到 $p-1$ 的所有数都与 $p$ 互质，所以 $\varphi(p) = p-1$。这恰好与费马小定理中的 $p-1$ 对上号——欧拉函数推广的第一步。

---

### 3.2 单点计算公式

如果已知 $n$ 的素数分解 $n = p_1^{e_1} p_2^{e_2} \cdots p_k^{e_k}$，那么：

$$
\boxed{\varphi(n) = n \prod_{p \mid n} \left(1 - \frac{1}{p}\right)}
$$

其中乘积取遍 $n$ 的所有不同素因子。

**推导（容斥原理）**：在 $1$ 到 $n$ 中，与 $n$ 不互质的数就是那些能被某个 $p_i$ 整除的数。用容斥原理：

$$
\varphi(n) = n - \sum_i \frac{n}{p_i} + \sum_{i<j} \frac{n}{p_i p_j} - \cdots = n \prod_{i} \left(1 - \frac{1}{p_i}\right)
$$

❓ **也可以从积性角度理解**：$\varphi(p^e) = p^e - p^{e-1} = p^e(1 - 1/p)$（在 $p^e$ 个数中，去掉所有 $p$ 的倍数）。由于 $\varphi$ 是积性函数（$\gcd(a,b)=1$ 时 $\varphi(ab) = \varphi(a)\varphi(b)$），$\varphi(n)$ 就是各素数幂的 $\varphi$ 值之积。

**单点计算代码**：用试除法分解 $n$，一边分解一边累乘 $(1 - 1/p)$。

```cpp
ll phi(ll n) {
    ll ans = n;
    for (ll p = 2; p * p <= n; p++) {
        if (n % p == 0) {
            ans = ans / p * (p - 1);   // ans *= (1 - 1/p)，先除后乘防溢出
            while (n % p == 0) n /= p;
        }
    }
    if (n > 1) ans = ans / n * (n - 1);  // 剩余的大素因子
    return ans;
}
```

时间复杂度 $O(\sqrt{n})$。

---

### 3.3 线性筛法——批量求 $\varphi(1)$ 到 $\varphi(n)$

需要 $1$ 到 $n$ 所有数的 $\varphi$ 值时，可以借助欧拉线性筛在 $O(n)$ 内批量计算。核心是利用 $\varphi$ 的积性：在筛的过程中根据 $i$ 是否被 $p$ 整除，分两种情况递推。

设 $p$ 为素数，$i$ 为当前外层循环的数：

- **情况一**：$p \nmid i$（即 $\gcd(i, p) = 1$）

  由积性：$\varphi(i \times p) = \varphi(i) \times \varphi(p) = \varphi(i) \times (p - 1)$。

- **情况二**：$p \mid i$（$p$ 是 $i$ 的因子）

  此时 $i$ 和 $p$ 不互质，不能用积性。但观察：$i \times p$ 和 $i$ 有完全相同的素因子集合（$i$ 已经包含了 $p$）。由公式 $\varphi(n) = n \prod (1 - 1/p)$：

  $$
  \varphi(i \times p) = (i \times p) \prod (1 - 1/q) = p \times \varphi(i)
  $$

  只多乘了一个因子 $p$。

于是：
$$
\varphi(i \times p) = \begin{cases}
\varphi(i) \times (p - 1) & p \nmid i \\[4pt]
\varphi(i) \times p & p \mid i
\end{cases}
$$

这与欧拉线性筛的外层循环完全兼容——在 `vis[i * p] = true` 之后加上一行更新 $\varphi(i \times p)$ 即可。

---

### 3.4 欧拉定理

> 若 $\gcd(a, m) = 1$，则
>
> $$
> a^{\varphi(m)} \equiv 1 \pmod{m}
> $$

**证明思路（简化剩余系法）**：

设 $S = \{ r_1, r_2, \dots, r_{\varphi(m)} \}$ 是模 $m$ 的简化剩余系（即所有与 $m$ 互质的剩余类）。当 $\gcd(a, m) = 1$ 时，将 $S$ 中每个元素乘以 $a$ 再取模 $m$，得到的集合 $aS = \{ ar_1 \bmod m, \dots, ar_{\varphi(m)} \bmod m \}$ 在模 $m$ 下仍然是 $S$ 的一个排列（因为乘以 $a$ 是一个双射——有逆元就可以"除回来"）。

因此两集合的乘积相等：

$$
\prod_{i=1}^{\varphi(m)} r_i \equiv \prod_{i=1}^{\varphi(m)} (a r_i) \equiv a^{\varphi(m)} \prod_{i=1}^{\varphi(m)} r_i \pmod{m}
$$

由于每个 $r_i$ 与 $m$ 互质，乘积也与 $m$ 互质，可以消去：

$$
1 \equiv a^{\varphi(m)} \pmod{m}
$$

这个证明与第 11 篇费马小定理的证明完全相同——只需把「$p-1$」替换为「$\varphi(m)$」。

---

### 3.5 扩展欧拉定理——指数降幂公式

在实际计算中，经常遇到指数 $b$ 极大的情况（如 $b$ 以字符串形式输入，位数可达 $10^5$）。此时不能直接算 $a^b \bmod m$，需要降幂：

> **扩展欧拉定理**：
>
> $$
> a^b \equiv \begin{cases}
> a^{b \bmod \varphi(m)} & \gcd(a, m) = 1 \\[6pt]
> a^b & b < \varphi(m) \\[6pt]
> a^{(b \bmod \varphi(m)) + \varphi(m)} & b \ge \varphi(m)
> \end{cases}
> \pmod{m}
> $$

**解释**：
- 情况 1（$\gcd = 1$）：就是普通欧拉定理的直接推论——指数可以模 $\varphi(m)$。
- 情况 3（$b \ge \varphi(m)$ 且 $\gcd$ 可能不为 $1$）：指数降为 $(b \bmod \varphi(m)) + \varphi(m)$。多加上一个 $\varphi(m)$ 是必需的——Counterexample：$a = 2, m = 4, b = 3$。$\varphi(4) = 2$，$b \bmod 2 = 1$。若只用 $a^1 \equiv 2 \pmod{4}$，而不是 $a^{1+2} \equiv a^3 \equiv 0 \pmod{4}$，就会出错。

**代码实现**：高精度读入 $b$ 时，一边读一边同时计算 $b$ 的真实值和 $b \bmod \varphi(m)$，同时维护一个 flag 标记 $b$ 是否 $\ge \varphi(m)$。

## 四、代码实现

### 4.1 单点计算 $\varphi(n)$（$O(\sqrt{n})$）

```cpp
ll phi(ll n) {
    ll ans = n;
    for (ll p = 2; p * p <= n; p++) {
        if (n % p == 0) {
            ans = ans / p * (p - 1);
            while (n % p == 0) n /= p;
        }
    }
    if (n > 1) ans = ans / n * (n - 1);
    return ans;
}
```

### 4.2 线性筛批量求 $\varphi$（$O(n)$）

```cpp
vector<int> phi_range(int n) {
    vector<int> phi(n + 1), primes;
    vector<bool> vis(n + 1, false);
    phi[1] = 1;
    for (int i = 2; i <= n; i++) {
        if (!vis[i]) {
            primes.push_back(i);
            phi[i] = i - 1;                // 素数 p: φ(p) = p-1
        }
        for (int p : primes) {
            if ((ll)i * p > n) break;
            vis[i * p] = true;
            if (i % p == 0) {
                phi[i * p] = phi[i] * p;   // p | i 情况
                break;
            }
            phi[i * p] = phi[i] * (p - 1); // p ∤ i 情况
        }
    }
    return phi;
}
```

### 4.3 扩展欧拉定理降幂

```cpp
// 读入指数 b（字符串），同时计算 b_mod 和判断是否 >= phi_m
// 返回 a^b mod m
ll qpow_ext(ll a, const string &b_str, ll m) {
    ll phi_m = phi(m);
    ll b_mod = 0;
    bool big = false;
    for (char c : b_str) {
        b_mod = b_mod * 10 + (c - '0');
        if (b_mod >= phi_m) big = true;
        b_mod %= phi_m;
    }
    if (big) b_mod += phi_m;  // 扩展欧拉定理：多加一个 φ(m)
    return qpow(a, b_mod, m);
}
```

使用示例：

```cpp
cout << phi(12) << endl;                        // 4
auto phi_list = phi_range(20);
cout << phi_list[12] << endl;                   // 4

// 计算 2^1000 mod 7（2 和 7 互质，φ(7)=6，1000 mod 6 = 4）
cout << qpow(2, 1000 % 6, 7) << endl;           // 2^4 mod 7 = 2

// 扩展：计算 2^{10^100} mod 4（大指数降幂）
cout << qpow_ext(2, "1000000000000...", 4);     // 自动降幂
```

## 五、复杂度与正确性分析

| 操作 | 时间复杂度 | 空间 |
|------|-----------|------|
| 单点 $\varphi(n)$ | $O(\sqrt{n})$ | $O(1)$ |
| 线性筛 $\varphi(1..n)$ | $O(n)$ | $O(n)$ |
| 扩展欧拉定理降幂 | $O(\log m + \text{len}(b))$ | $O(1)$ |

欧拉函数线性筛的正确性取决于 $\varphi$ 在 $p \mid i$ 和 $p \nmid i$ 两种情形下的递推公式，这两个公式均在 3.3 节做了推导。扩展欧拉定理的严格证明超出本篇范围（涉及群论），但其三段式形式在竞赛中是公认的标准工具，可以直接使用。

## 六、典型应用场景

1. **模逆元的通用求法**：当模数不是素数时，$a^{-1} \equiv a^{\varphi(m)-1} \pmod{m}$（需 $\gcd=1$）。这是费马法的欧拉推广。

2. **大指数降幂**：指数以字符串输入的超大幂次问题（CF 17D Notepad、数论题中常见的 $a^{b^{c}}$ 型），必须用扩展欧拉定理将指数压入可计算范围。

3. **原根相关的预备知识**：第 15 篇阶与原根中，$\varphi(\varphi(p))$ 是模 $p$ 下原根的数量，$\varphi$ 的积性是原根存在性判定的基础。

**推荐练习题**：

- [洛谷 P2158](https://www.luogu.com.cn/problem/P2158) — 仪仗队（SDOI2008）。求 $\sum_{i=1}^{n-1} \varphi(i)$，线性筛模板的直接应用
- [Codeforces 17D](https://codeforces.com/problemset/problem/17/D) — Notepad。$(b-1) \cdot b^{n-1} \bmod c$，指数极大，需要扩展欧拉定理降幂
- [洛谷 P5091](https://www.luogu.com.cn/problem/P5091) — 【模板】扩展欧拉定理。指数以字符串输入，三元判断式直接套用

## 七、小结

$\varphi(n)$ 的核心是乘积公式 $\varphi(n) = n \prod (1 - 1/p)$。由此派生出单点计算、线性筛法、欧拉定理 $a^{\varphi(m)} \equiv 1$ 和扩展降幂三段式四条分支。在竞赛中，线性筛 $\varphi$ 是积性函数筛法系列（$\mu, d, \sigma$）的第一个例子，其「分支递推」的模板结构会在第 22 篇被反复复用。
