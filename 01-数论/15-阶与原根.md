# 阶与原根

> Made by Aki.
> 最后更新于 2026.04.30

第 11 篇费马小定理告诉我们 $a^{p-1} \equiv 1 \pmod{p}$，但 $p-1$ 不一定是使这个同余式成立的最小指数。阶就是来回答「最小是几」的问题；原根则是那些「恰好要 $p-1$ 步才回到 $1$」的特殊元素。

---

## 一、引言

考虑模 $7$ 下 $2$ 的幂：

$$
2^1 \equiv 2,\; 2^2 \equiv 4,\; 2^3 \equiv 8 \equiv 1 \pmod{7}
$$

$2^3 \equiv 1$，只用了 $3$ 步就回到了 $1$，远小于 $\varphi(7) = 6$。但 $3$ 的表现不同：

$$
3^1 \equiv 3,\; 3^2 \equiv 2,\; 3^3 \equiv 6,\; 3^4 \equiv 4,\; 3^5 \equiv 5,\; 3^6 \equiv 1 \pmod{7}
$$

$3$ 用了整整 $6$ 步才回到 $1$——$3$ 就是模 $7$ 的一个**原根**。原根的重要性在于：它的幂遍历了整个模 $7$ 的乘法群 $\{1,2,3,4,5,6\}$。这使得我们可以用离散指数（对数）来表示乘法群中的任一元——这是 BSGS 和 NTT 的基础。

## 三、核心推导

### 3.1 阶的定义与性质

> 若 $\gcd(a, m) = 1$，使 $a^d \equiv 1 \pmod{m}$ 成立的最小正整数 $d$ 称为 $a$ 模 $m$ 的**阶**，记作 $\operatorname{ord}_m(a)$。

**核心性质**：$\operatorname{ord}_m(a) \mid \varphi(m)$。

**证明**：由欧拉定理 $a^{\varphi(m)} \equiv 1$。设 $d = \operatorname{ord}_m(a)$，用带余除法 $\varphi(m) = dq + r$（$0 \le r < d$）。则：

$$
1 \equiv a^{\varphi(m)} \equiv a^{dq+r} \equiv (a^d)^q \times a^r \equiv 1^q \times a^r \equiv a^r \pmod{m}
$$

所以 $a^r \equiv 1$。由 $d$ 是最小正指数，只能 $r = 0$。因此 $d \mid \varphi(m)$。

**推论——快速判定阶**：要验证 $d$ 是否是阶，只需检验 $a^d \equiv 1$ 且对 $d$ 的每个素因子 $p$，$a^{d/p} \not\equiv 1 \pmod{m}$。因为任何 $d$ 的真因子必然至少缺少一个 $d$ 的素因子。

---

### 3.2 原根

> 若 $\operatorname{ord}_m(g) = \varphi(m)$，则 $g$ 称为模 $m$ 的**原根**。

当 $g$ 是原根时，集合 $\{g^0, g^1, g^2, \dots, g^{\varphi(m)-1}\}$ 在模 $m$ 下恰好给出了所有与 $m$ 互质的剩余类。这意味着乘法群 $\mathbb{Z}_m^*$ 是以 $g$ 为生成元的循环群。

**原根的存在条件**（高斯定理）：模 $m$ 存在原根当且仅当：

$$
m = 2,\; 4,\; p^k,\; 2p^k
$$

其中 $p$ 为奇素数，$k \ge 1$。其他模数（如 $m=8, 15, 2^k(k\ge3)$）没有原根。竞赛中最常见的场景是模奇素数 $p$——此时原根一定存在。

---

### 3.3 寻找原根

假设 $m$ 满足原根存在条件。算法：

1. 计算 $\varphi(m)$，并质因数分解 $\varphi(m)$。
2. 从 $g = 2$ 开始枚举。
3. 对 $\varphi(m)$ 的每个素因子 $q$，检查 $g^{\varphi(m)/q} \not\equiv 1 \pmod{m}$。若全部通过，$g$ 就是原根。

**为什么这足够了？** 对任意真因子 $d \mid \varphi(m)$ 且 $d < \varphi(m)$，$d$ 至少缺少 $\varphi(m)$ 的某个素因子 $q$，所以 $d \mid \varphi(m)/q$。因此如果对每个 $q$ 都有 $g^{\varphi(m)/q} \not\equiv 1$，那么对任意真因子 $d$ 也不可能有 $g^d \equiv 1$——否则 $g^{\varphi(m)/q} \equiv (g^d)^{\varphi(m)/(qd)} \equiv 1$，矛盾。

**复杂度**：枚举次数通常极少（最小的原根一般不超过 $m^{1/4}$ 量级，实践中 $m \le 10^9$ 时原根通常 $\le 100$）。实际上，模素数 $p$ 的最小原根平均非常小（绝大多数素数的最小原根在 $100$ 以内）。

## 四、代码实现

```cpp
using ll = long long;

// 求 a 模 m 的阶（前提：gcd(a, m) = 1）
ll order(ll a, ll m, ll phi_m, const vector<ll> &pfactors) {
    ll d = phi_m;
    for (ll q : pfactors) {           // φ(m) 的不同素因子
        while (d % q == 0 && qpow(a, d / q, m) == 1)
            d /= q;                   // 尽可能缩小指数
    }
    return d;                         // 最终 d 就是阶
}

// 求模 m 的最小原根（前提：m 是 2,4,p^k,2p^k 之一）
ll primitive_root(ll m) {
    ll phi_m = phi(m);
    auto f = factorize(phi_m);        // 第 6 篇的质因数分解
    vector<ll> pfactors;
    for (auto &[p, e] : f) pfactors.push_back(p);

    for (ll g = 2; g < m; g++) {
        if (gcd(g, m) != 1) continue;
        bool ok = true;
        for (ll q : pfactors)
            if (qpow(g, phi_m / q, m) == 1) { ok = false; break; }
        if (ok) return g;
    }
    return -1;                         // 无原根（理论上不会走到这里）
}

// 求所有原根（数量为 φ(φ(m))）
vector<ll> all_primitive_roots(ll m) {
    ll g = primitive_root(m);
    if (g == -1) return {};
    ll phi_m = phi(m);
    vector<ll> roots;
    for (ll k = 1; k <= phi_m; k++) {
        if (gcd(k, phi_m) == 1)
            roots.push_back(qpow(g, k, m));
    }
    sort(roots.begin(), roots.end());
    return roots;
}
```

使用示例：

```cpp
ll g = primitive_root(998244353);  // NTT 常用素数
// g = 3（模 998244353 的最小原根是 3）

auto roots = all_primitive_roots(7);
// roots = {3, 5}（φ(7)=6，φ(φ(7))=2，恰好两个原根）

ll ord = order(2, 7, 6, {2, 3});
// ord = 3（2^3 ≡ 1 mod 7，且 2^1, 2^2 都不 ≡ 1）
```

## 五、复杂度与正确性分析

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 单点阶 | $O(\log^2 m)$ | 外层枚举 $\varphi(m)$ 的素因子，内层快速幂 |
| 找最小原根 | $O(g_0 \cdot \omega \cdot \log m)$ | $g_0$ 是最小原根（通常很小），$\omega$ 是 $\varphi(m)$ 的素因子个数 |
| 求所有原根 | $O(\varphi(m) \log m)$ | 需要枚举所有 $k \perp \varphi(m)$ |

阶的正确性来自「$d$ 是阶」的等价条件：$a^d \equiv 1$ 且对任意 $d$ 的素因子 $q$，$a^{d/q} \not\equiv 1$。原根的存在条件由高斯定理保证，找原根的算法基于上述等价条件。

## 六、典型应用场景

1. **BSGS 离散对数**（第 16 篇）：在模素数下，任意非零元可以写成 $g^e$ 的形式，离散对数问题就是在给定 $g^e$ 时求 $e$。

2. **NTT（数论变换）**（第 24 篇）：NTT 需要的单位根由原根的幂生成。素数 $p = k \cdot 2^m + 1$ 的原根提供了 $2^m$ 阶单位根。

3. **指数循环分析**：$\operatorname{ord}_m(a)$ 就是 $a$ 的幂在模 $m$ 下的循环长度，这在周期性问题中无处不在。

**推荐练习题**：

- [洛谷 P6091](https://www.luogu.com.cn/problem/P6091) — 【模板】原根。判断一个数是否有原根，有则输出最小原根
- [洛谷 P5491](https://www.luogu.com.cn/problem/P5491) — 【模板】二次剩余。Cipolla 算法依赖原根寻找二次非剩余

## 七、小结

阶回答的是「$a$ 的幂要乘几次才回到 $1$」，原根回答的是「谁的幂能遍历整个乘法群」。两者共同构建了模乘法群的循环结构理论——这是离散对数、NTT、二次剩余等更高层应用的根基。

下一篇是**离散对数与 BSGS**——给定 $a$ 和 $b$，求 $x$ 使得 $a^x \equiv b \pmod{p}$。这是阶和原根理论的直接应用。
