# 积性函数与线性筛

> Made by Aki.
> 最后更新于 2026.04.30

欧拉线性筛不仅能筛素数——它的框架可以同时在线性时间内计算 $\varphi$、$\mu$、$d$（约数个数）、$\sigma$（约数和）等一系列积性函数。可以说，一个欧拉筛 = 一个素数筛 + $k$ 个积性函数的批量生成器。

---

## 一、引言

第 10 篇我们在线性筛中顺带计算了 $\varphi$，第 21 篇计算了 $\mu$。这两个函数共享一个关键属性——积性。本篇将积性函数与线性筛的关系系统化：给定一个积性函数 $f$，只要我们知道 $f(p^e)$（素数幂处的值），就可以在线性筛中 $O(n)$ 生成 $f(1)$ 到 $f(n)$。

## 三、核心推导

### 3.1 积性函数定义

> $f$ 是**积性函数**当且仅当 $\gcd(a, b) = 1$ 时 $f(ab) = f(a)f(b)$。

如果对任意 $a, b$（不要求互质）都满足 $f(ab) = f(a)f(b)$，则 $f$ 是**完全积性函数**。

常见积性函数：

| 函数 | 符号 | 定义 | 完全积性？ |
|------|------|------|:---:|
| 欧拉函数 | $\varphi(n)$ | 与 $n$ 互质的数的个数 | ❌ |
| 莫比乌斯函数 | $\mu(n)$ | 无平方因子则为 $(-1)^k$ | ❌ |
| 约数个数 | $d(n)$ | $\sum_{d \mid n} 1$ | ❌ |
| 约数和 | $\sigma(n)$ | $\sum_{d \mid n} d$ | ❌ |
| 恒等函数 | $I(n) = n$ | — | ✅ |

---

### 3.2 积性函数在线性筛中的递推

回顾欧拉线性筛的结构：

```cpp
for (int i = 2; i <= n; i++) {
    if (!vis[i]) primes.push_back(i);  // i 是素数
    for (int p : primes) {
        if (i * p > n) break;
        vis[i * p] = true;
        if (i % p == 0) break;          // ★
    }
}
```

在筛的过程中，每当我们标记 $v = i \times p$ 为合数时，有两种情况：

**情况 A**：$p \nmid i$（$p$ 和 $i$ 互质）

此时 $f(i \times p) = f(i) \times f(p)$（积性定义直接应用）。

**情况 B**：$p \mid i$（$p$ 是 $i$ 的因子）

此时 $i$ 和 $p$ 不互质，不能用积性。但我们可以利用 $p$ 是 $i$ 的**最小质因子**这一事实。设 $i = p^e \times k$，其中 $p \nmid k$。则：

对于每种积性函数，需要推导 $f(p^{e+1}k)$ 与已知值（如 $f(p^e k)$）的递推关系。这就是为什么需要知道 $f(p^e)$——素数幂处的值。

---

### 3.3 各函数的递推公式

**$\varphi(n)$**（第 10 篇已述）：
- $p \nmid i$：$\varphi(i \cdot p) = \varphi(i) \cdot (p-1)$
- $p \mid i$：$\varphi(i \cdot p) = \varphi(i) \cdot p$

**$\mu(n)$**（第 21 篇已述）：
- $p \nmid i$：$\mu(i \cdot p) = -\mu(i)$
- $p \mid i$：$\mu(i \cdot p) = 0$

**$d(n)$（约数个数）**：
维护辅助数组 $e[i]$ 表示 $i$ 的最小质因子的指数。则：
- $p \nmid i$：$d(i \cdot p) = d(i) \times 2$，$e(i \cdot p) = 1$
- $p \mid i$：$d(i \cdot p) = d(i) / (e(i)+1) \times (e(i)+2)$，$e(i \cdot p) = e(i) + 1$

**$\sigma(n)$（约数和）**：
维护辅助数组 $g[i]$ 表示 $i$ 的最小质因子的等比级数和 $1+p+\dots+p^e$。
- $p \nmid i$：$\sigma(i \cdot p) = \sigma(i) \times (1+p)$，$g(i \cdot p) = 1+p$
- $p \mid i$：$\sigma(i \cdot p) = \sigma(i) / g(i) \times (g(i) \cdot p + 1)$，$g(i \cdot p) = g(i) \cdot p + 1$

## 四、代码实现

```cpp
// 一次性筛出 φ, μ, d, σ
struct MultiFunc {
    vector<int> primes;
    vector<bool> vis;
    vector<ll> phi, mu, d, sigma;
    vector<int> e_cnt;   // d 的辅助：最小质因子指数
    vector<ll> g_sum;    // σ 的辅助：最小质因子的等比级数和

    MultiFunc(int n) : vis(n+1), phi(n+1), mu(n+1), d(n+1),
                       sigma(n+1), e_cnt(n+1), g_sum(n+1) {
        phi[1] = mu[1] = d[1] = sigma[1] = 1;
        for (int i = 2; i <= n; i++) {
            if (!vis[i]) {
                primes.push_back(i);
                phi[i] = i - 1;  mu[i] = -1;
                d[i] = 2;        sigma[i] = i + 1;
                e_cnt[i] = 1;    g_sum[i] = i + 1;
            }
            for (int p : primes) {
                if ((ll)i * p > n) break;
                int v = i * p;
                vis[v] = true;
                if (i % p == 0) {  // p | i
                    phi[v] = phi[i] * p;
                    mu[v] = 0;
                    e_cnt[v] = e_cnt[i] + 1;
                    d[v] = d[i] / e_cnt[v] * (e_cnt[v] + 1);
                    g_sum[v] = g_sum[i] * p + 1;
                    sigma[v] = sigma[i] / g_sum[i] * g_sum[v];
                    break;
                }
                // p ∤ i
                phi[v] = phi[i] * (p - 1);
                mu[v] = -mu[i];
                e_cnt[v] = 1;
                d[v] = d[i] * 2;
                g_sum[v] = p + 1;
                sigma[v] = sigma[i] * (p + 1);
            }
        }
    }
};
```
