# 二次剩余

> Made by Aki.
> 最后更新于 2026.04.30

方程 $x^2 \equiv n \pmod{p}$ 在模素数下什么时候有解？有解时怎么求？二次剩余理论给出了完整的答案——从 Legendre 符号的判定到 Cipolla 算法的构造。

---

## 一、引言

「$2$ 是不是模 $7$ 下的完全平方数？」等价于问：是否存在 $x$ 使得 $x^2 \equiv 2 \pmod{7}$？

检验：$1^2=1$, $2^2=4$, $3^2=9\equiv2$ ✓。所以 $2$ 是模 $7$ 的二次剩余，$x=3$ 是它的一个平方根。

换一个：$3$ 是不是模 $7$ 的二次剩余？$1^2=1$, $2^2=4$, $3^2=2$, $4^2=2$, $5^2=4$, $6^2=1$。$3$ 从未出现——$3$ 是二次非剩余。恰好一半的非零元素是二次剩余，一半不是。

## 三、核心推导

### 3.1 Legendre 符号与 Euler 判别法

> **Legendre 符号** $\displaystyle\left(\frac{n}{p}\right)$ 定义为：
>
> $$
> \left(\frac{n}{p}\right) = \begin{cases}
> \;0 & n \equiv 0 \pmod{p} \\
> \;1 & n \text{ 是模 } p \text{ 的二次剩余} \\
> -1 & n \text{ 是模 } p \text{ 的二次非剩余}
> \end{cases}
> $$

**Euler 判别法**给出了 Legendre 符号的计算公式：

$$
\boxed{\left(\frac{n}{p}\right) \equiv n^{\frac{p-1}{2}} \pmod{p}}
$$

**证明**：由费马小定理，$n^{p-1} \equiv 1 \pmod{p}$。因此 $(n^{(p-1)/2})^2 \equiv 1$，从而 $n^{(p-1)/2} \equiv \pm 1 \pmod{p}$。

若 $n$ 是二次剩余，设 $n \equiv x^2$，则 $n^{(p-1)/2} \equiv x^{p-1} \equiv 1 \pmod{p}$（费马小定理）。

若 $n$ 是二次非剩余：模素数下恰有一半的元素不是二次剩余，而方程 $y^{(p-1)/2} \equiv 1 \pmod{p}$ 在模 $p$ 的乘法群中最多有 $(p-1)/2$ 个解（因为乘法群是循环群，该方程的解就是指数为偶数的那些元素——恰好一半）。二次剩余已经占了这 $(p-1)/2$ 个位置，所以所有二次非剩余的 $n^{(p-1)/2} \equiv -1$。

这同时说明了：**模奇素数 $p$ 下恰好有 $(p-1)/2$ 个二次剩余和同样多的二次非剩余**。

---

### 3.2 Cipolla 算法——构造平方根

Euler 判别法告诉你**有没有**解，Cipolla 告诉你**解是什么**。

算法的核心思想是**域扩张**——在模 $p$ 的世界里引入一个「虚数单位」。

**步骤**：

1. 找一个 $a$ 使得 $a^2 - n$ 是模 $p$ 的二次非剩余（随机尝试，期望 $2$ 次成功）。
2. 设 $\omega = \sqrt{a^2 - n}$（形式上定义一个满足 $\omega^2 \equiv a^2 - n$ 的元素）。在扩域 $\mathbb{F}_p[\omega]$ 中计算 $(a + \omega)^{(p+1)/2}$。
3. 这个结果的「实部」就是 $n$ 的一个平方根。

**为什么成立？** 在扩域中，利用 $(a+\omega)^p \equiv a^p + \omega^p \equiv a - \omega \pmod{p}$（因为 $\omega^p = \omega \cdot (\omega^2)^{(p-1)/2} = \omega \cdot (a^2-n)^{(p-1)/2} \equiv \omega \cdot (-1) = -\omega$，因为 $a^2-n$ 是二次非剩余）。于是 $(a+\omega)^{p+1} \equiv (a-\omega)(a+\omega) = a^2 - \omega^2 = n$。所以 $(a+\omega)^{(p+1)/2}$ 的平方 $= n$——正是所求的平方根。

## 四、代码实现

```cpp
using ll = long long;
ll p;  // 模数

// 扩域元素：x + y·ω，其中 ω² = a² - n
struct Field {
    ll x, y;
    Field(ll x = 0, ll y = 0) : x(x % p), y(y % p) {}
};

ll w2;  // ω² = a² - n mod p（即 (a² - n) % p）

Field mul(const Field &u, const Field &v) {
    return Field(u.x * v.x + u.y * v.y % p * w2,
                 u.x * v.y + u.y * v.x);
}

Field qpow_field(Field a, ll b) {
    Field res(1, 0);
    while (b) {
        if (b & 1) res = mul(res, a);
        a = mul(a, a);
        b >>= 1;
    }
    return res;
}

// Cipolla：返回 n 的一个平方根（若存在），否则返回 -1
ll cipolla(ll n) {
    n %= p;
    if (n == 0) return 0;
    if (qpow(n, (p - 1) / 2, p) != 1) return -1;  // Euler 判别：无解

    ll a;
    do { a = rand() % (p - 1) + 1; }              // 1..p-1
    while (qpow((a * a - n % p + p) % p, (p - 1) / 2, p) == 1);

    w2 = (a * a - n % p + p) % p;                  // ω²
    Field ans = qpow_field(Field(a, 1), (p + 1) / 2);
    return ans.x;                                   // 实部即平方根
}
```

使用示例：

```cpp
p = 1000000007;
ll x = cipolla(2);   // 模 1e9+7 下 2 的平方根
// x = 59713600（另一个根是 p - x）
// 验证: x*x % p == 2 ✓
```

## 五、复杂度与正确性分析

Euler 判别法：一次快速幂 $O(\log p)$ 即可判定。Cipolla 算法：期望 $O(1)$ 次随机找到 $a$（因为恰好一半的 $a$ 满足 $a^2-n$ 是二次非剩余），加一次扩域快速幂 $O(\log p)$。总期望 $O(\log p)$。

## 六、典型应用场景

1. **判定二次剩余**：Euler 判别法 $n^{(p-1)/2} \equiv \pm 1$ 直接给出 Legendre 符号的值，常用于构造和判定问题。
2. **求模平方根**：Cipolla 算法是模素数下求平方根的标准方法，在密码学和数论构造题中频繁出现。
3. **二次互反律的前置知识**：Legendre 符号是二次互反律（Gauss）的基础——不过二次互反律在竞赛中出现较少，如果需要可查阅专题资料。
