# 连分数与佩尔方程

> Made by Aki.
> 最后更新于 2026.04.30

$\sqrt{2}$ 写成小数是 $1.4142\dots$，永远写不完。但写成连分数只需一行：$[1; 2, 2, 2, \dots]$。这种表示不仅能精确刻画无理数，还能导出佩尔方程 $x^2 - Dy^2 = 1$ 的完整整数解。

---

## 一、引言

佩尔方程 $x^2 - Dy^2 = 1$（$D$ 不是完全平方数）是数论中最古老的不定方程之一。它的解有无穷多组，且全部可以通过 $\sqrt{D}$ 的连分数展开来生成。这个联系——连分数给出最佳有理逼近，而最佳逼近恰好对应佩尔方程的解——是初等数论中最优美的结果之一。

## 三、核心推导

### 3.1 连分数的表示

一个实数 $\alpha$ 的连分数展开为：

$$
\alpha = a_0 + \cfrac{1}{a_1 + \cfrac{1}{a_2 + \cfrac{1}{\ddots}}}
$$

记作 $[a_0; a_1, a_2, \dots]$。

对于有理数，连分数有限（这是判断有理性的标准）。对于二次无理数（形如 $\frac{P+\sqrt{D}}{Q}$），连分数**循环**，且循环节的结构与佩尔方程的最小解有确定关系。

---

### 3.2 渐进分数——最佳有理逼近

截断连分数到第 $k$ 项得到的分数 $p_k/q_k$ 称为第 $k$ 个**渐进分数**。它们满足递推：

$$
\begin{cases}
p_k = a_k \cdot p_{k-1} + p_{k-2} \\
q_k = a_k \cdot q_{k-1} + q_{k-2}
\end{cases}
$$

其中初始值 $p_{-2}=0, p_{-1}=1, q_{-2}=1, q_{-1}=0$。

渐进分数有一个关键性质：$\displaystyle\left|\alpha - \frac{p_k}{q_k}\right| < \frac{1}{q_k^2}$。即在所有分母不超过 $q_k$ 的分数中，$p_k/q_k$ 是对 $\alpha$ 的最佳逼近。

---

### 3.3 佩尔方程与连分数的关系

> 设 $\sqrt{D}$ 的连分数循环节长度为 $L$。则：
> - 若 $L$ 为偶数：$x^2 - Dy^2 = 1$ 的最小解 $(x_1, y_1) = (p_{L-1}, q_{L-1})$
> - 若 $L$ 为奇数：$x^2 - Dy^2 = 1$ 的最小解 $(x_1, y_1) = (p_{2L-1}, q_{2L-1})$

且所有解由 $(x_1 + y_1\sqrt{D})^n$ 展开生成：

$$
x_n + y_n\sqrt{D} = (x_1 + y_1\sqrt{D})^n
$$

这给出了解的递推：$x_{n+1} = x_1 x_n + D y_1 y_n$，$y_{n+1} = x_1 y_n + y_1 x_n$。

## 四、代码实现

```cpp
// 求佩尔方程 x² - D·y² = 1 的最小正整数解
pair<ll, ll> pell(ll D) {
    ll a0 = sqrt(D);
    if (a0 * a0 == D) return {-1, -1};  // D 是完全平方数，无解

    ll m = 0, d = 1, a = a0;
    ll p_minus2 = 0, p_minus1 = 1;
    ll q_minus2 = 1, q_minus1 = 0;
    ll p = a0, q = 1;

    if (p * p - D * q * q == 1) return {p, q};  // 第 0 项恰好是解

    while (true) {
        m = d * a - m;
        d = (D - m * m) / d;
        a = (a0 + m) / d;

        ll p_next = a * p + p_minus1;
        ll q_next = a * q + q_minus1;
        p_minus1 = p; q_minus1 = q;
        p = p_next; q = q_next;

        if (p * p - D * q * q == 1) return {p, q};
    }
}
```

```cpp
auto [x1, y1] = pell(2);   // x² - 2y² = 1 的最小解
// x1=3, y1=2  → 3² - 2·2² = 9-8 = 1 ✓

auto sol = pell(61);       // x² - 61y² = 1 的最小解
// 这是历史上著名的「牛群问题」的退化版，最小解非常大
// x=1766319049, y=226153980
```

## 五、典型应用场景

1. **求解佩尔方程**：$x^2 - Dy^2 = 1$ 的最小解由 $\sqrt{D}$ 的连分数展开直接给出，所有解由 $(x_1 + y_1\sqrt{D})^n$ 生成。
2. **最佳有理逼近**：连分数的渐进分数 $p_k/q_k$ 是对 $\sqrt{D}$ 的最佳有理逼近，误差 $< 1/q_k^2$。这在数值计算和几何问题中有应用。
3. **循环连分数的结构分析**：二次无理数的连分数展开一定是循环的，循环节长度与类数等深层数论量相关。
