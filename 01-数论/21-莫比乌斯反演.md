# 莫比乌斯反演

> Made by Aki.
> 最后更新于 2026.04.30

莫比乌斯反演是竞赛数论中最强大的工具之一。它将「求和于约数」与「求和于倍数」互相转化，在 gcd 计数、互质统计、整除条件求和等问题中出场率极高。

---

## 一、引言

一个常见问题：给定 $n$，求 $\sum_{i=1}^{n} \sum_{j=1}^{n} [\gcd(i, j) = 1]$。

暴力双重循环 $O(n^2)$ 不可行。但如果能把条件 $\gcd(i,j)=1$ 转化为某种可批量求和的形式，就能优化。莫比乌斯函数 $\mu(n)$ 提供了这个转化：

$$
[\gcd(i, j) = 1] = \sum_{d \mid \gcd(i, j)} \mu(d)
$$

## 三、核心推导

### 3.1 莫比乌斯函数 $\mu(n)$

$$
\mu(n) = \begin{cases}
1 & n = 1 \\
0 & n \text{ 含有平方因子（即存在素数 } p \text{ 使得 } p^2 \mid n \text{）} \\
(-1)^k & n = p_1 p_2 \cdots p_k\;（k \text{ 个不同素数的乘积）}
\end{cases}
$$

例如：$\mu(1)=1$, $\mu(6)=(-1)^2=1$, $\mu(30)=(-1)^3=-1$, $\mu(12)=0$（有平方因子 $4$）。

**核心性质**（反演的基础）：

$$
\sum_{d \mid n} \mu(d) = [n = 1] = \begin{cases} 1 & n = 1 \\ 0 & n > 1 \end{cases}
$$

这个性质使 $\mu$ 成为「检测 $n$ 是否为 $1$」的指示器。

**证明**：设 $n = p_1^{e_1} \cdots p_k^{e_k}$。$\sum_{d \mid n} \mu(d)$ 的贡献仅来自那些不含平方因子的因子，即 $\prod p_i$ 的任意子集。子集大小为偶数的贡献 $+1$，奇数的贡献 $-1$：

$$
\sum_{d \mid n} \mu(d) = \sum_{i=0}^{k} \binom{k}{i} (-1)^i = (1-1)^k = \begin{cases} 1 & k=0\;(n=1) \\ 0 & k>0 \end{cases}
$$

### 3.2 莫比乌斯反演公式

> $$
> g(n) = \sum_{d \mid n} f(d) \;\iff\; f(n) = \sum_{d \mid n} \mu\left(\frac{n}{d}\right) g(d)
> $$

等价地：

$$
g(n) = \sum_{d \mid n} f(d) \;\iff\; f(n) = \sum_{d \mid n} \mu(d) \cdot g\left(\frac{n}{d}\right)
$$

**推导**（代入验证）：将 $g(d) = \sum_{k \mid d} f(k)$ 代入右边：

$$
\sum_{d \mid n} \mu(n/d) \sum_{k \mid d} f(k) = \sum_{k \mid n} f(k) \sum_{\substack{d: k \mid d \mid n}} \mu(n/d)
$$

令 $m = n/k$，$d' = n/d$，内层和变为 $\sum_{d' \mid m} \mu(d') = [m=1] = [k=n]$。仅 $k=n$ 时有贡献 $f(n)$。

### 3.3 倍数形式

还有第二个常用形式（求和于倍数而非约数）：

$$
g(n) = \sum_{n \mid d} f(d) \;\iff\; f(n) = \sum_{n \mid d} \mu\left(\frac{d}{n}\right) g(d)
$$

在竞赛中，这经常用来做「整除分块」（第 27 篇）优化。

### 3.4 典型应用：$\gcd$ 求和

回到引言问题 $\sum_{i,j} [\gcd(i,j)=1]$：

$$
\begin{aligned}
\sum_{i=1}^{n} \sum_{j=1}^{n} [\gcd(i,j)=1]
&= \sum_{i,j} \sum_{d \mid \gcd(i,j)} \mu(d) \\
&= \sum_{d=1}^{n} \mu(d) \sum_{\substack{1 \le i,j \le n \\ d \mid i, d \mid j}} 1 \\
&= \sum_{d=1}^{n} \mu(d) \cdot \left\lfloor \frac{n}{d} \right\rfloor^2
\end{aligned}
$$

原来 $O(n^2)$ 的枚举变成 $O(n)$ 的求和。结合整除分块可以进一步优化到 $O(\sqrt{n})$——但这需要第 27 篇的技术。

## 四、Dirichlet 卷积视角

莫比乌斯反演可以更优雅地表述为 Dirichlet 卷积。定义两个数论函数的卷积：

$$
(f * g)(n) = \sum_{d \mid n} f(d) \cdot g\left(\frac{n}{d}\right)
$$

单位函数 $\epsilon(n) = [n=1]$ 满足 $f * \epsilon = f$。常数函数 $1(n) = 1$，恒等函数 $I(n) = n$。

则莫比乌斯函数是常数函数 $1$ 的卷积逆：$\mu * 1 = \epsilon$（即 $\sum_{d|n} \mu(d) = [n=1]$）。

反演公式 $g = f * 1 \iff f = g * \mu$ 就是两边卷积 $\mu$ 的结果。

## 五、代码实现

```cpp
// 线性筛预处理 μ(1..n)
vector<int> mobius_range(int n) {
    vector<int> mu(n + 1), primes;
    vector<bool> vis(n + 1, false);
    mu[1] = 1;
    for (int i = 2; i <= n; i++) {
        if (!vis[i]) {
            primes.push_back(i);
            mu[i] = -1;                      // 单个素数：μ(p) = -1
        }
        for (int p : primes) {
            if ((ll)i * p > n) break;
            vis[i * p] = true;
            if (i % p == 0) {
                mu[i * p] = 0;               // 有平方因子：μ = 0
                break;
            }
            mu[i * p] = -mu[i];              // 增加一个不同素因子，(-1)^k → (-1)^{k+1}
        }
    }
    return mu;
}
```

```cpp
auto mu = mobius_range(100);
// mu[1]=1, mu[2]=-1, mu[3]=-1, mu[4]=0, mu[5]=-1,
// mu[6]=1, mu[7]=-1, mu[8]=0, ...
```
