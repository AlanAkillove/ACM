# 快速幂与慢速乘

> Made by Aki.
> 最后更新于 2026.04.29

计算 $a^b \bmod m$——这个操作在竞赛中出现的频率可能比任何其他数学操作都高。本篇解决两个问题：如何算得快（快速幂），以及如何算得稳（慢速乘防止乘法溢出）。

---

## 一、引言

第 4 篇的 Miller-Rabin 算法里，我们悄悄用了一个叫 `qpow` 的函数，它用不到 10 行代码就在 $O(\log b)$ 时间内算出了 $a^b \bmod m$。你可能会问：正常算 $a^b$ 不就是一个循环乘 $b$ 次吗？怎么做到 $O(\log b)$ 的？

```
正常做法：a * a * a * ... * a    ← 乘 b 次，需要 O(b)
快速幂：  a * a^2 * a^4 * ...    ← 巧用指数的二进制分解，需要 O(log b)
```

当 $b$ 大到 $10^{18}$ 时，一个做 $10^{18}$ 次循环，一个做 $60$ 次——差距是天文学级别的。

## 三、核心推导

### 3.1 快速幂——把指数写成二进制

**核心观察**：$a^8$ 不需要乘 $8$ 次。把 $a$ 平方三次就够了：

$$
a \xrightarrow{\text{平方}} a^2 \xrightarrow{\text{平方}} a^4 \xrightarrow{\text{平方}} a^8
$$

三次乘法得到 $a^8$。那 $a^{13}$ 呢？把指数 $13$ 写成二进制：

$$
13 = 1101_2 = 1 \cdot 2^3 + 1 \cdot 2^2 + 0 \cdot 2^1 + 1 \cdot 2^0 = 8 + 4 + 0 + 1
$$

于是：

$$
a^{13} = a^8 \times a^4 \times a^0 \times a^1 = a^8 \times a^4 \times a^1
$$

其中 $a^8, a^4, a^2, a^1$ 可以通过连续平方逐个得到。二进制位为 $1$ 的项贡献到答案，为 $0$ 的项跳过。

**算法流程**：

```
初始化 ans = 1
当 b > 0：
    如果 b 的最低位是 1：ans = ans * a  mod m   （当前位贡献）
    a = a * a  mod m                              （准备下一二进制位对应的 a 的幂）
    b 右移一位                                    （处理下一个二进制位）
```

❓ **为什么 `a` 要不断平方？** 因为在二进制表示中，第 $i$ 位对应的值是 $a^{2^i}$。从 $a^{2^i}$ 到 $a^{2^{i+1}}$ 只需一次平方：$(a^{2^i})^2 = a^{2^{i+1}}$。所以 `a = a * a` 是跟着 `b` 的右移同步推进的。

---

#### 3.1.1 手动模拟（$3^{13} \bmod 7$）

| 步骤 | $b$（二进制） | 当前 $a$ | $b \bmod 2$？ | ans |
|:----:|:------------:|:--------:|:-----------:|:---:|
| 初始 | $13 \; (1101)$ | $3$ | — | $1$ |
| 1 | $1101$ → 最低位是 $1$ | $3$ | ✅ | $1 \times 3 = 3$ |
| | $b$ 右移 → $6$，$a$ 平方 | $3^2=9 \equiv 2$ | | $3$ |
| 2 | $0110$ → 最低位是 $0$ | $2$ | ❌ | $3$（跳过） |
| | $b$ 右移 → $3$，$a$ 平方 | $2^2=4$ | | $3$ |
| 3 | $0011$ → 最低位是 $1$ | $4$ | ✅ | $3 \times 4 = 12 \equiv 5$ |
| | $b$ 右移 → $1$，$a$ 平方 | $4^2=16 \equiv 2$ | | $5$ |
| 4 | $0001$ → 最低位是 $1$ | $2$ | ✅ | $5 \times 2 = 10 \equiv 3$ |
| | $b$ 右移 → $0$，结束 | — | | $\mathbf{3}$ |

验证：$3^{13} = 1594323$，$1594323 \bmod 7 = 3$。✓

**关键观察**：$b = 13$ 的二进制长度是 $4$ 位，算法恰好循环 $4$ 次。对于任意 $b$，循环次数为 $\lfloor \log_2 b \rfloor + 1$，即 $O(\log b)$。

### 3.2 慢速乘——当乘法本身也会溢出

快速幂里有一行 `ans = ans * a % m`。如果 $m \approx 10^{18}$，`ans * a` 在取模之前可以达到 $(10^{18}-1)^2 \approx 10^{36}$，**远超 `long long` 的 $9.2 \times 10^{18}$**。

解决方案：**慢速乘（龟速乘）**——用加法模拟乘法。

$$
a \times b = \underbrace{a + a + \cdots + a}_{b \text{ 次}}
$$

与快速幂一模一样的二分思想，把 $b$ 写成二进制：

$$
a \times 13 = a \times 8 + a \times 4 + a \times 1
$$

```cpp
ll slow_mul(ll a, ll b, ll m) {
    ll ans = 0;
    while (b) {
        if (b & 1) ans = (ans + a) % m;
        a = (a + a) % m;       // 相当于 a *= 2
        b >>= 1;
    }
    return ans;
}
```

结构跟快速幂完全对称——只是把乘法换成了加法。时间复杂度 $O(\log b)$。

**什么时候需要慢速乘？** 当 $m > 10^9$ 且两个在 $[0, m)$ 内的数需要相乘时。在竞赛中，如果你用了 `__int128`（GCC 扩展，第 4 篇的 Miller-Rabin 代码中已出现过），可以直接写 `(__int128)ans * a % m`，比慢速乘快很多。慢速乘的适用场景是编译器不支持 `__int128` 或者需要严格标准 C++ 的环境。

---

### 3.3 快速幂的递归版——另一种理解方式

有时递归版本更直观：

$$
a^b = \begin{cases}
1 & b = 0 \\
(a^{b/2})^2 & b \text{ 为偶数} \\
a \times (a^{b/2})^2 & b \text{ 为奇数}
\end{cases}
$$

直接翻译成代码：

```cpp
ll qpow_rec(ll a, ll b, ll m) {
    if (b == 0) return 1;
    ll half = qpow_rec(a, b / 2, m);
    half = half * half % m;
    if (b % 2 == 1) half = half * a % m;
    return half;
}
```

递归版本的深度为 $O(\log b)$，不会爆栈。迭代版（3.1 节）在竞赛中更常用，因为常数更小、没有函数调用开销。

---

### 3.4 矩阵快速幂——递推式加速

如果递推式是**线性常系数**的，可以用矩阵快速幂在 $O(k^3 \log n)$ 时间内求出第 $n$ 项，其中 $k$ 是递推阶数。

以斐波那契数列为例：$F_0 = 0$，$F_1 = 1$，$F_n = F_{n-1} + F_{n-2}$。

将递推写成矩阵形式：

$$
\begin{bmatrix} F_n \\ F_{n+1} \end{bmatrix} =
\begin{bmatrix} 0 & 1 \\ 1 & 1 \end{bmatrix}
\begin{bmatrix} F_{n-1} \\ F_n \end{bmatrix}
$$

反复代入：

$$
\begin{bmatrix} F_n \\ F_{n+1} \end{bmatrix} =
\begin{bmatrix} 0 & 1 \\ 1 & 1 \end{bmatrix}^n
\begin{bmatrix} F_0 \\ F_1 \end{bmatrix} =
\begin{bmatrix} 0 & 1 \\ 1 & 1 \end{bmatrix}^n
\begin{bmatrix} 0 \\ 1 \end{bmatrix}
$$

求 $F_n$ 转化为了求一个 $2 \times 2$ 矩阵的 $n$ 次幂——直接用矩阵快速幂，把数的乘法替换为矩阵乘法即可。

**矩阵乘法的结合性**是矩阵快速幂成立的关键：$(A \times B) \times C = A \times (B \times C)$。这意味着我们同样可以用二进制的思路来分解矩阵的幂。注意矩阵乘法**不满足交换律**——但这不影响结合性。

## 四、代码实现

### 4.1 快速幂（迭代版，标准竞赛模板）

```cpp
using ll = long long;

ll qpow(ll a, ll b, ll m) {
    ll ans = 1 % m;          // 注意 m=1 的情况，此时任何数 mod 1 = 0
    while (b) {
        if (b & 1) ans = (__int128)ans * a % m;
        a = (__int128)a * a % m;
        b >>= 1;
    }
    return ans;
}
```

### 4.2 慢速乘（不需要 `__int128` 时使用）

```cpp
ll slow_mul(ll a, ll b, ll m) {
    ll ans = 0;
    while (b) {
        if (b & 1) ans = (ans + a) % m;
        a = (a + a) % m;
        b >>= 1;
    }
    return ans;
}
```

### 4.3 矩阵快速幂

```cpp
using Matrix = vector<vector<ll>>;

Matrix mul(const Matrix &A, const Matrix &B, ll mod) {
    int n = A.size();
    Matrix C(n, vector<ll>(n, 0));
    for (int i = 0; i < n; i++)
        for (int k = 0; k < n; k++)
            if (A[i][k])  // 稀疏优化：跳过零元
                for (int j = 0; j < n; j++)
                    C[i][j] = (C[i][j] + (__int128)A[i][k] * B[k][j]) % mod;
    return C;
}

Matrix qpow_mat(Matrix A, ll b, ll mod) {
    int n = A.size();
    Matrix ans(n, vector<ll>(n, 0));
    for (int i = 0; i < n; i++) ans[i][i] = 1;  // 单位矩阵
    while (b) {
        if (b & 1) ans = mul(ans, A, mod);
        A = mul(A, A, mod);
        b >>= 1;
    }
    return ans;
}
```

使用示例：

```cpp
// 求 2^100 mod 1000000007
cout << qpow(2, 100, 1000000007) << endl;  // 976371285

// 求 F_50 mod 1000000007（斐波那契第 50 项）
Matrix M = {{0, 1}, {1, 1}};
Matrix Mn = qpow_mat(M, 50, 1000000007);
ll F50 = Mn[0][1];  // F_50 = 12586269025 mod 1e9+7
```

## 五、复杂度与正确性分析

| 算法 | 时间复杂度 | 说明 |
|------|-----------|------|
| 快速幂 | $O(\log b)$ | 循环 $\lfloor \log_2 b \rfloor + 1$ 次 |
| 慢速乘 | $O(\log b)$ | 同快速幂结构，每次加法+取模 |
| 矩阵快速幂 | $O(k^3 \log b)$ | $k$ 为矩阵阶数，矩阵乘法 $O(k^3)$ |

**正确性**：快速幂的核心是恒等式 $a^b = a^{b_0 \cdot 2^0} \times a^{b_1 \cdot 2^1} \times \cdots$，其中 $b_i$ 是 $b$ 的二进制第 $i$ 位。这个等式成立的根基是 $a^{x+y} = a^x \times a^y$ 和 $(a^{2^i})^2 = a^{2^{i+1}}$。结合模乘法的「先取模再算」性质（第 1 篇），整个过程在模意义下保持正确。

## 六、典型应用场景

快速幂的用途极其广泛。以下是它在本系列中已经或将要出现的场景：

1. **素性测试**（第 4 篇 Miller-Rabin）：`qpow(a, d, n)` 计算 $a^d \bmod n$。
2. **乘法逆元**（第 9 篇）：当模数为素数时，$a^{-1} \equiv a^{m-2} \pmod{m}$，需要快速幂。
3. **矩阵加速递推**：任何线性递推式可以用矩阵快速幂从 $O(n)$ 加速到 $O(\log n)$。
4. **模意义下的高次幂**：任何需要计算大指数模幂的场合。

**推荐练习题**（仅涉及已学知识：模运算 + 素数 + 快速幂）：

- [洛谷 P1226](https://www.luogu.com.cn/problem/P1226) — 【模板】快速幂。$a^b \bmod p$ 的纯模板题，代码直接套
- [洛谷 P3390](https://www.luogu.com.cn/problem/P3390) — 【模板】矩阵快速幂。求 $n \times n$ 矩阵的 $k$ 次幂，代码框架与普通快速幂完全一致
- [洛谷 P1962](https://www.luogu.com.cn/problem/P1962) — 斐波那契数列。求 $F_n \bmod 10^9+7$，$n \le 2^{63}$。用矩阵快速幂将 $O(n)$ 递推压缩到 $O(\log n)$

## 七、小结

快速幂的核心思想只有一句话：**把指数拆成二进制，每一位要么贡献要么跳过**。这个思想可以迁移到任何满足**结合律**的运算上——数的乘法、矩阵乘法、甚至群论中的抽象运算。

反过来，慢速乘是同一个思想的降级应用——当乘法不安全时，退回用加法模拟乘法。二者的代码几乎对称：一个用 `*`，一个用 `+`；一个初始化 `ans=1`，一个初始化 `ans=0`。
