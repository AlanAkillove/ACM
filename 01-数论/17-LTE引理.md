# LTE 引理

> Made by Aki.
> 最后更新于 2026.04.30

当题目问「$3^{100} - 1$ 能被 $2$ 的几次幂整除」时，直接算 $3^{100}$ 是不现实的。LTE（Lifting The Exponent）引理正是为这类问题而生——它告诉你一个表达式中某个素因子的最高幂次，而无需真正计算出那个数。

---

## 一、引言

看一个问题：

> 求 $2^{100} + 1$ 和 $3^{100} - 1$ 的最大公约数。

直接算出 $2^{100}$ 和 $3^{100}$（天文数字）再去求 $\gcd$ 不可行。但如果知道 $v_p(a^n \pm b^n)$（即 $p$ 在 $a^n \pm b^n$ 中的最高幂次），就可以对每个可能的公共素因子逐一分析。

LTE 做的事情就是给出 $v_p(a^n \pm b^n)$ 的简洁公式。它在竞赛中出现的频率不算极高，但一旦出现，没有 LTE 几乎无从下手——属于典型的「知道了就简单，不知道就坐牢」的工具。

## 三、核心公式

LTE 引理分成两类情形：$p$ 整除差（$a-b$）和 $p$ 整除和（$a+b$）。

### 类型 1：$p \mid (a - b)$

设 $p$ 为奇素数，$p \mid (a-b)$ 且 $p \nmid ab$。则对任意正整数 $n$：

$$
\boxed{v_p(a^n - b^n) = v_p(a - b) + v_p(n)}
$$

**直观理解**：$a^n - b^n$ 比 $a-b$ 多出了 $v_p(n)$ 个因子 $p$。当 $n$ 本身是 $p$ 的倍数时，$a^n - b^n$ 会额外获得与 $n$ 中 $p$ 的幂次相同的增量。

**证明思路（归纳法）**：基础是 $n=p$ 的情形。将 $a^p - b^p$ 因式分解为 $(a-b)(a^{p-1} + a^{p-2}b + \cdots + b^{p-1})$。由于 $a \equiv b \pmod{p}$，括号中每一项 $\equiv a^{p-1} \pmod{p}$，共 $p$ 项，和 $\equiv p \cdot a^{p-1} \equiv 0 \pmod{p}$ 但不一定被 $p^2$ 整除——事实上恰好被 $p^1$ 整除。由此建立递推，归纳至一般 $n$。

**当 $p=2$ 时**：公式略有修改。若 $4 \mid (a-b)$，则 $v_2(a^n - b^n) = v_2(a-b) + v_2(n)$。若仅 $2 \mid (a-b)$ 但 $4 \nmid (a-b)$，则还需要添加一些额外条件（与 $n$ 的奇偶性有关）。竞赛中 $p$ 为奇素数的情况占绝对多数。

---

### 类型 2：$p \mid (a + b)$（$n$ 为奇数时）

设 $p$ 为奇素数，$p \mid (a+b)$ 且 $p \nmid ab$。则对任意**奇数** $n$：

$$
\boxed{v_p(a^n + b^n) = v_p(a + b) + v_p(n)}
$$

与类型 1 完全对称——只需将 $b$ 替换为 $-b$，并限制 $n$ 为奇数（否则 $a^n + b^n$ 不能被 $a+b$ 整除）。

---

### 当 $n$ 为偶数的处理

如果 $n$ 是偶数，$a^n + b^n$ 中只有因子 $2$ 需要特殊讨论（因为 $a^n + b^n$ 不整除于 $a+b$ 当 $n$ 偶时）。对于 $p=2$，如果 $a, b$ 都是奇数且 $n$ 是偶数，$a^n - b^n$ 的形式可以用平方差公式拆开处理。

竞赛中绝大多数 LTE 应用题都是类型 1（差）或类型 2（奇数幂和），这两种已经覆盖了 90% 的场景。

---

## 四、LTE 的应用示例

### 例 1：$v_5(6^{100} - 1)$

$p=5$，$a=6$，$b=1$。$5 \mid (6-1) = 5$，满足条件。$v_5(6-1) = v_5(5) = 1$。$n=100=5^2 \times 4$，$v_5(n) = 2$。

所以 $v_5(6^{100} - 1) = 1 + 2 = 3$。

验证：$6^{100} - 1$ 能被 $5^3 = 125$ 整除，但不能被 $5^4 = 625$ 整除。

### 例 2：引入问题

求 $\gcd(2^{100} + 1, 3^{100} - 1)$。分析公共素因子：
- 两个数都是奇数，不考虑 $2$。
- 对任意奇素数 $p$：如果 $p \mid (2^{100}+1)$，则 $2^{100} \equiv -1 \pmod{p}$，$2^{200} \equiv 1$。这意味着 $\operatorname{ord}_p(2) \mid 200$ 但不整除 $100$。
- 如果 $p \mid (3^{100}-1)$，则 $\operatorname{ord}_p(3) \mid 100$。

结合 LTE 对每个候选 $p$ 计算 $v_p$——虽然这道题的完整求解仍需一些工作，但 LTE 提供了系统分析每个素因子幂次的方法。

## 五、代码实现

LTE 通常不作为一个独立的函数调用，而是在解题过程中**手动分析**。但可以封装一个辅助函数：

```cpp
using ll = long long;

// 计算 p 在 n 中的幂次（又名 v_p(n)）
int vp(ll n, int p) {
    int cnt = 0;
    while (n % p == 0) n /= p, cnt++;
    return cnt;
}

// LTE 类型 1：p | (a-b), p 为奇素数，p ∤ ab
// 返回 v_p(a^n - b^n)
int lte_diff(ll a, ll b, ll n, int p) {
    return vp(a - b, p) + vp(n, p);
}

// LTE 类型 2：p | (a+b), p 为奇素数，p ∤ ab，n 为奇数
// 返回 v_p(a^n + b^n)
int lte_sum(ll a, ll b, ll n, int p) {
    return vp(a + b, p) + vp(n, p);
}
```

## 六、复杂度与正确性分析

LTE 的计算复杂度仅取决于 $v_p$ 的计算：$O(\log_p |a-b|)$ 用于求 $v_p(a-b)$，$O(\log_p n)$ 用于求 $v_p(n)$。LTE 公式本身是 $O(1)$ 的。

LTE 的严格证明需要二项式展开和归纳法，属于初等数论的范畴但步骤繁琐。在竞赛中，**可以直接使用结论**——LTE 是公认的标准工具，引用时不需要自证。

---

## 七、典型应用场景

1. **整除性与最大幂次**：求 $v_p(a^n \pm b^n)$，如「$3^{2024}+1$ 能被 $2$ 的几次幂整除」。
2. **GCD 问题**：求 $\gcd(a^n - 1, a^m - 1) = a^{\gcd(n,m)} - 1$ 配合 LTE 分析素因子幂次。
3. **数论不等式**：需要证明某个数被 $p^k$ 整除或不被 $p^{k+1}$ 整除的题目。

**推荐练习题**：

- [LightOJ 1278](http://lightoj.com/volume_showproblem.php?problem=1278) — Sum of Consecutive Integers。将 $n$ 写成至少两个连续正整数之和的方案数，用 LTE 分析 $v_2(n)$ 转化为奇因子计数
- [CF 837E](https://codeforces.com/problemset/problem/837/E) — Vasya's Function。通过质因子逐步消去分析 $f(a, b)$ 的下降过程，LTE 思维在迭代算法中的体现

## 八、小结

LTE 的核心公式只有两个——$v_p(a^n - b^n) = v_p(a-b) + v_p(n)$（条件：奇素数 $p \mid a-b$，$p \nmid ab$）及其加法版本。在竞赛中，它是素因子幂次分析的终极捷径。
