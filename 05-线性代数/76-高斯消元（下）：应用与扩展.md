## 一、引言

在上篇中，我们用高斯消元法求解线性方程组——给定 $Ax = b$，解出 $x$。但高斯消元的用途远不止于此。

考虑一个问题：给定一个 $n \times n$ 的矩阵 $A$，我们想知道它的行列式是多少？秩是多少？可逆吗？如果可逆，逆矩阵是多少？

对于 $n=2$ 或 $n=3$，我们有公式或对角线法则直接算。但 $n=1000$ 时呢？

答案还是高斯消元——只不过我们不把 $A$ 和 $b$ 拼成增广矩阵，而是直接对 $A$ 进行消元。在消元的过程中，我们顺便就能得到秩、行列式、逆矩阵等所有信息。

## 三、核心推导

### 3.1 矩阵的秩

矩阵的秩（rank）是矩阵行向量（或列向量）中线性无关的最大个数。直观理解：**秩就是矩阵中「真正有效」的信息维度数**。

在高斯消元的过程中，消元完成后行阶梯形中的**非零行数**就是矩阵的秩。

因为我们在消元中做的行变换不会改变行空间（行的线性组合关系不变），阶梯形中的零行对应原矩阵中可以被其他行线性表示的行。

**手算小例子**：

$$
A = \begin{bmatrix}
1 & 2 & 3 \\
2 & 4 & 6 \\
0 & 1 & 1
\end{bmatrix}
$$

消元过程：

第 1 步：$R_2 \leftarrow R_2 - 2R_1$，得到：

$$
\begin{bmatrix}
1 & 2 & 3 \\
0 & 0 & 0 \\
0 & 1 & 1
\end{bmatrix}
$$

第 2 步：第 2 列，第 2 行第 2 列为 0，往下找主元，发现第 3 行第 2 列为 1，交换 $R_2 \leftrightarrow R_3$：

$$
\begin{bmatrix}
1 & 2 & 3 \\
0 & 1 & 1 \\
0 & 0 & 0
\end{bmatrix}
$$

消元完成，非零行数为 2，所以 $\operatorname{rank}(A) = 2$。

验证一下：第 2 行 $[2,4,6]$ 确实是第 1 行 $[1,2,3]$ 的 2 倍，所以第 2 行是线性相关的。

### 3.2 行列式

对于方阵 $A$，行列式 $\det(A)$ 可以看作矩阵的「体积缩放因子」。在高斯消元的过程中，我们可以通过跟踪行变换对行列式的影响来计算行列式。

三种行变换对行列式的影响：

1. **交换两行**：$\det$ 变号（乘以 $-1$）
2. **某行乘以常数 $k$**：$\det$ 乘以 $k$
3. **某行加上另一行的倍数**：$\det$ 不变（这是关键！）

所以，我们只需要对矩阵进行消元，使之变成**上三角矩阵**，然后对角线元素相乘，再乘以每一步的符号变化，就得到了行列式。

$$
\det(A) = (-1)^{\text{交换次数}} \times \prod_{i=1}^{n} U_{ii}
$$

其中 $U$ 是消元后的上三角矩阵。

**手算小例子**：求 $A = \begin{bmatrix}2 & 1 & 1 \\ 1 & 2 & 1 \\ 1 & 1 & 2\end{bmatrix}$ 的行列式。

第 1 步：$R_2 \leftarrow R_2 - \frac{1}{2}R_1$，$R_3 \leftarrow R_3 - \frac{1}{2}R_1$。

$$
\det \text{不变} \quad \begin{bmatrix}
2 & 1 & 1 \\
0 & 1.5 & 0.5 \\
0 & 0.5 & 1.5
\end{bmatrix}
$$

第 2 步：$R_3 \leftarrow R_3 - \frac{1}{3}R_2$（因为 $\frac{0.5}{1.5} = \frac{1}{3}$）。

$$
\det \text{不变} \quad \begin{bmatrix}
2 & 1 & 1 \\
0 & 1.5 & 0.5 \\
0 & 0 & \frac{4}{3}
\end{bmatrix}
$$

没有交换行，所以交换次数 = 0。

$$
\det(A) = 2 \times 1.5 \times \frac{4}{3} = 4
$$

❓ **为什么行变换对行列式的影响是这样的？**

行列式的一个基本性质是它关于行是「多线性交错函数」：

- 多线性：如果把某一行写成 $u+v$，行列式等于两个行列式之和，且常数可以提出来
- 交错：两行相同则行列式为 0，交换两行变号

从这两个性质出发，可以推导出所有行变换的影响。实际上，对初学者来说，记住上面三条规则就够了。

### 3.3 逆矩阵

求逆矩阵的方法也很直接：把 $A$ 和单位矩阵 $I$ 并列成 $[A \mid I]$，然后通过行变换把左边变成 $I$，右边就是 $A^{-1}$。

原理很简单：我们做行变换相当于左乘一个初等矩阵。如果一系列初等矩阵 $E_1, E_2, \dots, E_k$ 能把 $A$ 变成 $I$，即：

$$
E_k \cdots E_2 E_1 A = I
$$

那么 $E_k \cdots E_2 E_1 = A^{-1}$，而 $E_k \cdots E_2 E_1 I = A^{-1}$，所以右边的 $I$ 经过同样的变换就变成了 $A^{-1}$。

**手算小例子**：求 $A = \begin{bmatrix}1 & 2 \\ 3 & 4\end{bmatrix}$ 的逆矩阵。

$$
[A \mid I] = \left[\begin{array}{cc|cc}
1 & 2 & 1 & 0 \\
3 & 4 & 0 & 1
\end{array}\right]
$$

$R_2 \leftarrow R_2 - 3R_1$：

$$
\left[\begin{array}{cc|cc}
1 & 2 & 1 & 0 \\
0 & -2 & -3 & 1
\end{array}\right]
$$

现在左边是上三角，我们需要把左边变成单位矩阵。先处理对角线：$R_2 \leftarrow -\frac{1}{2}R_2$。

$$
\left[\begin{array}{cc|cc}
1 & 2 & 1 & 0 \\
0 & 1 & \frac{3}{2} & -\frac{1}{2}
\end{array}\right]
$$

$R_1 \leftarrow R_1 - 2R_2$：

$$
\left[\begin{array}{cc|cc}
1 & 0 & -2 & 1 \\
0 & 1 & \frac{3}{2} & -\frac{1}{2}
\end{array}\right]
$$

所以 $A^{-1} = \begin{bmatrix}-2 & 1 \\ \frac{3}{2} & -\frac{1}{2}\end{bmatrix}$。

验证一下：$A \times A^{-1} = \begin{bmatrix}1\cdot(-2)+2\cdot\frac{3}{2} & 1\cdot1+2\cdot(-\frac{1}{2}) \\ 3\cdot(-2)+4\cdot\frac{3}{2} & 3\cdot1+4\cdot(-\frac{1}{2})\end{bmatrix} = \begin{bmatrix}1 & 0 \\ 0 & 1\end{bmatrix}$。正确！

### 3.4 模意义下的高斯消元

当我们在模素数 $p$ 下做高斯消元时，情况会更方便：我们可以用乘法逆元代替浮点数除法，避免精度问题。

模 $p$ 下的行变换：

1. 交换两行
2. $R_i \leftarrow k \cdot R_i$，其中 $k \neq 0 \pmod{p}$
3. $R_i \leftarrow R_i + k \cdot R_j$

消元因子 $c = a_{ik} \cdot (a_{kk})^{-1} \pmod{p}$，这样所有操作都是整数运算。

**手算小例子**：在模 $7$ 下解方程组 $\begin{cases}3x + 2y \equiv 1 \pmod{7} \\ 5x + 4y \equiv 3 \pmod{7}\end{cases}$

增广矩阵：$\left[\begin{array}{cc|c}3 & 2 & 1 \\ 5 & 4 & 3\end{array}\right]$

主元 $3$ 在模 $7$ 下的逆元是 $3^{-1} \equiv 5 \pmod{7}$（因为 $3 \times 5 = 15 \equiv 1 \pmod{7}$）。

用主元行消去第 2 行第 1 列：消元因子 $c = 5 \times 5 = 25 \equiv 4 \pmod{7}$（这里 $5 \times 5$ 是当前行的第一个元素 $5$ 乘上主元的逆元 $5$……不对，我们来算对）。

消元因子 $c = \frac{a_{21}}{a_{11}} = 5 \times 3^{-1} = 5 \times 5 = 25 \equiv 4 \pmod{7}$。

$R_2 \leftarrow R_2 - 4R_1 \pmod{7}$：

$$
[5-4\times3,\; 4-4\times2,\; 3-4\times1] = [5-12,\; 4-8,\; 3-4] = [-7,\; -4,\; -1]
$$

模 $7$ 下：$[0,\; 3,\; 6]$。

矩阵变为：

$$
\left[\begin{array}{cc|c}3 & 2 & 1 \\ 0 & 3 & 6\end{array}\right]
$$

第 2 行：$3y \equiv 6 \pmod{7}$，$y \equiv 6 \times 3^{-1} = 6 \times 5 = 30 \equiv 2 \pmod{7}$。

代入第 1 行：$3x + 2 \times 2 \equiv 1 \pmod{7}$，$3x + 4 \equiv 1$，$3x \equiv -3 \equiv 4 \pmod{7}$，$x \equiv 4 \times 3^{-1} = 4 \times 5 = 20 \equiv 6 \pmod{7}$。

所以解为 $x \equiv 6, y \equiv 2 \pmod{7}$。

❓ **模数不是素数时怎么办？**

如果模数不是素数（例如模 $6$），某些元素可能没有逆元。这时需要用扩展欧几里得算法来判断是否可逆，如果不可逆，需要额外处理——有时把原始矩阵的每行同时除以 $\gcd$ 可以继续消元，有时就直接判无解了。

对于模合数的情况，更系统的做法是使用辗转相除思想的高斯消元（又称「中国剩余定理」风格的消元），但超出了本文的范围。竞赛中绝大多数模数消元都是模素数。

## 四、代码实现

### 4.1 求秩

```cpp
#include <bits/stdc++.h>
using ll = long long;
using db = double;
const db EPS = 1e-9;

int rank_matrix(vector<vector<db>> a) {
    int n = (int)a.size(), m = (int)a[0].size();
    int rank = 0;
    for (int col = 0; col < m && rank < n; col++) {
        int sel = rank;
        for (int i = rank; i < n; i++)
            if (fabs(a[i][col]) > fabs(a[sel][col]))
                sel = i;
        if (fabs(a[sel][col]) < EPS) continue;
        swap(a[rank], a[sel]);
        // 用当前主元消去下方行
        for (int i = rank + 1; i < n; i++) {
            db c = a[i][col] / a[rank][col];
            for (int j = col; j < m; j++)
                a[i][j] -= a[rank][j] * c;
        }
        rank++;
    }
    return rank;
}
```

### 4.2 求行列式

```cpp
db det(vector<vector<db>> a) {
    int n = (int)a.size();
    int sign = 1;
    for (int i = 0; i < n; i++) {
        int sel = i;
        for (int j = i + 1; j < n; j++)
            if (fabs(a[j][i]) > fabs(a[sel][i]))
                sel = j;
        if (fabs(a[sel][i]) < EPS) return 0;  // 行列式为 0
        if (sel != i) {
            swap(a[i], a[sel]);
            sign = -sign;
        }
        // 消去下方行
        for (int j = i + 1; j < n; j++) {
            db c = a[j][i] / a[i][i];
            for (int k = i; k < n; k++)
                a[j][k] -= a[i][k] * c;
        }
    }
    db ans = sign;
    for (int i = 0; i < n; i++) ans *= a[i][i];
    return ans;
}
```

### 4.3 求逆矩阵

```cpp
// 返回逆矩阵，如果不可逆返回空矩阵
vector<vector<db>> inverse(vector<vector<db>> a) {
    int n = (int)a.size();
    // 构造 [A | I]
    vector<vector<db>> aug(n, vector<db>(2 * n));
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) aug[i][j] = a[i][j];
        aug[i][n + i] = 1;
    }
    for (int i = 0; i < n; i++) {
        int sel = i;
        for (int j = i + 1; j < n; j++)
            if (fabs(aug[j][i]) > fabs(aug[sel][i]))
                sel = j;
        if (fabs(aug[sel][i]) < EPS) return {};  // 不可逆
        swap(aug[i], aug[sel]);
        // 主元归一
        db piv = aug[i][i];
        for (int j = 0; j < 2 * n; j++) aug[i][j] /= piv;
        // 消去其他行
        for (int j = 0; j < n; j++) {
            if (j != i) {
                db c = aug[j][i];
                for (int k = 0; k < 2 * n; k++)
                    aug[j][k] -= c * aug[i][k];
            }
        }
    }
    // 提取右侧的逆矩阵
    vector<vector<db>> inv(n, vector<db>(n));
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            inv[i][j] = aug[i][n + j];
    return inv;
}
```

### 4.4 模素数下的高斯消元

```cpp
#include <bits/stdc++.h>
using ll = long long;

ll mod_pow(ll a, ll b, ll p) {
    ll res = 1;
    while (b) {
        if (b & 1) res = res * a % p;
        a = a * a % p;
        b >>= 1;
    }
    return res;
}
ll inv(ll a, ll p) { return mod_pow(a, p - 2, p); }

// 模 p 意义下高斯消元，返回 0-唯一解, 1-无穷多解, -1-无解
int gauss_mod(vector<vector<ll>> &a, vector<ll> &sol, ll p) {
    int n = (int)a.size(), m = (int)a[0].size() - 1;
    vector<int> where(m, -1);
    int row = 0;
    for (int col = 0; col < m && row < n; col++) {
        int sel = row;
        for (int i = row; i < n; i++)
            if (a[i][col] != 0) { sel = i; break; }
        if (a[sel][col] == 0) continue;
        swap(a[row], a[sel]);
        where[col] = row;
        ll piv_inv = inv(a[row][col], p);
        for (int i = 0; i < n; i++) {
            if (i != row && a[i][col] != 0) {
                ll c = a[i][col] * piv_inv % p;
                for (int j = col; j <= m; j++)
                    a[i][j] = (a[i][j] - a[row][j] * c % p + p) % p;
            }
        }
        row++;
    }
    for (int i = row; i < n; i++)
        if (a[i][m] != 0) return -1;
    if (row < m) return 1;
    sol.assign(m, 0);
    for (int i = 0; i < m; i++)
        sol[i] = a[where[i]][m] * inv(a[where[i]][i], p) % p;
    return 0;
}
```

## 五、复杂度与正确性分析

**复杂度**：所有三个算法（秩、行列式、逆矩阵）的时间复杂度都是 $O(n^3)$。模意义下的消元因为使用整数模运算而不是浮点运算，实际常数可能更小。

**空间复杂度**：$O(n^2)$。求逆矩阵需要 $O(2n^2)$ 的增广矩阵空间。

**正确性**：
- 秩：行变换不改变行空间，阶梯形的非零行数即秩
- 行列式：基于行变换对行列式的三条影响规则
- 逆矩阵：基于「行变换等价于左乘初等矩阵」的原理

## 六、典型应用场景

1. **判定矩阵是否可逆**：行列式是否非零（实数域）或与模互素（模域）
2. **求矩阵的秩**：判断向量组是否线性相关
3. **模意义下解方程**：密码学、网格图计数问题中的模方程
4. **生成树计数**：矩阵树定理（Kirchhoff 定理）的核心就是计算 Laplacian 矩阵的行列式

**推荐练习题**：

- [洛谷 P4783 【模板】矩阵求逆](https://www.luogu.com.cn/problem/P4783) — 模 $10^9+7$ 下求逆矩阵
- [洛谷 P2455 线性方程组](https://www.luogu.com.cn/problem/P2455) — 需要判断无解/无穷多解的高斯消元
- [Codeforces 963A Alternating Sum](https://codeforces.com/problemset/problem/963/A) — 利用模逆元化简求和

## 七、小结

本篇扩展了高斯消元的应用范围：求秩、求行列式、求逆矩阵、模意义下的消元。这些看似不同的任务，底层都是同一个算法——只不过在消元过程中我们「顺便」追踪了不同的附加信息。
