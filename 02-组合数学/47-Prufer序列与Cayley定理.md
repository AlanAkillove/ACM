# Prufer 序列与 Cayley 定理

> Made by Aki.
> 最后更新于 2026.04.30

> $n$ 个有标号的结点，可以构成多少棵不同的无根树？

答案是 $n^{n-2}$（Cayley 定理）。这个结果本身就很惊人，但更有价值的是它的证明工具——Prufer 序列，它建立了「有标号树」和「长度为 $n-2$ 的任意序列（每项 $\in [1,n]$）」之间的一一对应。这个双射使得树的计数问题可以被转化为纯组合的序列计数。

---

## 一、Prufer 序列的构造

给定一棵结点标号为 $1$ 到 $n$ 的树，按以下步骤生成 Prufer 序列：

1. 找到当前树中标号最小的**叶子**（度数为 $1$ 的结点）。
2. 将该叶子的**邻居**的标号追加到序列末尾。
3. 删除该叶子。
4. 重复 $n-2$ 次，直到剩下 $2$ 个结点。

**手动模拟**：对于 $n=5$，边为 $(1,3), (2,3), (3,4), (4,5)$ 的树（路径 1-3-4-5 外加 2-3）：

- 最小叶子是 $1$，邻居是 $3$，追加 $3$，删除 $1$。
- 最小叶子是 $2$，邻居是 $3$，追加 $3$，删除 $2$。
- 最小叶子是 $3$（此时 $3$ 度数为 $1$，因为 $1$ 和 $2$ 已被删），邻居是 $4$，追加 $4$。

Prufer 序列 = $[3, 3, 4]$。长度 = $5-2 = 3$。

---

## 二、解码——从 Prufer 序列恢复树

给定长度为 $n-2$ 的序列，每个元素 $\in [1,n]$，恢复原树：

1. 统计每个结点的度数（初始度数 = $1$ + 在 Prufer 序列中出现的次数）。
2. 重复 $n-2$ 次：找到度数为 $1$ 且标号最小的结点 $u$，设序列当前首元素为 $v$，添加边 $(u,v)$，将 $u$ 的度数减为 $0$，将 $v$ 的度数减 $1$。
3. 最后两个度数为 $1$ 的结点连边。

**编码和解码互为逆映射**——因此是双射。

---

## 三、Cayley 定理的推论

因为 Prufer 序列可以取任意长度为 $n-2$、值域为 $[1,n]$ 的序列，总数为 $n^{n-2}$。

> **给定度数序列的树计数**：若要求结点 $i$ 的度数为 $d_i$，则 $i$ 在 Prufer 序列中出现 $d_i - 1$ 次。满足该度数序列的树的数量为多重集排列数：
>
> $$
> \frac{(n-2)!}{(d_1-1)!(d_2-1)!\cdots(d_n-1)!}
> $$

**应用**：[AtCoder ABC 303 Ex](https://atcoder.jp/contests/abc303/tasks/abc303_h) 中直接考察了给定度数序列的树计数。

## 四、代码实现

```cpp
// 从树（邻接表）生成 Prufer 序列
vector<int> tree_to_prufer(int n, const vector<vector<int>> &g) {
    vector<int> deg(n+1), prufer;
    priority_queue<int, vector<int>, greater<int>> pq;  // 最小堆
    for (int i = 1; i <= n; i++) {
        deg[i] = g[i].size();
        if (deg[i] == 1) pq.push(i);
    }
    for (int k = 0; k < n - 2; k++) {
        int u = pq.top(); pq.pop();
        int v = -1;
        for (int w : g[u]) if (deg[w] > 0) { v = w; break; }
        prufer.push_back(v);
        deg[u]--; deg[v]--;
        if (deg[v] == 1) pq.push(v);
    }
    return prufer;
}
```
