# 隔板法（Stars and Bars）

> Made by Aki.
> 最后更新于 2026.04.30

「将 $10$ 个相同的球放入 $3$ 个不同的盒子，盒子可以为空——有多少种放法？」这类问题在竞赛中高频出现，而它们的统一解法就是**隔板法**（Stars and Bars）。名字来自可视化：球是星星（stars），盒子之间的分隔是竖线（bars）。

---

## 一、引言

第 29 篇的组合数 $\binom{n}{k}$ 回答的是「从 $n$ 个**不同**元素中选出 $k$ 个」。但隔板法处理的是另一种场景——当**物品相同**、**位置不同**时，计数公式长什么样？

这两种场景恰好是排列组合中最基础的两个维度的组合：「物品同/异 × 位置同/异」，构成了十二重计数法的骨架（第 42 篇会系统梳理）。

## 三、核心推导

### 3.1 正整数解——盒子不能空

> 方程 $x_1 + x_2 + \cdots + x_n = k$ 的**正整数解**个数为：

$$
\boxed{\binom{k - 1}{n - 1}}
$$

**推导**：想象把 $k$ 个相同的球排成一排：

```
● ● ● ● ● ● ● ● ● ●   （k=10）
```

在它们之间的 $k-1 = 9$ 个空隙中插入 $n-1 = 2$ 个隔板，将球分成 $n=3$ 段：

```
● ● ● | ● ● | ● ● ● ● ●
  x₁=3   x₂=2   x₃=5
```

每段至少有 $1$ 个球。从 $k-1$ 个空隙中选 $n-1$ 个放隔板——方案数就是 $\binom{k-1}{n-1}$。

**前提**：必须 $k \ge n$，否则无解（每个变量至少 $1$，加起来至少 $n$）。

❓ **隔板插在球的「空隙」里，而不是球的「两端」——为什么？** 因为隔板必须严格将球分成三段，不能有段为空。如果允许空段，隔板可以插在两端——那就是下一节的非负整数解。

**例题**：$x_1 + x_2 + x_3 = 10$ 的正整数解——把 $10$ 个球分 $3$ 份，每份至少 $1$：

$$
\binom{10-1}{3-1} = \binom{9}{2} = 36
$$

---

### 3.2 非负整数解——盒子可以为空

> 方程 $x_1 + x_2 + \cdots + x_n = k$ 的**非负整数解**个数为：

$$
\boxed{\binom{k + n - 1}{n - 1} = \binom{k + n - 1}{k}}
$$

**推导（变量替换法）**：令 $y_i = x_i + 1$，则 $y_i \ge 1$ 且 $y_1 + y_2 + \cdots + y_n = k + n$。问题转化为上一节的正整数解：

$$
\binom{(k+n) - 1}{n - 1} = \binom{k + n - 1}{n - 1}
$$

**推导（直接隔板法）**：$k$ 个球和 $n-1$ 个隔板排成一排，总共 $k + n - 1$ 个位置。选 $n-1$ 个位置放隔板（其余放球）：

```
● ● | | ● ● ● | ● ● | ●
x₁=2  x₂=0  x₃=3  x₄=2  x₅=1
```

相邻的隔板之间可能有 $0$ 个球（对应 $x_i = 0$）。隔板可以放在任意位置，包括两端。

**例题**：$x_1 + x_2 + x_3 = 10$ 的非负整数解——$10$ 个球分 $3$ 份，允许空：

$$
\binom{10+3-1}{3-1} = \binom{12}{2} = 66
$$

❓ **正整数解和非负整数解的区别，一句话说清？** 非负版本比正整数版本多了 $n$ 个虚拟的「备用球」。每个 $x_i$ 先分到 $1$ 个备用球（保证正整数），然后剩下 $k$ 个随意分。所以从 $k$ 变成 $k+n$，公式从 $\binom{k-1}{n-1}$ 变成 $\binom{k+n-1}{n-1}$。

---

### 3.3 带下界约束

如果每个变量 $x_i \ge L_i$（下界不一定是 $0$ 或 $1$）：

> 令 $y_i = x_i - L_i$，则 $y_i \ge 0$，且 $\sum y_i = k - \sum L_i$。套非负整数解公式。

**例题**：$x_1 + x_2 + x_3 = 20$，$x_1 \ge 3, x_2 \ge 5, x_3 \ge 2$。

$y_1 = x_1-3, y_2 = x_2-5, y_3 = x_3-2$，则 $y_1 + y_2 + y_3 = 10$，$y_i \ge 0$。解数 = $\binom{10+3-1}{3-1} = \binom{12}{2} = 66$。

---

### 3.4 带上界约束——容斥原理出场

如果 $x_i \le U_i$（同时有上界），直接转化失效。策略：先算**无上界**的解，再用**容斥原理**（第 33 篇）减去违反上界的部分。

具体来说，对每个 $i$，违反 $x_i \ge U_i+1$ 的解都可以用下界转化法计算（令 $y_i = x_i - (U_i+1)$），然后被容斥加减。这是隔板法和容斥原理的经典配合。

---

### 3.5 竞赛中的识别模式

题目中出现这些关键词时，考虑隔板法：

- 「相同的球 / 物品」分配到「不同的盒子 / 位置」
- 「方程 $x_1 + \cdots + x_n = k$ 的解的个数」
- 「和为 $k$ 的 $n$ 个正整数 / 非负整数」
- 「$n$ 种物品各选若干，共选 $k$ 个」（可重组合，完全等价）
- 「名额分配」「奖金分配」等实际场景

## 四、代码实现

```cpp
// 正整数解
ll positive_sol(ll k, ll n) {
    if (k < n) return 0;
    return C(k - 1, n - 1);
}

// 非负整数解
ll nonneg_sol(ll k, ll n) {
    return C(k + n - 1, n - 1);
}

// 带下界的解
ll bounded_sol(ll k, ll n, const vector<ll> &L) {
    ll sumL = 0;
    for (ll x : L) sumL += x;
    if (k < sumL) return 0;
    return C(k - sumL + n - 1, n - 1);
}
```

## 五、典型应用场景

1. **名额分配**：$n$ 个奖项分给 $k$ 个人（奖项相同、人不同）。
2. **可重组合**：$n$ 种物品选 $k$ 个 = 非负整数解。
3. **多项式展开的项数**：$(x_1 + \cdots + x_n)^k$ 展开有多少个不同的项？每个项对应一组指数 $(e_1,\dots,e_n)$ 满足 $\sum e_i = k$，$e_i \ge 0$。项数 = 非负整数解数 = $\binom{k+n-1}{n-1}$。

**推荐练习题**（仅涉及已学知识）：

- 网格路径计数（$n \times m$ 网格从 $(0,0)$ 到 $(n,m)$ 的路径数，共走 $n+m$ 步，选 $n$ 步向右——等价于隔板法中的选位置）
- [Codeforces 1288C](https://codeforces.com/problemset/problem/1288/C) — Two Arrays。构造递增序列 → 隔板法的巧妙转化
- [洛谷 P5520](https://www.luogu.com.cn/problem/P5520) — 青原樱。带下界约束的隔板法
