# 容斥原理

> Made by Aki.
> 最后更新于 2026.04.30

加法原理要求各类之间「互斥」——但现实比理想复杂。当集合之间存在重叠时，直接相加会重复计算那些落入交集的部分。容斥原理就是用来校正这个重复的：先全加起来，减去重复的，加回多减的，交替进行直到精确。

---

## 一、引言

> $1$ 到 $100$ 中，有多少个数能被 $3$ 或 $5$ 整除？

直觉写：$\lfloor 100/3 \rfloor + \lfloor 100/5 \rfloor = 33 + 20 = 53$。但是 $15$（$3$ 和 $5$ 的最小公倍数）的倍数被算了两次——既被算在「被 $3$ 整除」里，又被算在「被 $5$ 整除」里。减去 $\lfloor 100/15 \rfloor = 6$，最终 $33 + 20 - 6 = 47$。

这就是容斥的最简形式——两个集合的容斥。

## 三、核心推导

### 3.1 从两个到三个——规律的浮现

**两个集合**：

$$
|A \cup B| = |A| + |B| - |A \cap B|
$$

**三个集合**：

$$
|A \cup B \cup C| = |A|+|B|+|C| - |A \cap B| - |B \cap C| - |A \cap C| + |A \cap B \cap C|
$$

**规律**：奇数个集合的交——加。偶数个集合的交——减。符号是 $(-1)^{|S|-1}$。

---

### 3.2 一般形式

$$
\boxed{\left|\bigcup_{i=1}^{n} A_i\right| = \sum_{\emptyset \neq S \subseteq \{1..n\}} (-1)^{|S|-1} \left|\bigcap_{i \in S} A_i\right|}
$$

**为什么符号交替？** 我们来追踪一个同时属于 $m$ 个集合的元素。它在单个集合的求和 $\sum |A_i|$ 中被计算了 $\binom{m}{1}$ 次，在两两交集中被减去了 $\binom{m}{2}$ 次，在三三交集中被加回了 $\binom{m}{3}$ 次……总计：

$$
\binom{m}{1} - \binom{m}{2} + \binom{m}{3} - \cdots + (-1)^{m-1} \binom{m}{m} = 1 - (1-1)^m = 1
$$

恰好被计数一次！这正是容斥公式正确性的组合证明——交替符号保证了每个元素最终都被恰好计数一次。

❓ **如果 $m=0$（元素不属于任何 $A_i$），公式还成立吗？** 成立。此时元素出现在 $\sum|A_i|$ 中 $0$ 次，后面的所有交集贡献也都是 $0$。补集形式中，这样的元素恰好属于「满足某性质」的部分——它们没有被任何 $A_i$ 捕获，所以需要从全集中保留下来。

---

### 3.3 补集形式——竞赛中最实用的用法

直接统计「满足某性质」很困难，但统计「不满足」却很容易时，用补集：

$$
\boxed{|\text{满足}| = \text{全集总数} - |\text{不满足}|}
$$

这看似简单，但再加一层容斥就是：

**例题——欧拉函数的容斥推导**：$1$ 到 $n$ 中与 $n$ 互质的数有多少个？

「与 $n$ 互质」不好直接统计。但「与 $n$ 有公共素因子 $p$」很容易——就是 $\lfloor n/p \rfloor$ 个 $p$ 的倍数。对 $n$ 的所有素因子容斥：

$$
\varphi(n) = n - \sum_{p \mid n} \frac{n}{p} + \sum_{p<q \mid n} \frac{n}{pq} - \cdots = n \prod_{p \mid n} \left(1 - \frac{1}{p}\right)
$$

这正是第 10 篇欧拉函数公式的容斥证明——当年可能觉得莫名其妙，现在回头看，它只是容斥原理在「除以各素因子的倍数」上的自然应用。

---

### 3.4 容斥 + 隔板法——带上界的方程

方程 $x_1 + \cdots + x_n = k$，$0 \le x_i \le U_i$（每个变量同时有上界）。

**策略**：
1. 先算无上界的解数（隔板法）：$\binom{k+n-1}{n-1}$
2. 设 $A_i$ 为违反 $x_i \ge U_i+1$ 的解集。$|A_i|$ = 令 $y_i = x_i - (U_i+1)$，转非负整数解
3. 容斥加减更高阶的交集

**示例**：[CF 451E Devu and Flowers](https://codeforces.com/problemset/problem/451/E) 就是这种模式的经典题——$n \le 20$，$s_i \le 10^{12}$，直接用 $O(2^n)$ 的容斥枚举子集。

## 四、代码实现

```cpp
// 容斥统计 [1, n] 中能被 primes 集中至少一个素数整除的数的个数
ll count_divisible(ll n, const vector<int> &primes) {
    int m = primes.size();
    ll ans = 0;
    for (int mask = 1; mask < (1 << m); mask++) {
        ll prod = 1;
        int bits = 0;
        for (int i = 0; i < m; i++)
            if (mask & (1 << i))
                prod *= primes[i], bits++;
        if (prod > n) continue;            // 乘积已经超过 n，该组合无贡献
        if (bits & 1) ans += n / prod;     // 奇数个集合：加
        else          ans -= n / prod;     // 偶数个集合：减
    }
    return ans;
}
```

## 五、典型应用场景

1. **「至少一个满足」→ 容斥求和**：统计能被某集合中任意元素整除的数的个数。
2. **「恰好 $k$ 个满足」→ 二项式反演 + 容斥**（第 36 篇）。
3. **带上界的方程**：隔板法 + 容斥。
4. **概率与期望中的并事件概率**：$P(\bigcup A_i) = \sum P(A_i) - \sum P(A_i A_j) + \cdots$

**推荐练习题**：

- [洛谷 P1450](https://www.luogu.com.cn/problem/P1450) — 硬币购物。背包 + 容斥处理上界限制
