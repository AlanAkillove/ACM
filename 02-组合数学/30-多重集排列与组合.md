# 多重集排列与组合

> Made by Aki.
> 最后更新于 2026.04.30

第 29 篇假设所有待排列的元素是互不相同的。但现实中大量问题涉及重复元素——「MISSISSIPPI」的字母排列、从无限供应的糖果中选几颗——这些都需要多重集（Multiset）的计数公式。

---

## 一、引言

> 字符串 `"AAB"` 的三个字符，有多少种不同的排列？如果三个字符全不同（如 `ABC`），答案是 $3! = 6$ 种。但是 `A` 出现了两次，把两个 `A` 互换位置不会产生新排列——$6$ 种里有一半是重复的。实际只有 $3$ 种：`AAB, ABA, BAA`。

这个简单例子揭示了多重集计数的核心思想：**先假装所有元素全不同，算出全排列，再除以相同元素之间的内部排列数**。

## 三、核心推导

### 3.1 多重集全排列——有重复元素的排列

> 有 $n$ 个元素，分为 $k$ 种类型，第 $i$ 种有 $n_i$ 个不可区分的副本（$\sum n_i = n$）。所有元素的排列数为：

$$
\boxed{\frac{n!}{n_1! \; n_2! \; \cdots \; n_k!}}
$$

**推导**：先给所有元素贴上临时标签（变成全不同）——有 $n!$ 种排列。对于第 $i$ 种类型，这 $n_i$ 个元素之间的 $n_i!$ 种内部排列实际上都对应同一个多重集排列。由于各类型之间独立，除以所有 $n_i!$。

**例题**：单词 `MISSISSIPPI` 有 $11$ 个字母：M 出现 $1$ 次、I 出现 $4$ 次、S 出现 $4$ 次、P 出现 $2$ 次。排列数为：

$$
\frac{11!}{1! \times 4! \times 4! \times 2!} = \frac{39916800}{1 \times 24 \times 24 \times 2} = 34650
$$

❓ **为什么不是 $11! \div (1+4+4+2)!$？** 因为每种类型内部的重复是独立的。M 的 $1!$ 只消除 1 个字母内部的排列，I 的 $4!$ 消除 4 个 I 之间的排列——它们是乘在一起的（各自独立），不是加在一起的。

---

### 3.2 圆排列——围成一圈时的修正

> $n$ 个不同元素围成一圈的排列数为：

$$
\boxed{(n-1)!}
$$

**推导**：线性排列有 $n!$ 种。但圆上旋转视为同一——$n$ 种不同的旋转对应同一个圆排列（把所有人都挪一个位置不会改变圆上的相对顺序）。所以 $n! / n = (n-1)!$。

**拓展（翻转对称）**：如果是项链（可以翻转），还要再除以 $2$：$(n-1)!/2$（$n \ge 3$ 时）。

---

### 3.3 可重组合——允许重复选择的组合

> 从 $n$ 种类型中**可重复地**选择 $k$ 个（每种无限供应，顺序不计）。方案数为：

$$
\boxed{\binom{n + k - 1}{k} = \binom{n + k - 1}{n - 1}}
$$

**推导**：这是隔板法的经典应用。将 $k$ 个不可区分的「球」放入 $n$ 个不同的「盒子」（每个盒子代表一种类型，盒中球数代表该类型选了几次）。在 $k$ 个球之间插入 $n-1$ 个隔板来分区。总共 $k + (n-1)$ 个位置，从中选 $n-1$ 个放隔板（或等价地选 $k$ 个放球）——方案数 $\binom{k+n-1}{n-1}$。

**例题**：A、B、C 三种糖果各无限供应，选 $5$ 颗（顺序不计）：

$$
\binom{3+5-1}{5} = \binom{7}{5} = 21
$$

如果把 $21$ 种列出来：`AAAAA, AAAAB, AAAAC, AAABB, AAABC, AAACC, AABBB, AABBC, AABCC, AACCC, ABBBB, ABBBC, ABBCC, ABCCC, ACCCC, BBBBB, BBBBC, BBBCC, BBCCC, BCCCC, CCCCC`。确实 $21$ 种。

---

### 3.4 可重排列——每种类型有限供应时的排列

如果第 $i$ 种元素恰好有 $n_i$ 个不可区分的副本（共 $n$ 个），取其中 $k$ 个做排列——这时情况复杂，需要生成函数或多重求和。在竞赛中，通常用 DP 或指数生成函数处理。基础篇不做展开，留到第 43 篇生成函数。

## 四、代码实现

```cpp
// 多重集全排列（仅适合小 n，结果在 ll 范围内）
ll multiset_perm(ll n, const vector<ll> &cnt) {
    ll num = 1;
    for (ll i = 2; i <= n; i++) num *= i;
    for (ll c : cnt)
        for (ll i = 2; i <= c; i++) num /= i;
    return num;
}

// 可重组合：C(n + k - 1, k)
ll comb_repeat(ll n, ll k) {
    return C(n + k - 1, k);  // C 来自第 29 篇
}
```

## 五、典型应用场景

1. **含重复字符的字符串排列**：如「有多少个不同的字母重排」= 多重集全排列。
2. **可重组合 = 隔板法**：选物品、分配名额、方程非负整数解——这些看似无关的问题共享同一公式。
3. **圆排列**：围桌问题、项链计数（配合 Burnside 引理，第 49 篇）。
