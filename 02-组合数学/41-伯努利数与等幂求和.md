# 伯努利数与等幂求和

> Made by Aki.
> 最后更新于 2026.04.30

> $\sum_{i=1}^{n} i^{100}$ 怎么快速算？$n$ 巨大（$10^{18}$ 级别），暴力 $O(n)$ 不可能。

答案是：等幂求和是一个 $k+1$ 次多项式，而伯努利数恰好是这个多项式的系数。$O(k^2)$ 预处理伯努利数后，单次求值 $O(k)$。

---

## 一、伯努利数的定义

伯努利数由指数生成函数定义：

$$
\frac{x}{e^x - 1} = \sum_{n=0}^{\infty} B_n \frac{x^n}{n!}
$$

前几项：$B_0=1, B_1=-\frac{1}{2}, B_2=\frac{1}{6}, B_3=0, B_4=-\frac{1}{30}, B_5=0, B_6=\frac{1}{42}$。

奇数下标 $n\ge3$ 全为零。

**递推公式**：$\sum_{j=0}^{n} \binom{n+1}{j} B_j = 0 \;(n\ge1)$，可 $O(k^2)$ 预处理 $B_0$ 到 $B_k$。

---

## 二、Faulhaber 公式——等幂求和的闭形式

$$
\boxed{\sum_{i=1}^{n-1} i^k = \frac{1}{k+1} \sum_{j=0}^{k} \binom{k+1}{j} B_j \cdot n^{k+1-j}}
$$

这个公式对 $n$ 是 $O(k)$ 的——与 $n$ 的大小无关。当 $n$ 巨大（如 $10^{18}$）时价值无限。

**替代方案——下降幂 + 斯特林数**（不需要伯努利数）：

$$
i^k = \sum_{j=0}^{k} S(k,j) \cdot i^{\underline{j}}
$$

$$
\sum_{i=1}^{n} i^k = \sum_{j=0}^{k} S(k,j) \cdot \frac{(n+1)^{\underline{j+1}}}{j+1}
$$

用第 38 篇的 $S(k,j)$ 和下降幂的离散积分。在模素数环境下，这也只需要 $O(k^2)$ 预处理斯特林数。

## 三、代码实现

```cpp
// 递推求 B[0..n]（模 p），O(n^2)
vector<ll> bernoulli(int n, ll p) {
    vector<ll> B(n+1), fact(n+2,1), inv_fact(n+2);
    for (int i=1; i<=n+1; i++) fact[i] = fact[i-1]*i%p;
    inv_fact[n+1] = qpow(fact[n+1], p-2, p);
    for (int i=n+1; i>=1; i--) inv_fact[i-1] = inv_fact[i]*i%p;
    B[0] = 1;
    for (int i = 1; i <= n; i++) {
        ll sum = 0;
        for (int j = 0; j < i; j++)
            sum = (sum + C(i+1, j) * B[j]) % p;
        B[i] = (p - sum * qpow(i+1, p-2, p) % p) % p;
    }
    return B;
}
```
