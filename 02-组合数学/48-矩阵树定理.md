# 矩阵树定理（Kirchhoff）

> Made by Aki.
> 最后更新于 2026.04.30

Prufer 序列给出了**有标号树**的计数，矩阵树定理（Matrix-Tree Theorem）则给出了**任意给定图**的生成树数量。它通过图的拉普拉斯矩阵（Laplacian）的行列式来计算——将一个图论计数问题转化为线性代数的行列式计算。

---

## 一、定理陈述

> 对于无向图 $G$，其生成树的数量等于拉普拉斯矩阵 $L$ 的**任意一个余子式**（即去掉任意一行一列后的行列式）。

拉普拉斯矩阵 $L$ 定义为：

$$
L_{ij} = \begin{cases}
\deg(i) & i = j \\
-1 & i \neq j \text{ 且 } (i,j) \in E \\
0 & i \neq j \text{ 且 } (i,j) \notin E
\end{cases}
$$

---

## 二、为什么成立？（直觉说明）

拉普拉斯矩阵的每个余子式等于 $\sum_{T} 1$（对所有生成树求和）。这个结论的证明思路（Binet-Cauchy 定理）：

将 $L$ 去掉某行某列后写为 $B \cdot B^T$ 的形式（$B$ 是关联矩阵去掉一行后的结果）。Binet-Cauchy 定理说 $\det(B B^T) = \sum (\det B_S)^2$，其中 $B_S$ 取 $B$ 的 $n-1$ 列。如果这些列对应的边构成一个生成树，$\det B_S = \pm 1$；否则 $\det B_S = 0$。因此行列式恰好等于生成树个数。

---

## 三、有向图版本

矩阵树定理可以推广到有向图：

- 以 $r$ 为根的**内向树**（所有边指向 $r$）：去掉 $L^{out}$ 的第 $r$ 行第 $r$ 列后的行列式。
- 以 $r$ 为根的**外向树**（所有边从 $r$ 发出）：去掉 $L^{in}$ 的第 $r$ 行第 $r$ 列后的行列式。

其中 $L^{out}_{ii} = $ 出度，$L^{in}_{ii} = $ 入度。

---

## 四、代码实现

```cpp
// 求无向图的生成树个数（模 p）
ll count_spanning_trees(int n, const vector<pair<int,int>> &edges, ll p) {
    vector<vector<ll>> L(n, vector<ll>(n, 0));
    for (auto &[u, v] : edges) {
        u--, v--;
        L[u][u]++; L[v][v]++;
        L[u][v] = (L[u][v] - 1 + p) % p;
        L[v][u] = (L[v][u] - 1 + p) % p;
    }
    // 去掉第 n-1 行第 n-1 列，求 (n-1)×(n-1) 行列式（高斯消元，见第 75 篇）
    return determinant(L, n - 1, p);
}
```
