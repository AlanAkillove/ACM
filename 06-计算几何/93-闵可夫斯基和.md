> Made by Aki.
> 最后更新于 2026.04.30

## 引言

想象你有一个凸多边形 $A$ 和一个凸多边形 $B$。你想知道：把 $A$ 沿着 $B$ 的每个点"移动"一遍，$A$ 扫过的区域是什么样的？或者说，如果有一个机器人（形状为 $A$）在障碍物（形状为 $B$）附近移动，怎么判断它们是否碰撞？

这两个问题背后是同一个数学概念——**闵可夫斯基和（Minkowski Sum）**。它在计算几何中是一个强大的工具，特别是在碰撞检测和形态学操作中。

两个点集 $A$ 和 $B$ 的闵可夫斯基和定义为：
$$A \oplus B = \{a + b \mid a \in A, b \in B\}$$

如果用凸多边形 $A$ 的每个顶点去加 $B$ 的每个顶点，结果是一个新的凸多边形（如果 $A, B$ 都是凸的）。而且这个新多边形的边恰好是 $A$ 和 $B$ 的边按极角排序后的合并。

## 1. 闵可夫斯基和的几何意义

$A \oplus B$ 的几何解释：**拿 $B$ 贴着 $A$ 的边界走一圈，$B$ 的参考点所覆盖的区域**。

或者反过来看：如果 $A$ 是一个机器人，$B$ 是一个障碍物，用 $A$ 的参考点（比如中心）来衡量碰撞——那么所有会导致碰撞的参考点位置就是 $A$ 与 $-B$ 的闵可夫斯基和（其中 $-B$ 是 $B$ 关于原点的对称）。

$$\text{Collision region} = A \oplus (-B)$$

这就是**碰撞检测的闵可夫斯基和方法**：如果 $A$ 的参考点不在 $A \oplus (-B)$ 内，则 $A$ 与 $B$ 不相碰。

## 2. 凸多边形的闵可夫斯基和

两个凸多边形 $P$（顶点 $p_0, p_1, \ldots, p_{n-1}$）和 $Q$（顶点 $q_0, q_1, \ldots, q_{m-1}$）的闵可夫斯基和仍然是凸多边形。

**性质**：$P \oplus Q$ 的边集恰好是 $P$ 和 $Q$ 的所有边按极角排序后的结果。

> ❓ 为什么边集是合并后的？
>
> 想象两个凸多边形的边是向量。$P$ 的边向量和 $Q$ 的边向量各自按极角排好（逆时针）。闵可夫斯基和的新边就是把这两组向量按极角 merge 起来。结果多边形在某个方向上的"跨度"等于两个多边形在该方向上的跨度之和。

### 算法：归并边向量

```cpp
// 获取多边形的边向量（逆时针顺序）
vector<Vector> getEdges(const vector<Point> &poly) {
    int n = poly.size();
    vector<Vector> edges;
    for (int i = 0; i < n; i++) {
        edges.push_back(poly[(i + 1) % n] - poly[i]);
    }
    return edges;
}
```

归并过程类似归并排序的 merge。但注意：边的极角排序是 $\text{atan2}(y, x)$，范围 $(-\pi, \pi]$。对于凸多边形的边，它们自然就是按极角排序的。

```cpp
vector<Point> minkowskiSum(const vector<Point> &P, const vector<Point> &Q) {
    int n = P.size(), m = Q.size();
    
    // 先找到最左下角的点作为起点
    auto cmp = [](const Point &a, const Point &b) {
        if (sgn(a.y - b.y) != 0) return a.y < b.y;
        return a.x < b.x;
    };
    int pi = 0, qi = 0;
    for (int i = 1; i < n; i++) if (cmp(P[i], P[pi])) pi = i;
    for (int i = 1; i < m; i++) if (cmp(Q[i], Q[qi])) qi = i;
    
    // 从最左下角点开始
    vector<Point> res;
    res.push_back(P[pi] + Q[qi]);
    
    // 获取边向量并归并
    vector<Vector> eP = getEdges(P);
    vector<Vector> eQ = getEdges(Q);
    
    int i = 0, j = 0;
    while (i < n && j < m) {
        Vector cur;
        // 比较两个边向量的极角，选极角较小的
        if (sgn(cross(eP[i], eQ[j])) >= 0) {
            cur = eP[i++];
        } else {
            cur = eQ[j++];
        }
        res.push_back(res.back() + cur);
    }
    while (i < n) {
        res.push_back(res.back() + eP[i++]);
    }
    while (j < m) {
        res.push_back(res.back() + eQ[j++]);
    }
    
    // 去除共线点
    vector<Point> hull;
    for (const auto &p : res) {
        while (hull.size() >= 2) {
            Point a = hull[hull.size() - 2];
            Point b = hull.back();
            if (sgn(cross(b - a, p - b)) == 0) {
                hull.pop_back();  // 移除共线点
            } else break;
        }
        hull.push_back(p);
    }
    if (hull.size() > 1 && sgn(cross(hull.back() - hull[hull.size() - 2], hull[0] - hull.back())) == 0)
        hull.pop_back();
    
    return hull;
}
```

### 手算小例子

$$
P = \{(0,0), (2,0), (2,1), (0,1)\} \text{（一个 $2\times 1$ 的矩形）}
$$
$$
Q = \{(0,0), (1,0), (1,1), (0,1)\} \text{（一个 $1\times 1$ 的正方形）}
$$

$P$ 的边向量（逆时针，从最左下角 $(0,0)$ 开始）：
$(2,0), (0,1), (-2,0), (0,-1)$

$Q$ 的边向量（从 $(0,0)$ 开始）：
$(1,0), (0,1), (-1,0), (0,-1)$

归并（按极角，从 $0^\circ$ 开始逆时针）：

| 步 | 比较 | 选哪个 | 边向量 | 新点 |
|----|------|--------|--------|------|
| 0 | | 起点 | | $(0,0)+(0,0)=(0,0)$ |
| 1 | $(2,0)$ vs $(1,0)$，叉积=0 | $P$ 的 $(2,0)$ | $(2,0)$ | $(2,0)$ |
| 2 | $(0,1)$ vs $(1,0)$，叉积 $=-1<0$ | $Q$ 的 $(1,0)$ | $(1,0)$ | $(3,0)$ |
| 3 | $(0,1)$ vs $(0,1)$，叉积=0 | $P$ 的 $(0,1)$ | $(0,1)$ | $(3,1)$ |
| 4 | $(-2,0)$ vs $(0,1)$，叉积 $=-2$ | $Q$ 的 $(0,1)$ | $(0,1)$ | $(3,2)$ |
| 5 | $(-2,0)$ vs $(-1,0)$，叉积=0 | $P$ 的 $(-2,0)$ | $(-2,0)$ | $(1,2)$ |
| 6 | $(0,-1)$ vs $(-1,0)$，叉积 $=1>0$ | $Q$ 的 $(-1,0)$ | $(-1,0)$ | $(0,2)$ |
| 7 | $(0,-1)$ vs $(0,-1)$，叉积=0 | $P$ 的 $(0,-1)$ | $(0,-1)$ | $(0,1)$ |
| 8 | 只剩 $Q$ 的 $(0,-1)$ | | $(0,-1)$ | $(0,0)$（回到起点） |

结果：$(0,0) \to (2,0) \to (3,0) \to (3,1) \to (3,2) \to (1,2) \to (0,2) \to (0,1)$。去掉共线点后得到凸五边形 $(0,0),(3,0),(3,2),(0,2)$（等等，这是一个 $3\times 2$ 的矩形）。

验证：$P$ 是 $2\times1$，$Q$ 是 $1\times1$，闵可夫斯基和应该是 $3\times2$ 的矩形。正确！

## 3. 碰撞检测应用

碰撞检测的核心结论：

**两个凸多边形 $A$ 和 $B$ 不相交，当且仅当原点不在 $A \oplus (-B)$ 内。**

其中 $-B$ 表示 $B$ 绕原点旋转 $180^\circ$：对 $B$ 的每个顶点取负。

```cpp
bool checkCollision(const vector<Point> &A, const vector<Point> &B) {
    // 计算 B 关于原点的对称
    vector<Point> negB;
    for (const auto &p : B)
        negB.push_back(Point(-p.x, -p.y));
    
    // 计算闵可夫斯基和
    vector<Point> sum = minkowskiSum(A, negB);
    
    // 判断原点是否在 sum 内
    return pointInConvexPolygon(Point(0, 0), sum) != 0;
}
```

这个方法的优势在于**不需要实际计算闵可夫斯基和**也能判断碰撞——分治法或双指针判断原点是否在 $A \oplus (-B)$ 内，可以做到 $O(n+m)$。

## 4. 闵可夫斯基和的其他性质

- **交换律**：$A \oplus B = B \oplus A$
- **结合律**：$(A \oplus B) \oplus C = A \oplus (B \oplus C)$
- **凸性保持**：如果 $A$ 和 $B$ 是凸的，$A \oplus B$ 也是凸的
- **支持函数**：$h_{A\oplus B}(u) = h_A(u) + h_B(u)$，其中 $h_S(u) = \max_{p\in S} \langle p, u\rangle$ 是点集 $S$ 在方向 $u$ 上的支持函数

## 5. 补充：闵可夫斯基差

有些问题涉及**闵可夫斯基差** $A \ominus B = A \oplus (-B)$。注意这不是闵可夫斯基和的逆运算（闵可夫斯基和没有逆运算）。

判断 $A$ 和 $B$ 是否相交等价于检查原点是否在 $A \ominus B$ 内。

## 小结

闵可夫斯基和是两个点集的向量加法，对凸多边形来说有非常简洁的计算方式：

- **$O(n+m)$ 归并边向量**：两个凸多边形的边按极角合并就是结果
- **碰撞检测**：$A$ 和 $B$ 不相交 $\Leftrightarrow$ 原点不在 $A \ominus B$ 内
- **凸性保持**：凸多边形的闵可夫斯基和仍然是凸多边形

理解闵可夫斯基和，能让你在碰撞检测、形态学操作和某些几何不等式问题中有更深刻的视角。
