# 圆的计算

> Made by Aki.
> 最后更新于 2026.04.30

## 引言

圆在计算几何中无处不在。如果说直线和线段是"线性"的基石，那么圆就是"非线性"的代表。判断圆和直线是否相交、求两个圆的交点、找两个圆的公切线——这些看起来像是高中数学题，但写成代码却有一堆边界情况需要处理。

更重要的是，有一类经典问题叫**最小圆覆盖**——给你 $n$ 个点，找一个半径最小的圆覆盖所有点。这东西在三维扫描、碰撞检测里都有应用。还有一个更难的——**圆的面积并**——多个圆重叠在一起，总面积是多少？

这一篇我们把圆的相关操作系统整理一遍。

## 1. 圆的基本表示

```cpp
struct Circle {
    Point c;  // 圆心
    double r; // 半径
    Circle(Point c = Point(), double r = 0) : c(c), r(r) {}
};
```

判断点与圆的位置关系：

```cpp
// 0 = 圆内, 1 = 圆上, 2 = 圆外
int pointCircleRelation(const Point &p, const Circle &cir) {
    double d = length(p - cir.c);
    if (sgn(d - cir.r) < 0) return 0;  // 内
    if (sgn(d - cir.r) == 0) return 1; // 上
    return 2;  // 外
}
```

## 2. 圆与直线的交点

给定一个圆和一条直线（点向式），求它们的交点。

思路：把直线参数化 $P(t) = P_0 + t\vec{v}$，代入圆的方程 $(P(t)-O)^2 = r^2$，得到关于 $t$ 的一元二次方程。

```cpp
// 圆与直线的交点（直线用点加方向向量表示）
// 返回交点个数并填充结果数组
int circleLineIntersect(const Circle &cir, const Point &p, const Vector &v,
                         vector<Point> &res) {
    Vector oc = p - cir.c;
    double a = dot(v, v);
    double b = 2 * dot(v, oc);
    double c = dot(oc, oc) - cir.r * cir.r;
    double delta = b * b - 4 * a * c;
    
    if (sgn(delta) < 0) return 0;
    
    if (sgn(delta) == 0) {
        double t = -b / (2 * a);
        res.push_back(p + v * t);
        return 1;
    }
    
    double t1 = (-b - sqrt(delta)) / (2 * a);
    double t2 = (-b + sqrt(delta)) / (2 * a);
    res.push_back(p + v * t1);
    res.push_back(p + v * t2);
    return 2;
}
```

注意：`delta` 可能是浮点数，用 `sgn(delta)` 判断等于 0 时要小心。实际实现中对 delta 做 `max(delta, 0.0)` 再开方更安全。

> ❓ 为什么不用斜率式直线公式？
>
> 用斜率式 $y = kx + b$ 会漏掉竖直的直线（$k$ 无穷大）。点向式 $(P_0,\vec{v})$ 可以表示所有直线，更安全。

## 3. 圆与圆的交点

两个圆 $(O_1, r_1)$ 和 $(O_2, r_2)$ 的交点。设圆心距 $d = |O_1O_2|$。

判断位置关系：
- $d > r_1 + r_2$：相离，无交点
- $d = r_1 + r_2$：外切，1 个交点
- $|r_1 - r_2| < d < r_1 + r_2$：相交，2 个交点
- $d = |r_1 - r_2|$：内切，1 个交点
- $d < |r_1 - r_2|$：内含，无交点

求交点的方法：用余弦定理确定交点在 $O_1O_2$ 连线方向上的投影，再用法向量确定偏移。

```cpp
int circleCircleIntersect(const Circle &c1, const Circle &c2,
                           vector<Point> &res) {
    Vector v = c2.c - c1.c;
    double d = length(v);
    
    // 相离或内含
    if (sgn(d - c1.r - c2.r) > 0 || sgn(d - fabs(c1.r - c2.r)) < 0 || sgn(d) == 0)
        return 0;
    
    double a = (c1.r * c1.r - c2.r * c2.r + d * d) / (2 * d);
    double h = sqrt(max(0.0, c1.r * c1.r - a * a));
    
    Vector va = v * (a / d);     // O1 到交点在 O1O2 方向上的投影
    Vector vh = rotate(v, PI / 2) * (h / d); // 垂直方向偏移
    
    res.push_back(c1.c + va + vh);
    if (sgn(h) > 0)
        res.push_back(c1.c + va - vh);
    
    return res.size();
}
```

这里用到了 `rotate(v, theta)`——将向量 $v$ 旋转 $\theta$ 角度。对于二维向量，逆时针旋转 $\theta$ 的公式是：
$(x\cos\theta - y\sin\theta, x\sin\theta + y\cos\theta)$。旋转 $90^\circ$ 时简化为 $(-y, x)$ 或 $(y, -x)$。

```cpp
Vector rotate(const Vector &a, double theta) {
    return Vector(a.x * cos(theta) - a.y * sin(theta),
                  a.x * sin(theta) + a.y * cos(theta));
}
```

### 手算例子

圆 $C_1((0,0), 5)$ 和 $C_2((8,0), 5)$：

圆心距 $d=8$，$r_1=r_2=5$。
$a = (25 - 25 + 64) / (2 \times 8) = 64/16 = 4$
$h = \sqrt{25 - 16} = 3$

投影点：$(0,0) + (8,0) \times (4/8) = (4,0)$
垂直偏移：旋转 $(8,0)$ 得到 $(0,8)$，乘以 $3/8$ 得到 $(0,3)$

交点：$(4,3)$ 和 $(4,-3)$。正确！

## 4. 圆的公切线

两个圆的公切线分两类：**外公切线**（两圆在切线同侧）和**内公切线**（两圆在切线异侧）。

### 外公切线

把两个圆的半径"缩小"成点：把大圆半径减去小圆半径，问题转化为"求过圆外一点到圆的切线"。

```cpp
// 过圆外一点求圆的切线，返回切点
int tangentPoint(const Circle &cir, const Point &p, vector<Point> &res) {
    Vector v = p - cir.c;
    double d = length(v);
    if (sgn(d - cir.r) < 0) return 0;  // 点在圆内
    
    double base = acos(cir.r / d);  // 夹角
    double angle = atan2(v.y, v.x); // 向量方向
    
    res.push_back(cir.c + Vector(cos(angle + base), sin(angle + base)) * cir.r);
    res.push_back(cir.c + Vector(cos(angle - base), sin(angle - base)) * cir.r);
    return 2;
}

// 求两个圆的外公切线（两条），返回直线的条数
int outerTangent(const Circle &c1, const Circle &c2, vector<Line> &lines) {
    Circle c = c1;
    c.r -= c2.r;  // 将大圆缩小为点，小圆缩小为点
    // 实际上，处理为：将 c1 半径减去 c2.r，求从 c1 圆心到"缩小后圆"的切线
    // 然后平移回正确位置
    // 这里实现的是简化的版本
    
    if (sgn(c1.r - c2.r) < 0) {
        return outerTangent(c2, c1, lines);  // 保证 c1 半径更大
    }
    
    Vector v = c2.c - c1.c;
    double d = length(v);
    
    if (sgn(d - (c1.r - c2.r)) <= 0) return 0;  // 内含或内切
    
    double angle = atan2(v.y, v.x);
    double phi = acos((c1.r - c2.r) / d);
    
    Point p1 = c1.c + Vector(cos(angle + phi), sin(angle + phi)) * c1.r;
    Point p2 = c2.c + Vector(cos(angle + phi), sin(angle + phi)) * c2.r;
    lines.push_back(Line(p1, p2));
    
    p1 = c1.c + Vector(cos(angle - phi), sin(angle - phi)) * c1.r;
    p2 = c2.c + Vector(cos(angle - phi), sin(angle - phi)) * c2.r;
    lines.push_back(Line(p1, p2));
    
    return 2;
}
```

### 内公切线

类似，但把 $c_2$ 的半径取反：$c_1.r + c_2.r$。如果 $d < c_1.r + c_2.r$ 时不存在内公切线（两圆相交或内含）。

```cpp
int innerTangent(const Circle &c1, const Circle &c2, vector<Line> &lines) {
    Vector v = c2.c - c1.c;
    double d = length(v);
    
    if (sgn(d - (c1.r + c2.r)) <= 0) return 0;  // 相交或内含
    
    double angle = atan2(v.y, v.x);
    double phi = acos((c1.r + c2.r) / d);
    
    Point p1 = c1.c + Vector(cos(angle + phi), sin(angle + phi)) * c1.r;
    Point p2 = c2.c - Vector(cos(angle + phi), sin(angle + phi)) * c2.r;
    lines.push_back(Line(p1, p2));
    
    p1 = c1.c + Vector(cos(angle - phi), sin(angle - phi)) * c1.r;
    p2 = c2.c - Vector(cos(angle - phi), sin(angle - phi)) * c2.r;
    lines.push_back(Line(p1, p2));
    
    return 2;
}
```

## 5. 最小圆覆盖（Minimal Enclosing Circle）

用**最小的圆**覆盖所有 $n$ 个点。这个圆也叫最小外接圆（Smallest Enclosing Circle）。

**Welzl 随机增量算法**：每次随机加入一个点，如果点在当前圆内，继续；否则以该点为边界，重新计算覆盖已处理点的最小圆。

复杂度：**期望 $O(n)$**。

### 算法思想

```
最小圆覆盖(点集 S):
  如果 |S| == 0 或 已经确定了边界点:
    返回根据边界点确定的圆(0~3个点确定圆)
  
  随机选一个点 p ∈ S
  递归计算 S\{p} 的最小圆 C
  如果 p 在 C 内: 返回 C
  否则: 把 p 加入边界点, 重新计算
```

边界点最多 3 个（三点确定一个圆，或两点确定直径）。

### 辅助函数

```cpp
// 由两点确定的圆（以这两点为直径）
Circle circleBy2(const Point &a, const Point &b) {
    Point c((a.x + b.x) / 2, (a.y + b.y) / 2);
    return Circle(c, length(a - c));
}

// 由三点确定的圆（外接圆）
Circle circleBy3(const Point &a, const Point &b, const Point &c) {
    // 用两条中垂线的交点求圆心
    double ax = a.x, ay = a.y, bx = b.x, by = b.y, cx = c.x, cy = c.y;
    double d = 2 * (ax * (by - cy) + bx * (cy - ay) + cx * (ay - by));
    if (sgn(d) == 0) return Circle(Point(0, 0), 0);  // 三点共线
    
    double ux = ((ax * ax + ay * ay) * (by - cy) + (bx * bx + by * by) * (cy - ay) + (cx * cx + cy * cy) * (ay - by)) / d;
    double uy = ((ax * ax + ay * ay) * (cx - bx) + (bx * bx + by * by) * (ax - cx) + (cx * cx + cy * cy) * (bx - ax)) / d;
    Point center(ux, uy);
    return Circle(center, length(center - a));
}
```

### 主算法

```cpp
// 递推函数：边界点集 R 中的点必须在圆上
Circle welzl(vector<Point> &pts, vector<Point> R, int n) {
    if (n == 0 || R.size() == 3) {
        switch (R.size()) {
            case 0: return Circle(Point(0, 0), 0);
            case 1: return Circle(R[0], 0);
            case 2: return circleBy2(R[0], R[1]);
            case 3: return circleBy3(R[0], R[1], R[2]);
        }
    }
    
    // 随机选一个点
    int idx = rand() % n;
    Point p = pts[idx];
    swap(pts[idx], pts[n - 1]);  // 移到末尾，便于递归
    
    Circle C = welzl(pts, R, n - 1);
    
    if (sgn(length(p - C.c) - C.r) <= 0) {
        // p 在圆内，交换回来
        swap(pts[idx], pts[n - 1]);
        return C;
    }
    
    // p 在圆外，把它加入边界
    R.push_back(p);
    C = welzl(pts, R, n - 1);
    swap(pts[idx], pts[n - 1]);
    return C;
}

Circle minEnclosingCircle(vector<Point> pts) {
    random_shuffle(pts.begin(), pts.end());  // 随机化，保证期望 O(n)
    vector<Point> R;
    return welzl(pts, R, pts.size());
}
```

> ❓ 为什么期望是 $O(n)$？
>
> 对于随机排列的点，每个点被选为"边界点"的概率为 $3/i$（第 $i$ 个点）。只有当点在当前圆外时才需要额外 $O(i)$ 的代价。期望代价为 $\sum_{i=1}^n O(1) + \frac{3}{i} \cdot O(i) = O(n)$。

## 6. 圆的面积并（自适应 Simpson 积分）

多个圆重叠在一起，求总面积。直接几何计算非常复杂（需要求所有圆之间的交并区域），竞赛中常用**自适应 Simpson 积分**来数值计算。

### Simpson 公式

用二次函数近似曲线：$\int_a^b f(x)dx \approx \frac{b-a}{6}(f(a) + 4f(\frac{a+b}{2}) + f(b))$

### 自适应

把区间不断二分，直到 Simpson 近似足够精确：

```cpp
// 在 x 处，所有圆在竖直方向上的总覆盖长度（并集）
double f(double x, const vector<Circle> &circles) {
    vector<pair<double, double>> segs;
    for (const auto &c : circles) {
        double dx = fabs(x - c.c.x);
        if (sgn(dx - c.r) >= 0) continue;
        double dy = sqrt(c.r * c.r - dx * dx);
        segs.push_back({c.c.y - dy, c.c.y + dy});
    }
    if (segs.empty()) return 0.0;
    
    sort(segs.begin(), segs.end());
    double len = 0, l = segs[0].first, r = segs[0].second;
    for (size_t i = 1; i < segs.size(); i++) {
        if (sgn(segs[i].first - r) > 0) {
            len += r - l;
            l = segs[i].first;
            r = segs[i].second;
        } else {
            r = max(r, segs[i].second);
        }
    }
    len += r - l;
    return len;
}

double simpson(double l, double r, const vector<Circle> &circles) {
    double mid = (l + r) / 2;
    return (r - l) / 6 * (f(l, circles) + 4 * f(mid, circles) + f(r, circles));
}

double adaptiveSimpson(double l, double r, double eps,
                        const vector<Circle> &circles) {
    double mid = (l + r) / 2;
    double S = simpson(l, r, circles);
    double SL = simpson(l, mid, circles);
    double SR = simpson(mid, r, circles);
    if (fabs(SL + SR - S) < 15 * eps) return SL + SR + (SL + SR - S) / 15;
    return adaptiveSimpson(l, mid, eps / 2, circles) +
           adaptiveSimpson(mid, r, eps / 2, circles);
}
```

使用自适应 Simpson 时，积分区间需要覆盖所有圆的左右边界。通常取所有圆 $x$ 方向的最小值和最大值。

## 小结

圆的计算是计算几何中"非线性"的部分，需要小心处理各种边界情况：

- **圆与直线相交**：点向式代入圆的方程，解一元二次方程
- **圆与圆相交**：余弦定理求投影，注意内切外切的边界
- **公切线**：将半径加减转化为点到圆的切线问题
- **最小圆覆盖**：Welzl 随机增量，期望 $O(n)$，实现非常简洁
- **圆的面积并**：自适应 Simpson 积分，精度足够即可
